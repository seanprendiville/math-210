    Subsection 2.1 Self-test questions for lecture 3

    - 2.1. Monotone convergence principle.

    (a)
    Define what it means for a real-valued function/sequence to be increasing/decreasing.
    Solution.
    Definition 2.2. Increasing/decreasing/monotone.
    Let \(f : D \to\R\) be a real-valued function with domain \(D\subset \R\text{.}\) We say that \(f\) is increasing on \(D\) if
    \begin{equation*} (\forall x, y \in D)[x \leq y \implies f(x) \leq f(y)] \end{equation*}
    We say that \(f\) is strictly increasing on \(D\) if
    \begin{equation*} (\forall x, y \in D)[x \lt y \implies f(x) \lt f(y)] \end{equation*}
    We say that \(f\) is decreasing on \(D\) if
    \begin{equation*} (\forall x, y \in D)[x \leq y \implies f(x) \geq f(y)] \end{equation*}
    We say that \(f\) is strictly decreasing on \(D\) if
    \begin{equation*} (\forall x, y \in D)[x \lt y \implies f(x) \gt f(y)] \end{equation*}
    We say that \(f\) is monotone if it is either increasing or decreasing.
    Recall that a sequence of reals \((a_n)_{n=1}^\infty\) is a function \(f : \N \to \R\) where \(f(n) := a_n\) for each \(n \in \N\text{.}\) One can check that such a sequence \((a_n)_{n=1}^\infty\) is increasing (as defined above) if and only if
    \begin{equation*} (\forall n \in \N)[a_n \leq a_{n+1}]. \end{equation*}
    A similar characterisation holds for decreasing sequences.
    (b)
    State and prove the monotone convergence principle.
    Solution.
    Proposition 2.3. Monotone convergence principle.
    Let \((a_n)_{n=1}^\infty\) be a sequence of real numbers. Suppose that the sequence is increasing and bounded above. Then the sequence converges.
    Proof.
    Since the sequence is bounded above, the set \(\set{a_n: n \in \N}\) is bounded above. It is also non-empty (containing, e.g., \(a_1\)), hence has a supremum \(a = \sup_{n \in \N} a_n \in \R\) by the completeness principle (see Theorem 1.15). We claim that \(a_n \to a\text{.}\) According to the definition of a real limit (Definition 1.23), we must verify that (1.5) holds. To this end, let \(\eps \gt 0\) be arbitrary. Since decreasing the supremum gives a non-upper bound (Lemma 1.9) there exists \(a_N \in \set{a_n : n \in \N}\) such that \(a_N \gt a - \eps\text{.}\) Since the sequence is increasing, if \(n \geq N\) then \(a_n \gt a -\eps\text{.}\) As \(a = \sup_n a_n\) is an upper bound, we deduce that for all \(n \geq N\) we have
    \begin{equation*} a-\eps \leq a_n \leq a. \end{equation*}
    In particular
    \begin{equation*} n \geq N \implies |a-a_n| \leq \eps. \end{equation*}
    Therefore \(a_n \to a\) by definition (see (1.5)), completing our proof.

    - 2.4. Inequality rule.

    State and prove how inequalities are preserved under taking limits.
    Solution.
    Proposition 2.5.
    Let \((a_n)_n\) and \((b_n)_n\) be sequences of real numbers such that \(a_n\to a\) and \(b_n\to b\) as \(n\to\infty\text{.}\) Suppose that \(a_n \leq b_n\) for all \(n\text{.}\) Then \(a \leq b\text{.}\)
    The conclusion of this result still holds if one relaxes the assumption "\(a_n \leq b_n\) for all \(n\)’’ to "\(a_n \leq b_n\) for all but finitely 1  many \(n\)’’.
    Proof.
    We shall derive a contradiction from the assumption that \(a \gt b\text{.}\)
    By definition of the limit (Definition 1.23,) we have that
    \begin{align} \amp\forall \eps \gt 0\ \exists N_1 = N_1(\eps)\ \forall n \geq N_1 \text{ we have } \abs{a - a_n} \leq \eps,\tag{2.1}\\ \amp\forall \eps \gt 0\ \exists N_2 = N_2(\eps)\ \forall n \geq N_2 \text{ we have } \abs{b - b_n} \leq \eps .\tag{2.2} \end{align}
    Set
    \begin{equation*} \eps:= \trecip 3 (a-b) \gt 0 \qquad \text{and} \qquad n := \max\set{N_1(\eps), N_2(\eps)}. \end{equation*}
    Then by (2.1), (2.2) and the triangle inequality
    \begin{align*} a_n = a + (a_n-a)\amp = b + (a-b) + (a_n-a)\\ \amp = b_n + (b-b_n) + (a-b) + (a_n-a)\\ \amp \geq b_n +(a-b) - |b-b_n| - |a_n-a|\\ \amp \geq b_n + 3 \eps - \eps - \eps\\ \amp \gt b_n. \end{align*}
    This contradicts our assumption that \(a_n \leq b_n\) for all \(n\text{,}\) completing our proof.
    Example 2.6. An example of using the inequality rule.
    If \((a_n)_{n=1}^\infty\) is a convergent sequence of negative numbers, then \(\lim_{n\to \infty} a_n \leq 0\text{.}\)
    Proof.
    Set \(b_n:=0\) for all \(n \in \N\text{.}\) Then \(b_n\to 0\) as \(n\to \infty\text{,}\) and for all \(n\in \N\) we have \(a_n\leq b_n\text{.}\) Thus by the inequality rule
    \begin{equation*} \lim_{n\to \infty} a_n \leq \lim_{n\to \infty} b_n = 0. \end{equation*}
    Example 2.7. Limits may not respect strict inequalities.
    If \(a_n:=0\) and \(b_n:=1/n\) for all \(n\in \N\text{,}\) then we have convergent sequences satisfying \(a_n \lt b_n\) for all \(n\text{,}\) yet we don’t have
    \begin{equation*} \lim_{n\to\infty} a_n \lt \lim_{n\to \infty} b_n. \end{equation*}
    Example 2.8. The converse to the inequality rule may not hold.
    We can have
    \begin{equation*} \lim_{n\to\infty} a_n \leq \lim_{n\to \infty} b_n. \end{equation*}
    without having that \(a_n \leq b_n\) for all \(n\text{.}\) For instance take \(a_n := 1/n\) and \(b_n:=-1/n\text{.}\)

    - 2.9. Sandwich rule.

    State and prove the "sandwich rule" for sequence limits.
    Solution.
    Proposition 2.10. Sandwich rule.
    Let \((a_n)_{n=1}^\infty\text{,}\) \((b_n)_{n=1}^\infty\) and \((c_n)_{n=1}^\infty\) be sequences of real numbers. Suppose that for all \(n \in \N\) we have
    \begin{equation*} a_n \leq b_n \leq c_n\text{.} \end{equation*}
    If \(a_n \to a\) and \(c_n \to a\text{,}\) then \(b_n \to a\text{.}\)
    Proof.
    By definition of the limit Definition 1.23, we have that
    \begin{align*} \amp\forall \eps \gt 0\ \exists N_1 = N_1(\eps)\ \forall n \geq N_1 \text{ we have } a_n \in [a-\eps,a+\eps],\\ \amp\forall \eps \gt 0\ \exists N_2 = N_2(\eps)\ \forall n \geq N_2 \text{ we have } c_n \in [a-\eps,a+\eps] . \end{align*}
    Fixing \(\eps \gt 0\text{,}\) take
    \begin{equation*} N(\eps) := \max\set{N_1(\eps), N_2(\eps)}. \end{equation*}
    Then for any \(n \geq N(\eps)\) we have \(n \geq N_2(\eps)\) and so
    \begin{equation*} b_n \leq c_n \leq a+ \eps. \end{equation*}
    Similarly, if \(n \geq N(\eps)\) then \(n \geq N_1(\eps)\text{,}\) hence
    \begin{equation*} b_n \geq a_n \geq a -\eps. \end{equation*}
    Thus for any \(n \geq N(\eps)\) we have
    \begin{equation*} b_n \in [a-\eps, a+\eps]. \end{equation*}
    Equivalently, for any \(n \geq N(\eps)\) we have \(|b_n -a| \leq \eps.\) Therefore \(b_n \to a \) as \(n \to \infty\text{.}\)
    Example 2.11. An example of using the sandwich rule.
    Define
    \begin{equation*} b_n := \frac{(-1)^{n\text{th digit of }\pi}}{n}, \end{equation*}
    so that \(b_1 = -1\text{,}\) \(b_2 = -1/2\text{,}\) \(b_3 = 1/3\text{,}\) etc. The terms of this sequence oscillate between positive and negative in a seemingly arbitrary manner. Yet
    \begin{equation*} -1/n \leq b_n \leq 1/n \qquad (\forall n \in \N), \end{equation*}
    and
    \begin{equation*} \lim_{n\to\infty} (1/n) = \lim_{n\to\infty} (-1/n) = 0. \end{equation*}
    (The latter by the scaling rule combined with convergence of \(1/n\text{.}\)) Hence by the sandwich rule, we have
    \begin{equation*} \lim_{n\to \infty} b_n = 0. \end{equation*}
    Remark 2.12.
    The conclusion of the sandwich rule still holds if the inequalities \(a_n\leq b_n\leq c_n\) only hold for all but finitely 2  many values of \(n\text{.}\) For instance, the second inequality in
    \begin{equation*} 0 \leq 100/n^2 \leq 1/n \end{equation*}
    doesn’t hold for \(n \lt 100\text{.}\) Yet since these inequalities hold for all \(n\geq 100\text{,}\) we can still use the sandwich rule to conclude that \(100/n^2 \to 0\) as \(n\to\infty\text{.}\)

    Subsection 2.2 Self-test questions for lecture 4

    - 2.13. Addition rule.

    State and prove the "addition rule" for limits of convergent sequences.
    Solution.
    Proposition 2.14. Addition rule.
    Let \((a_n)_{n=1}^\infty\) and \((b_n)_{n =1}^\infty\) be convergent sequences of real numbers. Then
    \begin{equation*} \lim_{n \to \infty} (a_n+b_n) = \lim_{n \to \infty} a_n+\lim_{n \to \infty} b_n. \end{equation*}
    Proof.
    Let \(\displaystyle a : = \lim_{n \to \infty}a_n\) and \(\displaystyle b := \lim_{n \to \infty}b_n\text{.}\) By definition of the limit (Definition 1.23), we are required to prove that
    \begin{equation} \forall \eps \gt 0\ \exists N = N(\eps)\ \forall n \geq N \text{ we have } \abs{(a+b) - (a_n+b_n)} \leq \eps .\tag{2.3} \end{equation}
    Since the limits \(\displaystyle \lim_{n \to \infty} a_n\) and \(\displaystyle \lim_{n \to \infty} b_n\) both exist, we know that
    \begin{align} \amp \forall \eps \gt 0\ \exists N_1 = N_1(\eps)\ \forall n \geq N_1 \text{ we have } \abs{a - a_n} \leq \eps,\tag{2.4}\\ \amp \forall \eps \gt 0\ \exists N_2 = N_2(\eps)\ \forall n \geq N_2 \text{ we have } \abs{b - b_n} \leq \eps .\tag{2.5} \end{align}
    We have the key identity
    \begin{equation*} (a+b) - (a_n+b_n) = (a- a_n)+(b - b_n). \end{equation*}
    Notice that, by the triangle inequality, if \(|a - a_n| \leq \eps\) and \(|b - b_n| \leq \eps\) then
    \begin{equation*} \abs{(a+b) - (a_n+b_n)} = \abs{(a- a_n)+(b - b_n)} \leq \abs{a- a_n}+\abs{b - b_n}\leq 2\eps. \end{equation*}
    Writing \(N_1(\eps)\) and \(N_2(\eps)\) for the functions appearing in (2.4) and (2.5), set
    \begin{equation*} N_0(\eps) := \max\set{N_1(\eps), N_2(\eps)}. \end{equation*}
    Then we have proved that
    \begin{equation*} \forall \eps \gt 0\ \exists N_0 = N_0(\eps)\ \forall n \geq N_0 \text{ we have } \abs{(a+b) - (a_n+b_n)} \leq 2\eps . \end{equation*}
    Taking \(N(\eps) := N_0(\eps/2)\) then yields (2.3).
    Example 2.15. An example of using the addition rule.
    If \(a_n\to a\) as \(n\to \infty\) then
    \begin{equation*} a_n + \frac{1}{n} \to a + 0 = a\quad \text{as} \quad n \to \infty. \end{equation*}

    - 2.16. Multiplication rule.

    State the "multiplication rule" for limits of convergent sequences (the proof is non-examinable).
    Solution.
    Proposition 2.17. Multiplication rule.
    Let \((a_n)_{n=1}^\infty\) and \((b_n)_{n =1}^\infty\) be convergent sequences of real numbers. Then
    \begin{equation*} \lim_{n \to \infty} (a_n\cdot b_n) = \brac{\lim_{n \to \infty} a_n}\cdot \brac{\lim_{n \to \infty} b_n}. \end{equation*}
    Non-examinable proof.
    Let \(\displaystyle a : = \lim_{n \to \infty}a_n\) and \(\displaystyle b := \lim_{n \to \infty}b_n\text{.}\) By definition of the limit (Definition 1.23), we wish to prove that
    \begin{equation*} \forall \eps \gt 0\ \exists N = N(\eps)\ \forall n \geq N \text{ we have } |ab-a_nb_n| \leq \eps. \end{equation*}
    Since the limits \(\displaystyle \lim_{n \to \infty} a_n\) and \(\displaystyle \lim_{n \to \infty} b_n\) both exist and are finite, we know that
    \begin{align} \amp\forall \eps \gt 0\ \exists N_1 = N_1(\eps)\ \forall n \geq N_1 \text{ we have } \abs{a - a_n} \leq \eps,\tag{2.6}\\ \amp\forall \eps \gt 0\ \exists N_2 = N_2(\eps)\ \forall n \geq N_2 \text{ we have } \abs{b - b_n} \leq \eps .\tag{2.7} \end{align}
    We observe the key identity
    \begin{equation*} ab - a_nb_n = ab-a_nb +a_nb-a_nb_n = (a-a_n)b + a_n(b - b_n). \end{equation*}
    Since convergent sequences are bounded (Proposition 1.30), there exists \(M\in\R\) such that for all \(n\) we have \(|a_n| \leq M\text{.}\) Thus if \(|a - a_n| \leq \eps\) and \(|b - b_n| \leq \eps\) then
    \begin{equation*} |ab - a_nb_n| \leq |a-a_n||b| + M|b - b_n| \leq (|b|+M)\eps. \end{equation*}
    Writing \(N_1(\eps)\) and \(N_2(\eps)\) for the functions appearing in (2.6) and (2.7), set
    \begin{equation*} N(\eps) := \max\set{N_1\brac{\tfrac{\eps}{|b|+M}}, N_2\brac{\tfrac{\eps}{|b|+M}}}. \end{equation*}
    Then we have proved that
    \begin{equation*} \forall \eps \gt 0\ \exists N = N(\eps)\ \forall n \geq N \text{ we have } \abs{ab - a_nb_n} \leq \brac{|b|+M}\frac{\eps}{|b|+M} = \eps . \end{equation*}
    Thus \(a_nb_n\to ab\text{.}\)

    - 2.18. Non-zero limit gives a non-zero sequence, eventually.

    State and prove how a convergent sequence with non-zero limit has all terms non-zero from some point onwards.
    Solution.
    Proposition 2.19.
    Let \((a_n)_{n=1}^\infty\) be a convergent sequence with \(\lim_{n\to\infty} a_n \neq 0\text{.}\) Then there exists \(N \in \N\) such that for all \(n\geq N\) we have \(a_n\neq 0\text{.}\)
    Furthermore, if \(\lim_{n\to\infty} a_n \gt 0\) then the terms are (eventually) uniformly bounded away from zero, in that
    \begin{equation*} (\exists \eps \gt 0)(\exists N \in \N)(\forall n\geq N)[a_n\geq \eps]\text{.} \end{equation*}
    Proof.
    Set \(a:= \lim_{n\to\infty} a_n\text{.}\) Taking \(\eps := |a|/2 \gt 0\) in the definition of convergence (Definition 1.23), there exists \(N = N(\eps)\) such that for all \(n\geq N\) we have \(a_n \in [a-\eps, a+\eps]\text{.}\) We claim that this neighbourhood of \(a\) does not contain 0. We prove this by splitting into cases, depending on whether \(a\gt 0\) or \(a\lt 0\text{.}\) We give details for the former case and leave the latter case to the reader.
    Supposing that \(a\gt 0\text{,}\) we have that
    \begin{equation*} a-\eps = a-\frac{|a|}{2}=a-\frac{a}{2} = \frac{a}{2} = \eps > 0. \end{equation*}
    Therefore for all \(n\geq N(\eps)\) we have \(a_n \geq \eps > 0\text{,}\) proving the both claims made in Proposition 2.19 in this case.

    - 2.20. Reciprocal rule.

    State the "reciprocal rule" for sequence limits (the proof is not examinable).
    Solution.
    Proposition 2.21. Reciprocal rule.
    Let \((a_n)_{n=1}^\infty\) be a sequence of real numbers which converge to a non-zero limit. Then
    \begin{equation} \lim_{n \to \infty} \brac{\frac{1}{a_n}} = \frac{1}{\displaystyle \lim_{n\to \infty} a_n}. \tag{2.8} \end{equation}
    Remark 2.22. No need to assume all terms non-zero.
    Strictly speaking, (2.8) only makes sense if we know that \(a_n \neq 0\) for all \(n \in \N\text{,}\) so that the reciprocal \(1/a_n\) is well-defined for all values of \(n\text{.}\) However, the limiting value of a sequence does not depend on a finite number of initial terms: if you alter a finite number of terms of a sequence, you do not change the value of the limit. When the limit is non-zero, Proposition 2.19 tells us that \(a_n = 0\) for only a finite number of \(n\text{,}\) so we may discard a finite number of initial terms in order for the expression (2.8) to make sense.
    Non-examinable proof.
    Let \(\displaystyle a : = \lim_{n \to \infty}a_n\text{.}\) We wish to prove that
    \begin{equation} \forall \eps \gt 0\ \exists N = N(\eps)\ \forall n \geq N \text{ we have } |a^{-1}-a_n^{-1}| \leq \eps.\tag{2.9} \end{equation}
    We observe the key identity
    \begin{equation*} a^{-1} - a_n^{-1} = \frac{a_n-a}{aa_n}. \end{equation*}
    By definition of the limit (Definition 1.23), we know that
    \begin{equation*} \forall \eps \gt 0\ \exists N_1 = N_1(\eps)\ \forall n \geq N_1 \text{ we have } \abs{a - a_n} \leq \eps. \end{equation*}
    By the triangle inequality
    \begin{equation*} |a_n| = |a-(a-a_n)| \geq |a| - |a-a_n|. \end{equation*}
    Hence if \(|a-a_n| \leq |a|/2\) then \(|a_n| \geq |a|/2 \gt 0\text{.}\) Setting \(N_0 := N_1(|a|/2)\text{,}\) it follows that for all \(n \geq N_0\) we have \(|a_n| \geq |a|/2 \gt 0\text{,}\) in particular we have
    \begin{equation*} \abs{a^{-1} - a_n^{-1}} = \frac{|a_n-a|}{|aa_n|} \leq \frac{|a_n-a|}{|a|^2/2}\text{.} \end{equation*}
    Hence if \(n \geq N_0\) and \(|a - a_n| \leq \eps\) then
    \begin{equation*} \abs{a^{-1} - a_n^{-1}} = \frac{|a_n-a|}{|aa_n|} \leq \frac{\eps}{|a|^2/2} = \frac{2\eps}{|a|^2}. \end{equation*}
    We therefore obtain (2.9) on setting
    \begin{equation*} N(\eps) := \max\set{N_0, N_1\brac{\trecip{2}|a|^{2}\eps}}. \end{equation*}
