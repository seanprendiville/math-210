Section 9 Week 9

Lecture 9.1. Linearity of integration.

(a)

State and prove how suprema/infima of functions behave under scalar multiplication.
Solution.
Lemma 9.2. Scaling rule for suprema and infima.
Let \(f : S \to \R\) be a bounded function on an arbitrary non-empty set \(S\text{.}\) Let \(c \in \R\text{.}\) Then:
\begin{equation*} \sup_{S} \brac{cf} = \begin{cases} \displaystyle c \sup_{ S} f \amp \text{ if } c \geq 0, \\ \displaystyle c \inf_{ S} f \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Similarly
\begin{equation*} \inf_{ S} \brac{cf} = \begin{cases} \displaystyle c \inf_{ S} f \amp \text{ if } c \geq 0, \\ \displaystyle c \sup_{ S} f \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Proof.

If \(c \gt 0\) then both claims follows from the fact that

\begin{equation*} \set{M : M \text{ is an upper bound for } cf} = \set{cM : M \text{ is an upper bound for } f}, \end{equation*}

and similarly for lower bounds.

For \(c \lt 0\) we use the result for \(c\gt0\) as follows

\begin{equation*} \sup_{S} \brac{cf} = \sup_{S} \brac{-(-c)f} = -\inf_S \brac{(-c) f} = -(-c)\inf_S f = c\inf_S f, \end{equation*}

and similarly for the infimum.

(b)

State and prove how the Riemann integral respects scaling.
Solution.
Theorem 9.3. The Riemann integral scales appropriately.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f: [a,b] \to \R\) be Riemann integrable on \([a,b]\text{.}\) Let \(c \in \R\text{.}\) Then \(cf\) is Riemann integrable on \([a,b]\text{.}\) Moreover
\begin{equation*} \int_a^b cf = c\int_a^b f. \qquad \text{[scaling rule]} \end{equation*}
Proof.
We first employ the scaling rule for suprema/infima (Lemma 9.2) to deduce that
\begin{equation*} U(cf;x_0,\dots, x_N) = \begin{cases} c U(f;x_0,\dots, x_N) \amp \text{ if } c \geq 0,\\ c L(f;x_0,\dots, x_N) \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Similarly
\begin{equation*} L(cf;x_0,\dots, x_N) = \begin{cases} c L(f;x_0,\dots, x_N) \amp \text{ if } c \geq 0,\\ c U(f;x_0,\dots, x_N) \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Employing the scaling rule (Lemma 9.2) a second time, as we take sup/inf over all subdivisions, we obtain
\begin{equation*} U(cf;[a,b]) = \begin{cases} c U(f;[a,b]) \amp \text{ if } c \geq 0,\\ c L(f;[a,b]) \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Similarly
\begin{equation*} L(cf;[a,b]) = \begin{cases} c L(f;[a,b]) \amp \text{ if } c \geq 0,\\ c U(f;[a,b]) \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Regardless of whether \(c \geq 0\) or \(c \lt 0\text{,}\) the fact that \(U(f;[a,b]) = L(f;[a,b])\) gives that
\begin{equation*} U(cf;[a,b]) = L(cf;[a,b]) = c\int_a^b f. \end{equation*}

(c)

State and prove how suprema/infima of functions behave under addition.
Solution.
Lemma 9.4. Subadditivity of sup and inf.
\(f,g : S \to \R\)\(S\text{.}\)
\begin{align*} \sup_{ S} \brac{f + g}\amp \leq \sup_{ S} f +\sup_{ S} g;\\ \inf_{ S} \brac{f + g} \amp \geq \inf_{ S} f +\inf_{ S} g. \end{align*}
Proof.
If \(x_0 \in S\) then we have that
\begin{equation*} f(x_0) \leq \sup_{ S} f\quad \text{and} \quad g(x_0) \leq \sup_{S} g. \end{equation*}
Hence the sum
\begin{equation*} \sup_{S} f +\sup_{ S} g \end{equation*}
is an upper bound for the set
\begin{equation*} \set{f(x_0) + g(x_0) : x_0 \in S}. \end{equation*}
By definition of the supremum as the least upper bound (Definition 1.5) we deduce the subadditivity of the supremum. The argument for the infimum is analogous.

(d)

State and prove how Riemann sums behave under addition of functions.
Solution.
Corollary 9.5. Subadditivity of Riemann sums.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f,g : [a,b] \to \R\) be bounded. Let \(a = x_0 \lt \dots \lt x_N = b\) be a subdivision of \([a,b]\text{.}\) Then
\begin{equation*} U(f+g; x_0, \dots, x_N) \leq U(f;x_0, \dots, x_N) + U(g;x_0, \dots, x_N) \end{equation*}
and
\begin{equation*} L(f+g; x_0, \dots, x_N) \geq L(f;x_0, \dots, x_N) + L(g;x_0, \dots, x_N) . \end{equation*}
Proof.
This is a direct consequence of sub-additivity of suprema and infima (Lemma 9.4).

(e)

State and prove how the Riemann integral respects addition.
Solution.
Theorem 9.6. Additivity of integration.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f,g : [a,b] \to \R\) be functions which are Riemann integrable on \([a,b]\text{.}\) Then \(f+g\) is Riemann integrable on \([a,b]\text{.}\) Moreover
\begin{equation*} \int_a^b\brac{f+g}= \int_a^bf+\int_a^bg\qquad\text{[addition rule]}\text{.} \end{equation*}
Proof.
Fix \(\eps \gt 0\text{.}\) By the epsilon characterisation of Riemann integrability, (Proposition 8.23) there exists a subdivision \(a= y_0 \lt \dots \lt y_M = b\) such that
\begin{equation*} U(f;y_0, \dots, y_M) -\trecip{2}\eps \leq \int_a^b f \leq L(f;y_0, \dots, y_M)+\trecip{2}\eps. \end{equation*}
Similarly there exists a subdivision \(a= z_0 \lt \dots \lt z_T = b\) such that
\begin{equation*} U(g;z_0, \dots, z_T) -\trecip{2}\eps \leq \int_a^b g \leq L(g;z_0, \dots, z_T)+\trecip{2}\eps. \end{equation*}
Let \(a= x_0 \lt \dots \lt x_N = b\) denote the common refinement of \((y_0, \dots, y_M)\) and \((z_0, \dots, z_T)\text{.}\) Then by monotonicty of Riemann sums under refinements (Proposition 8.21) we have
\begin{equation*} U(f;x_0, \dots, x_N) -\trecip{2}\eps \leq \int_a^b f \leq L(f;x_0, \dots, x_N)+\trecip{2}\eps \end{equation*}
and
\begin{equation*} U(g;x_0, \dots, x_N) -\trecip{2}\eps \leq \int_a^b g \leq L(g;x_0, \dots, x_N)+\trecip{2}\eps. \end{equation*}
Adding inequalities together and employing the subadditivity of Riemann sums (Corollary 9.5) we deduce that
\begin{equation*} U(f+g;x_0, \dots, x_N) -\eps \leq \int_a^b f + \int_a^b g \leq L(f+g;x_0, \dots, x_N) +\eps. \end{equation*}
Thus
\begin{equation*} U(f+g;[a,b]) -\eps \leq \int_a^b f + \int_a^b g \leq L(f+g;[a,b]) +\eps. \end{equation*}
Since this holds for all \(\eps \gt 0\text{,}\) we conclude that
\begin{equation*} U(f+g;[a,b]) \leq \int_a^b f + \int_a^b g \leq L(f+g;[a,b]). \end{equation*}
Since lower Riemann sums are always bounded above by upper Riemann sums (8.6) we have \(L(f+g; [a,b]) \leq U(f+g;[a,b])\text{,}\) so we must have
\begin{equation*} U(f+g;[a,b]) = \int_a^b f + \int_a^b g = L(f+g;[a,b]). \end{equation*}
This establishes the addition rule.

(f)

Deduce how the Riemann integral respects pointwise inequalities.
Solution.
Corollary 9.7. Inequality rule.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f,g : [a,b] \to \R\) be functions which are Riemann integrable on \([a,b]\text{.}\) Suppose that \(f \leq g\) everywhere on \([a,b]\text{.}\) Then
\begin{equation*} \int_a^bf\leq \int_a^bg. \end{equation*}
Proof.
By the addition and scaling rules (Theorem 9.3 and Theorem 9.6) the function \(g-f\) is Riemann integrable. Using the addition and scaling rules, and the trivial lower bound for the integral (8.6), we have
\begin{equation*} \int_a^bg - \int_a^bf =\int_a^b(g-f) \geq (b-a)\inf_{[a,b]} (g-f). \end{equation*}
Since \(g-f\geq 0\) on \([a,b]\text{,}\) we have
\begin{equation*} (b-a)\inf_{[a,b]} (g-f) \geq 0. \end{equation*}
This yields the required inequality.

Lecture 9.8. Riemann integrable functions are closed under multiplication.

(a)

Define the positive and negative part of a real number.
Solution.
Definition 9.9. Positive and negative parts.
Given a real number \(x \in \R\) define the positive part by
\begin{equation*} x^+ := \max\set{0, x}. \end{equation*}
Using this notation, define the negative part by
\begin{equation*} x^-:= -\min\set{x,0} = |\min\set{x,0}| = (-x)^+. \end{equation*}
One may check that
\begin{equation} x = x^+ - x^- \quad\text{and}\quad |x| = x^+ + x^-.\tag{9.1} \end{equation}

(b)

State and prove how being Riemann integrable is preserved under taking positive parts.
Solution.
Lemma 9.10. Integrability is preserved under taking positive parts.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f : [a,b] \to \R\) be Riemann integrable. Then \(f^+\) is Riemann integrable on \([a,b]\text{:}\)
Proof.
Approximating the integral with a Riemann sum (see Proposition 8.23), for any \(\eps \gt 0\) there exists a subdivision \(a = x_0 \lt \dots \lt x_N = b\) such that
\begin{equation*} U(f;x_0, \dots, x_N)-L(f;x_0, \dots, x_N) \leq \eps. \end{equation*}
We claim that
\begin{align} U(f^+;x_0, \dots, x_N)- \amp L(f^+;x_0, \dots, x_N)\notag\\ \amp \leq U(f;x_0, \dots, x_N)- L(f;x_0, \dots, x_N),\tag{9.2} \end{align}
from which it follows that \(f^+\) is integrable by the existence of arbitrarily close upper and lower Riemann sums (Proposition 8.23). To establish our claim (9.2), we first observe that for \(x \geq 0\) and \(y \leq 0\) we have
\begin{equation} |x^+-y^+| = |x| = x \leq x+(-y) = |x-y|.\tag{9.3} \end{equation}
Checking the remaining cases, one may verify that \(|x^+-y^+| \leq |x-y|\) for any choice of \(x, y \in \R\text{.}\) Since decreasing a supremum gives a non upper bound (see Proposition 1.7), for any \(\eps \gt 0\) there exist \(y \in [x_{n-1}, x_n]\) such that
\begin{equation*} \sup_{ [x_{n-1}, x_n]} f^+ - \eps \leq f^+(y) \leq \sup_{ [x_{n-1}, x_n]} f^+. \end{equation*}
Similarly there exists \(z \in [x_{n-1}, x_n]\) such that
\begin{equation*} \inf_{ [x_{n-1}, x_n]} f^+ \leq f^+(z) \leq \inf_{ [x_{n-1}, x_n]} f^+ + \eps. \end{equation*}
Hence by the triangle inequality and (9.3) we have
\begin{align*} \sup_{ [x_{n-1}, x_n]} f^+ - \inf_{ [x_{n-1}, x_n]} f^+ \amp \leq |f^+(y) - f^+(z)| +2\eps\\ \amp\leq |f(y)-f(z)| + 2\eps\\\\ \amp\leq \sup_{ [x_{n-1}, x_n]} f - \inf_{ [x_{n-1}, x_n]} f+2\eps. \end{align*}
Since this holds for arbitrary \(\eps \gt 0\text{,}\) we deduce (9.2).

(c)

State and prove how being Riemann integrable is preserved under taking min, max and absolute values.
Solution.
Proposition 9.11. Integrability is preserved under taking min, max and absolute values.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f,g : [a,b] \to \R\) be functions which are Riemann integrable on \([a,b]\text{.}\) Then the following functions are all Riemann integrable on \([a,b]\text{:}\)
\begin{equation*} \max\set{f,g}, \qquad\min\set{f,g},\qquad f^+,\qquad f^-,\qquad |f|. \end{equation*}
Moreover we have the integral triangle inequality
\begin{equation*} \abs{\int_a^bf}\leq \int_a^b|f|. \end{equation*}
Proof.
We have \(\max\set{f, g} = (f-g)^+ + g\text{.}\) This sum is Riemann integrable by linearity of integration (Theorem 9.6). Since \(\min\set{f,g} = -\max\set{-f,-g}\text{,}\) we deduce that this is also Riemann integrable using the scaling rule (Theorem 9.6). Similarly, we have Riemann integrability of \(f^- = -\max\set{-f, 0}\text{.}\) Finally \(|f| = f^+ + f^-\) is integrable by the previous results and the addition rule (Theorem 9.6). For the triangle inequality, linearity of integration (Theorem 9.6) gives
\begin{equation*} \abs{\int_a^b f} = \abs{\int_a^b f^+ - \int_a^b f^-} \leq \int_a^b f^+ + \int_a^b f^- = \int_a^b\brac{ f^+ + f^-} = \int_a^b\abs{ f} . \end{equation*}

(d)

State and prove the sub-multiplicativity of suprema/infima.
Solution.
Lemma 9.12. Sub-multiplicativity of \(\sup\) and \(\inf\).
Let \(f,g : S \to [0,\infty)\) be bounded non-negative functions on an arbitrary set \(S\text{.}\) Then:
\begin{align*} \sup_{ S} \brac{f \cdot g} \amp\leq \sup_{ S} f \cdot \sup_{ S} g;\\ \inf_{ S} \brac{f \cdot g}\amp \geq \inf_{ S} f \cdot \inf_{ S} g. \end{align*}
Proof.
The argument is analogous to the proof of sub-additivity (Lemma 9.4) and is left to the reader.

(e)

State and prove how the set of Riemann integrable functions is closed under multiplication.
Solution.
Theorem 9.13. Riemann integrable functions are closed under multiplication.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f,g : [a,b] \to \R\) be functions which are Riemann integrable on \([a,b]\text{.}\) Then \(f\cdot g\) is Riemann integrable on \([a,b]\text{.}\)
Proof.
Splitting into positive and negative parts, we have that
\begin{equation*} f g = (f^+-f^-)(g^+-g^-) = f^+g^+ + f^-g^- - f^+g^- - f^-g^+. \end{equation*}
Using linearity of integration (Theorem 9.6) and the fact that integrability is preserved under taking positive/negative parts (Proposition 9.11), it therefore suffices to establish that \(f g\) is Riemann integrable when \(f\) and \(g\) are non-negative. Let us therefore assume that for all \(x\in [a,b]\) we have \(f(x)\geq 0\) and \(g(x) \geq 0\text{.}\) Write
\begin{equation*} S := \sup_{ [a,b]} f\quad \text{and}\quad T:= \sup_{ [a,b]} g. \end{equation*}
Fix \(\eps \gt 0\text{.}\) Since a Riemann integral can be approximated by Riemann sums (Proposition 8.23) there exists a subdivision \(a= y_0 \lt \dots \lt y_M = b\) such that
\begin{equation*} U(f;y_0, \dots, y_M) -\trecip{2}\eps \leq \int_a^b f \leq L(f;y_0, \dots, y_M)+\trecip{2}\eps. \end{equation*}
Similarly there exists a subdivision \(a= z_0 \lt \dots \lt z_T = b\) such that
\begin{equation*} U(g;z_0, \dots, z_T) -\trecip{2}\eps \leq \int_a^b g \leq L(g;z_0, \dots, z_T)+\trecip{2}\eps. \end{equation*}
Let \(a= x_0 \lt \dots \lt x_N = b\) denote the common refinement of \((y_0, \dots, y_M)\) and \((z_0, \dots, z_T)\text{.}\) Then by monotonicty of Riemann sums under refinements (Proposition 8.21) we have
\begin{equation*} U(f;x_0, \dots, x_N) -\trecip{2}\eps \leq \int_a^b f \leq L(f;x_0, \dots, x_N)+\trecip{2}\eps \end{equation*}
and
\begin{equation*} U(g;x_0, \dots, x_N) -\trecip{2}\eps \leq \int_a^b g \leq L(g;x_0, \dots, x_N)+\trecip{2}\eps. \end{equation*}
To ease notation, write
\begin{gather*} s_i := \inf_{ [x_{i-1}, x_i]} f, \quad S_i := \sup_{[x_{i-1}, x_i]} f,\\ t_i := \inf_{ [x_{i-1}, x_i]} g, \quad T_i := \sup_{[x_{i-1}, x_i]} g. \end{gather*}
Then for each \(i\) we have
\begin{equation*} S_iT_i - s_it_i = (S_i-s_i)T_i + s_i(T_i-t_i) \leq (S_i-s_i)T + S(T_i-t_i). \end{equation*}
By sub-multiplicativity of \(\sup\) and \(\inf\) (Lemma 9.12) we see that
\begin{equation*} \sup_{ [x_{i-1}, x_i]} (fg)\leq S_i T_i \quad \text{and}\quad \inf_{ [x_{i-1}, x_i]} (fg) \geq s_i t_i. \end{equation*}
Hence
\begin{align*} \amp U(fg;x_0, \dots, x_N) - L(fg;x_0,\dots, x_N) \leq \sum_{i=1}^N \brac{S_iT_i - s_it_i} (x_i-x_{i-1})\\ \amp \leq T \sum_{i=1}^N \brac{S_i - s_i} (x_i-x_{i-1}) + S \sum_{i=1}^N \brac{T_i - t_i} (x_i-x_{i-1}) \leq \eps. \end{align*}

Lecture 9.14. Integrating on a sub-interval.

(a)

Define an indicator function.
Solution.
Definition 9.15. Indicator function.
Given a set \(S\subset \R\) we define the indicator function \(1_S : \R \to \R\) by
\begin{equation*} 1_S(x) := \begin{cases} 1 \amp \text{ if } x \in S;\\ 0 \amp \text{ if } x \notin S. \end{cases} \end{equation*}

(b)

Prove that the indicator function of a set with one element is Riemann integrable.
Solution.
Example 9.16. Indicator of a point.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(c \in (a,b)\) and consider the indicator function \(1_{\set{c}} :[a,b]\to\R\text{.}\) We will show that \(1_{\set{c}}\) is Riemann integrable on \([a,b]\) with integral 0. For any \(0 \lt \eps \lt \min\set{c-a, b-c}\) we have a subdivision \(a \lt c-\eps \lt c+\eps \lt b\) satisfying
\begin{equation*} U(f;[a,b]) \leq U(1_{\set{c}};a , c-\eps ,c+\eps , b) = 2\eps. \end{equation*}
Letting \(\eps \to 0\text{,}\) we deduce that \(U(1_{\set{c}}; [a,b]) =0\text{.}\) Since \(1_{\set{c}}\) is non-negative everywhere, it follows that
\begin{equation*} 0 \leq L(1_{\set{c}}; [a,b]) \leq U(1_{\set{c}}; [a,b]) =0. \end{equation*}
Thus
\begin{equation} \int_a^b 1_{\set{c}}= 0.\tag{9.4} \end{equation}
A similar argument yields (9.4) in the case when \(c=a\) or \(c = b\text{.}\)

(c)

State and prove how altering a function on a finite set does not affect the value of its integral.
Solution.
Proposition 9.17. Integrals agree up to finite point sets.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f,g : [a,b] \to \R\) be functions which are Riemann integrable on \([a,b]\text{.}\) Suppose that \(f\) and \(g\) differ at only finitely many points, so that the set
\begin{equation*} \set{x \in [a,b] : f(x) \neq g(x)} \end{equation*}
is finite. Then
\begin{equation*} \int_a^b f = \int_a^b g. \end{equation*}
Proof.
Our hypothesis implies that there exist finitely many points \(x_1, \dots, x_n \in [a,b]\) and \(c_1, \dots, c_n \in \R\) such that
\begin{equation*} f-g = \sum_{i=1}^n c_i 1_{\set{x_i}}. \end{equation*}
By linearity of the integral we deduce that
\begin{equation*} \int_a^b f - \int_a^b g = \sum_{i=1}^n c_i \int_a^b1_{\set{x_i}} =\sum_{i=1}^n c_i \cdot 0 = 0. \end{equation*}

(d)

State and prove how the indicator function of an interval is Riemann integrable.
Solution.
Proposition 9.18. Indicator of an interval.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(c, d \in [a,b]\) with \(c \leq d\text{.}\) For any interval
\begin{equation*} I \in \set{(c,d), [c,d), (c,d], [c,d]} \end{equation*}
we have
\begin{equation*} \int_a^b 1_I = d- c = \mathrm{Length}(I). \end{equation*}
Proof.
We prove the result when \(a \lt c\lt d\lt b\text{,}\) the remaining cases being similar. We may also assume that \(I = (c,d)\text{,}\) since we can change the value of a function on a finite set of points without affecting the integral (Proposition 9.17). Consider the subdivision \(a \lt c \lt d \lt b\text{.}\) Then
\begin{equation*} U(1_{(c,d)}; a,c,d,b) = (c-a)\cdot 0 + (d-c)\cdot 1 + (b-d)\cdot 0 = d-c . \end{equation*}
Similarly
\begin{equation*} L(1_{(c,d)}; a,c,d,b) = d-c. \end{equation*}
It therefore follows that \(1_{(c,d)}\) is Riemann integrable on \([a,b]\) with integral \(d-c\text{.}\)

(e)

State and prove how the Riemann integral behaves on restricting to a sub-interval.
Solution.
Proposition 9.19. Restricting to a sub-interval preserves integrability.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(c, d \in [a,b]\) with \(c \lt d\text{.}\) Let \(f: [a,b] \to \R\) be functions which is Riemann integrable on \([a,b]\text{.}\) Then \(f\) is also Riemann integrable on \([c,d]\) and
\begin{equation*} \int_c^d f = \int_a^b f1_{[c,d]}. \end{equation*}
Furthermore, if \(c \in (a,b)\) then we have the interval additivity property
\begin{equation} \int_a^b f = \int_a^c f+ \int_c^b f.\tag{9.5} \end{equation}
Proof.

Since products of Riemann integrable functions are integrable (Theorem 9.13) and indicators of intervals are integrable (Proposition 9.18), the function \(f1_{(c,d)}\) is Riemann integrable on \([a, b]\text{.}\) By Riemann sum approximation (Proposition 8.23), for any \(\eps \gt 0\) there exists a subdivision \(a = x_0 \lt \dots \lt x_N = b\) such that

\begin{equation} U(f1_{(c,d)}; x_0, \dots, x_N) - \eps \leq \int_a^bf1_{(c,d)} \leq L(f1_{(c,d)}; x_0, \dots, x_N) + \eps.\tag{9.6} \end{equation}

Since refining a subdivision does not increase the upper sum and does not decrease the lower sum (Proposition 8.21), we may assume that the subdivision \(a = x_0 \lt \dots \lt x_N = b\) contains \(c\) and \(d\text{.}\) Let \(m \lt n\) be such that \(x_m = c\) and \(x_n = d\text{.}\) Notice that \(c= x_m \lt \dots \lt x_n = d\) is a subdivision of \([c,d]\text{.}\) Moreover

\begin{gather*} U(f1_{(c,d)}; x_0, \dots, x_N) = U(f1_{(c,d)};x_n, \dots, x_m),\\ L(f1_{(c,d)}; x_0, \dots, x_N) = L(f1_{(c,d)};x_n, \dots, x_m). \end{gather*}

Thus for every \(\eps \gt 0\) we have

\begin{equation*} U(f1_{(c,d)}; [c,d])-\eps \leq \int_a^b f1_{(c,d)} \leq L(f1_{(c,d)}; [c,d])+\eps \end{equation*}

Since this holds for every \(\eps \gt 0\text{,}\) we deduce that \(f1_{(c,d)}\) is Riemann integrable on \([c,d]\) with

\begin{equation*} \int_c^df1_{(c,d)} = \int_a^b f1_{(c,d)} \end{equation*}

Since altering a function on a finite set of points does not affect the integral (Proposition 9.17) we have

\begin{equation*} \int_a^b f1_{[c,d]} = \int_a^b f1_{(c,d)} = \int_c^d f1_{(c,d)} = \int_c^d f . \end{equation*}

To deduce (9.5) we use the above, together with linearity of integration (Theorem 9.6) and the fact that indicators of points have zero integral (Proposition 9.17) to yield

\begin{align*} \int_a^b f \amp = \int_a^b f(1_{[a,c)} + 1_{[c,b]}) = \int_a^b f1_{[a,c)} + \int_a^b1_{[c,b]}\\ \amp = \int_a^b f1_{[a,c]} + \int_a^b1_{[c,b]} = \int_a^c f + \int_c^bf. \end{align*}

Lecture 9.20. Classes of Riemann integrable functions.

(a)

State and prove how a monotone function on a closed interval is integrable.
Solution.
Proposition 9.21. Monotone functions are integrable.
Let \(a, b\in \R\) with \(a \lt b\text{.}\) Let \(f : [a,b] \to \R\) be a monotone function. Then \(f\) is Riemann integrable on \([a,b]\text{.}\)
Proof.
Let us suppose that \(f\) is increasing and that \([a,b] = [0,1]\text{,}\) the general case being similar. Since \(f\) is increasing, for any sub-interval \([c,d]\subset [0,1]\) we have
\begin{equation*} \sup_{[c,d]} f = f(d) \quad \text{and}\quad \inf_{[c,d]} f= f(c). \end{equation*}
In particular, \(f\) is bounded on \([0,1]\text{,}\) with \(\sup_{[0,1]} f = f(1)\) and \(\inf_{[0,1]}f= f(0)\text{.}\) For any \(N \in \N\) consider the subdivision
\begin{equation*} \frac{0}{N} \lt \frac{1}{N} \lt \frac{2}{N} \lt \dots \lt \frac{N}{N}. \end{equation*}
For this subdivision, we have
\begin{align*} U\brac{f; \tfrac{0}{N}, \tfrac{1}{N}, \dots, \tfrac{N}{N}} - \amp L\brac{f; \tfrac{0}{N}, \tfrac{1}{N}, \dots, \tfrac{N}{N}} \\ \amp = \sum_{n=1}^N \frac{f\brac{\frac{n}{N}}}{N} - \sum_{n=1}^{N} \frac{f\brac{\frac{n-1}{N}}}{N}\\ \amp = \frac{f\brac{1}}{N} - \frac{f\brac{0}}{N} . \end{align*}
Since the latter quantity tends to zero as \(N \to \infty\text{,}\) we have that \(f\) is integrable by Proposition 8.23.

(b)

State and prove the integral test for finite series.
Solution.
Proposition 9.22. Integral test.
Let \(a,b \in \Z\) with \(a \lt b\text{.}\) Let \(f : [a,b] \to \R\) be a decreasing function, hence Riemann integrable by Proposition 9.21. Then
\begin{equation*} f(a)+ \int_a^b f(n) \geq \sum_{n=a}^b f(n) \geq f(b)+ \int_a^b f(n). \end{equation*}
Proof.
Consider the subdivision of \([a,b]\) given by \(x_n := a + n\) for \(n = 0,1,\dots, b-a\text{.}\) Since \(f\) is decreasing, for each \(1\leq n\leq b-a\) we have
\begin{equation*} \sup_{[x_{n-1}, x_n]} f = f(x_{n-1})\quad \text{and}\quad \inf_{[x_{n-1}, x_n]} f = f(x_{n}). \end{equation*}
Therefore
\begin{align*} U(f;x_0, \dots, x_{b-a})\amp = \sum_{n=1}^{b-a} f(a+n-1),\\ L(f;x_0, \dots, x_{b-a}) \amp =\sum_{n=1}^{b-a} f(a+n). \end{align*}
Taking an infimum gives
\begin{equation*} \int_a^b f = U(f;[a,b]) \leq \sum_{n=a}^{b-1} f(n). \end{equation*}
Taking a supremum gives
\begin{equation*} \int_a^b f = L(f;[a,b]) \geq \sum_{n=a+1}^{b} f(n). \end{equation*}
The desired inequalities follow.

(c)

State and prove that a continuous function on a closed interval is integrable.
Solution.
Proposition 9.23. Continuous functions are integrable.
Let \(a, b\in \R\) with \(a \lt b\text{.}\) Let \(f : [a,b] \to \R\) be a continuous function. Then \(f\) is Riemann integrable on \([a,b]\text{.}\)
Proof.
Let us suppose that \([a,b] = [0, 1]\text{,}\) the general case being similar. Since \(f\) is a continuous function on a closed interval, it is uniformly continuous (Theorem 6.14). Hence by definition of uniform continuity (Definition 6.11):
\begin{equation*} (\forall \eps \gt 0)(\exists \delta = \delta(\eps) \gt 0)(\forall x,y\in [a,b]) [|x-y| \leq \delta \implies |f(x)-f(y)| \leq \eps]. \end{equation*}
Set
\begin{equation*} \delta := \delta\brac{\eps} \qquad \text{and} \qquad N := \ceil{\delta^{-1}}. \end{equation*}
Then for any \(n\) we have
\begin{equation*} \sup_{[\frac{n-1}{N}, \frac{n}{N}]} f - \inf_{[\frac{n-1}{N}, \frac{n}{N}]} f \leq \eps. \end{equation*}
Hence with respect to the subdivision \(0/N \lt 1/N \lt \dots \lt N/N\) we have
\begin{equation*} U\brac{f;\tfrac{0}{N}, \tfrac{1}{N} , \dots , \tfrac{N}{N}} - L\brac{f;\tfrac{0}{N}, \tfrac{1}{N} , \dots , \tfrac{N}{N}} \leq \sum_{n=1}^N \tfrac{\eps}{N} = \eps. \end{equation*}
It follows that \(f\) is integrable by Proposition 8.23.

(d)

Give an example of a bounded function on a closed interval which is not Riemann integrable. Justify this.
Solution.
Proposition 9.24.
The function \(1_{\Q \cap [0,1]}\) is not Riemann integrable on \([0,1]\text{.}\)
Proof.

As in Example 5.18, the key observation in establishing our claim is to note that every real number \(x_0\) has a sequence of rationals \(q_n \in \Q\) such that \(q_n \to x_0\text{.}\) For instance, one may take

\begin{equation*} q_n := \frac{\floor{nx_0}}{n} \in \left(x_0 -\frac{1}{n}, x_0\right]\text{,} \end{equation*}

and employ the two-sided sandwich rule (Proposition 2.6).

One consequence of this observation any non-empty open interval contains a rational number, since one may approximate the mid-point by a rational. It follows that if \(0 \leq x \lt y \leq 1\) then \((x,y)\) contains a rational \(q\in \Q\) and \((x+\sqrt{2},y+\sqrt{2})\) also contains a rational \(\tilde q \in \Q\text{.}\) Hence \((x,y)\) contains both a rational \(q\) and an irrational \(q-\sqrt{2}\text{.}\) It follows that

\begin{equation*} \sup_{[x,y]} 1_{\Q\cap [0,1]} = 1 \quad \text{and}\quad \inf_{[x,y]} 1_{\Q\cap [0,1]} = 0. \end{equation*}

Hence for any subdivision \(0 = x_0 \lt \dots \lt x_N = 1\) we have

\begin{equation*} U(f;x_0,\dots, x_N) = \sum_{n=1}^N (x_n-x_{n-1}) \sup_{[x_{n-1}, x_n]} 1_{\Q\cap [0,1]} = \sum_{n=1}^N (x_n-x_{n-1}) = 1. \end{equation*}

Similarly

\begin{equation*} L(f;x_0, \dots, x_N) = 0. \end{equation*}

Therefore for any subdivision we have

\begin{equation*} U(f;x_0,\dots, x_N) - L(f;x_0, \dots, x_N) \geq 1. \end{equation*}

By Proposition 8.23 the function \(1_{\Q\cap [0,1]}\) is not Riemann integrable.
