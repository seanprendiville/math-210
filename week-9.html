    Subsection 9.1 Self-test questions for lecture 17

    - 9.1. Definition of the Riemann integral.

    (a)
    Define a subdivision of a closed interval.
    Solution.
    Definition 9.2. Subdivision.
    Let \(a,b\in \R\) with \(a\lt b\text{.}\) A subdivision of the interval \([a,b]\) is a strictly increasing finite sequence of the form
    \begin{equation*} a= x_0 \lt x_1 \lt \dots \lt x_N = b \end{equation*}
    for some \(N \in \N\text{.}\) We sometimes denote such a subdivision by \((x_n)_{n=0}^N = (x_0, x_1, \dots, x_N)\text{.}\)
    (b)
    Define the upper/lower Riemann sum of a function with respect to a subdivision.
    Solution.
    Definition 9.3. Riemann sum.
    Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f : [a,b] \to \R\) be a bounded
     1 
    See Definition 1.2.
    function. Let \(a = x_0 \lt x_1\lt \dots \lt x_N = b\) be a subdivision of \([a,b]\text{.}\) We define the upper Riemann sum of \(f\) with respect to this subdivision by
    \begin{equation*} U(f; x_0, \dots, x_N) := \sum_{n=1}^N (x_n-x_{n-1}) \sup_{ [x_{n-1}, x_n]} f. \end{equation*}
    We define the lower Riemann sum of \(f\) with respect to this subdivision by
    \begin{equation*} L(f; x_0, \dots, x_N) := \sum_{n=1}^N (x_n-x_{n-1}) \inf_{ [x_{n-1}, x_n]} f. \end{equation*}
    Notice that both the upper and lower Riemann sums are finite real numbers, since the fact that \(f\) is bounded implies that
    \begin{align} -\infty \amp \lt (b-a)\inf_{[a,b]} f = \sum_{n=1}^N (x_n-x_{n-1}) \inf_{ [a,b]} f\notag\\ \amp \leq \sum_{n=1}^N (x_n-x_{n-1}) \inf_{ [x_{n-1}, x_n]} f = L(f; x_0, \dots, x_N)\notag\\ \amp\leq \sum_{n=1}^N (x_n-x_{n-1}) \sup_{ [x_{n-1}, x_n]} f = U(f; x_0, \dots, x_N)\notag\\ \amp \leq \sum_{n=1}^N (x_n-x_{n-1}) \sup_{ [a,b]} f \notag\\ \amp = (b-a)\sup_{[a,b]} f\lt \infty.\tag{9.1} \end{align}
    (c)
    Define the upper/lower Riemann integral of a function.
    Solution.
    Definition 9.4. Upper/lower Riemann integral.
    Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f : [a,b] \to \R\) be a bounded function. We define the upper Riemann integral of \(f\) to be the infimum of the upper Riemann sum over all possible subdivisions:
    \begin{equation*} U(f;[a,b]) := \inf\set{U(f;x_0,\dots, x_N): a= x_0 \lt \dots \lt x_N = b,\ N \in \N}. \end{equation*}
    We define the lower Riemann integral of \(f\) to be the supremum of the lower Riemann sum over all possible subdivisions:
    \begin{equation*} L(f;[a,b]) := \sup\set{L(f;x_0,\dots, x_N): a= x_0 \lt \dots \lt x_N = b,\ N \in \N}. \end{equation*}
    (d)
    Define what it means for a function to be Riemann integrable and define the Riemann integral.
    Solution.
    Definition 9.5. Riemann integrable and Riemann integral.
    Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f : [a,b] \to \R\) be a bounded function. We say that \(f\) is Riemann
     2 
    On occasion, we may say "\(f\) is integrable", by which we mean that \(f\) is Riemann integrable.
    integrable on \([a,b]\)} if the upper and lower Riemann integrals coincide, so that
    \begin{equation*} U(f;[a,b]) = L(f;[a,b]). \end{equation*}
    In this case we define the Riemann integral of \(f\) on \([a,b]\) to be this common value, denoted
    \begin{equation*} \int_a^b f \qquad \text{or} \qquad \int_a^b f(x) \intd x. \end{equation*}
    We also define
    \begin{equation*} \int_b^a f := -\int_a^b f. \end{equation*}

    - 9.6. Riemann sums are monotone with respect to refinements.

    (a)
    Define what it means to be a refinement of a subdivision.
    Solution.
    Definition 9.7. Refinement of a subdivision.
    Let \(a, b\in \R\) with \(a\lt b\text{.}\) We say that the subdivision \(a = y_0 \lt \dots \lt y_{M} = b\) is a refinement of the subdivision \(a = x_0 \lt \dots \lt x_{ N} = b\) if
    \begin{equation*} \set{x_0, x_1, \dots, x_N} \subset \set{y_0, y_1,\dots, y_{M}}. \end{equation*}
    Given subdivisions \(a = x_0 \lt \dots \lt x_{ N} = b\) and \(a = y_0 \lt \dots \lt y_{M} = b\) (not necessarily refinements of each other) their common refinement is the subdivision \(a = z_0 \lt \dots \lt z_T = b\) obtained by ordering the union
    \begin{equation*} \set{x_0, \dots , x_N} \cup \set{y_0, \dots, y_M} = \set{z_0, \dots, z_T}. \end{equation*}
    (b)
    State and prove how Riemann sums are monotone with respect to refinements.
    Solution.
    Proposition 9.8. Riemann sums are monotone with respect to refinements.
    Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f : [a,b] \to \R\) be a bounded function. Let \(a = y_0 \lt \dots \lt y_M = b\) be a refinement of \(a= x_0 \lt \dots \lt x_N = b\text{.}\) Then
    \begin{align*} L(f;x_0,\dots, x_N)\amp \leq L(f; y_0, \dots, y_M)\\ \amp \leq U(f; y_0, \dots, y_M) \leq U(f;x_0,\dots, x_N). \end{align*}
    Proof.
    Let \(x_1, x_2, x_3 \in [a,b]\) with \(x_1 \lt x_2 \lt x_3\text{.}\) Since taking a supremum over a larger set does not decrease the supremum, we have
    \begin{equation*} \sup_{t \in [x_1, x_2]} f(t) \leq \sup_{t \in [x_1, x_3]} f(t) \quad \text{and} \quad \sup_{t \in [x_2, x_3]} f(t) \leq \sup_{t \in [x_1, x_3]} f(t). \end{equation*}
    Therefore
    \begin{align*} \amp(x_2 - x_1) \sup_{t \in [x_1, x_2]} f(t) + (x_3 - x_2) \sup_{t \in [x_2, x_3]} f(t)\\ \amp\leq (x_2 - x_1) \sup_{t \in [x_1, x_3]} f(t) + (x_3 - x_2) \sup_{t \in [x_1, x_3]} f(t) = (x_3 - x_1) \sup_{t \in [x_1, x_3]} f(t). \end{align*}
    It follows that adding a point to a subdivision does not increase the value of the upper Riemann sum. Repeatedly applying this fact a finite number of times, we deduce that adding a finite number of points to a subdivision does not increase the value of the corresponding Riemann sum. This proves the inequality
    \begin{equation*} U(f; y_0, \dots, y_M) \leq U(f;x_0,\dots, x_N). \end{equation*}
    The argument for the inequality
    \begin{equation*} L(f;x_0,\dots, x_N)\leq L(f; y_0, \dots, y_M) \end{equation*}
    is similar (try it!). The inequality
    \begin{equation*} L(f; y_0, \dots, y_M) \leq U(f; y_0, \dots, y_M) \end{equation*}
    follows from the fact that
    \begin{equation*} \inf_{ [y_{n-1}, y_n]} f \leq \sup_{ [y_{n-1}, y_n]} f. \end{equation*}
    (c)
    Deduce how lower Riemann sums never exceed upper Riemann sums
    Solution.
    Corollary 9.9. Lower Riemann sums never exceed upper Riemann sums.
    Let \(a, b\in \R\) with \(a\lt b\) and let \(f : [a,b] \to \R\) be a bounded function. Then for any subdivisions \(a = y_0 \lt \dots \lt y_M = b\) and \(a= x_0 \lt \dots \lt x_N = b\) we have
    \begin{equation*} L(f;y_0, \dots, y_M)\leq U(f;x_0, \dots, x_N)\text{.} \end{equation*}
    In particular
    \begin{equation*} L(f;[a,b])\leq U(f;[a,b])\text{.} \end{equation*}
    Proof.
    Let \(a = z_0 \lt \dots \lt z_T = b\) be the common refinement of \(a = y_0 \lt \dots \lt y_M = b\) and \(a= x_0 \lt \dots \lt x_N = b\text{,}\) so that
    \begin{equation*} \set{x_0, \dots , x_N} \cup \set{y_0, \dots, y_M} = \set{z_0, \dots, z_T}\text{.} \end{equation*}
    Since Riemann sums are monotone with respect to refinements, and since \(\inf_{[z_{n-1},z_n]}f \leq \sup_{[z_{n-1},z_n]}f\text{,}\) we have
    \begin{align*} L(f;y_0, \dots, y_M) \leq L(f;z_0, \dots, z_T)\amp \leq U(f;z_0, \dots, z_T) \\ \amp \leq U(f; x_0, \dots, x_N)\text{.} \end{align*}
    Taking a supremum over the all possible \(a = y_0 \lt \dots \lt y_M = b\) gives that
    \begin{equation*} L(f;[a,b]) \leq U(f; x_0, \dots, x_N)\text{.} \end{equation*}
    Taking an infimum over all possible \(a= x_0 \lt \dots \lt x_N = b\) gives that
    \begin{equation*} L(f;[a,b]) \leq U(f; [a,b])\text{.} \end{equation*}

    - 9.10. Monotone functions are Riemann integrable.

    State and prove how a monotone function on a closed interval is integrable.
    Solution.
    Proposition 9.11. Monotone functions are integrable.
    Let \(a, b\in \R\) with \(a \lt b\text{.}\) Let \(f : [a,b] \to \R\) be a monotone function. Then \(f\) is Riemann integrable on \([a,b]\text{.}\)
    Proof.
    We are required to prove that \(f\) is bounded (so that the upper/lower Riemann sums/integrals are defined) and that \(L(f;[a,b]) = U(f;[a,b])\text{.}\) Let us suppose that \(f\) is increasing and that \([a,b] = [0,1]\text{,}\) the general case being similar.
    Since \(f\) is increasing, for any sub-interval \([c,d]\subset [0,1]\) we have
    \begin{equation*} \sup_{[c,d]} f = f(d) \quad \text{and}\quad \inf_{[c,d]} f= f(c). \end{equation*}
    In particular, since \(\sup_{[0,1]} f = f(1)\) and \(\inf_{[0,1]}f= f(0)\text{,}\) we see that \(f\) is bounded on \([0,1]\text{.}\)
    Next let us show that \(L(f;[a,b]) = U(f;[a,b])\text{.}\) For any \(N \in \N\) consider the subdivision
    \begin{equation*} \frac{0}{N} \lt \frac{1}{N} \lt \frac{2}{N} \lt \dots \lt \frac{N}{N}. \end{equation*}
    Then we have
    \begin{align*} 0\leq U(f;[a,b]) - L(f;[a,b])\amp \leq U\brac{f; \tfrac{0}{N}, \tfrac{1}{N}, \dots, \tfrac{N}{N}} - L\brac{f; \tfrac{0}{N}, \tfrac{1}{N}, \dots, \tfrac{N}{N}} \\ \amp = \sum_{n=1}^N \frac{f\brac{\frac{n}{N}}}{N} - \sum_{n=1}^{N} \frac{f\brac{\frac{n-1}{N}}}{N}\\ \amp = \frac{f\brac{1}}{N} - \frac{f\brac{0}}{N} . \end{align*}
    Since the latter quantity tends to zero as \(N \to \infty\text{,}\) we have that \(U(f;[a,b]) = L(f;[a,b])\text{.}\)

    Subsection 9.2 Self-test questions for lecture 18

    - 9.12. Linearity of integration.

    (a)
    State and prove how the Riemann integral respects addition and scaling.
    Solution.
    Theorem 9.13. Linearity of integration.
    Let \(a, b,c\in \R\) with \(a\lt b\text{.}\) Let \(f,g : [a,b] \to \R\) be functions which are Riemann integrable on \([a,b]\text{.}\) Then \(f+g\) and \(cf\) are both Riemann integrable on \([a,b]\text{.}\) Moreover
    \begin{equation*} \int_a^b\brac{f+g}= \int_a^bf+\int_a^bg \quad\text{and}\quad \int_a^b(cf) = c\int_a^bf\text{.} \end{equation*}
    Proof.
    Let us proof the claim about addition (the claim about scaling follows a similar argument). Fix \(\eps \gt 0\text{.}\) Since increasing an infimum gives a non-lower bound, there exist subdivisions \(a= x_0 \lt \dots \lt x_N = b\) and \(a = y_0 \lt \dots \lt y_M = b\) such that
    \begin{equation*} U(f;x_0, \dots, x_N) \leq \int_a^bf+\eps \quad\text{and}\quad U(g;y_0, \dots, y_M) \leq \int_a^bg+\eps. \end{equation*}
    Letting \(a= z_0 \lt \dots \lt z_T = b\) denote the common refinement of \((x_0, \dots, x_N)\) and \((y_0, \dots, y_M)\text{,}\) monotonicty of Riemann sums under refinements (Proposition 9.8) gives that
    \begin{equation*} U(f;z_0,\dots, z_T) + U(g;z_0, \dots, z_T) \leq \int_a^b f + \int_a^b g + 2\eps\text{.} \end{equation*}
    Since
    \begin{equation*} \sup_{[z_{n-1}, z_n]} f + \sup_{[z_{n-1}, z_n]} g \end{equation*}
    is an upper bound for the set
    \begin{equation*} \set{f(x)+g(x) : x \in [z_{n-1}, z_n]}\text{,} \end{equation*}
    we have that
    \begin{equation*} \sup_{[z_{n-1}, z_n]} (f + g) \leq\sup_{[z_{n-1}, z_n]} f + \sup_{[z_{n-1}, z_n]} g\text{.} \end{equation*}
    Therefore
    \begin{equation*} U(f+g;z_0, \dots, z_T)\leq U(f;z_0,\dots, z_T) + U(g;z_0, \dots, z_T)\text{,} \end{equation*}
    and thus
    \begin{equation*} U(f+g;[a,b]) \leq \int_a^b f + \int_a^b g + 2\eps\text{.} \end{equation*}
    A similar argument shows that
    \begin{equation*} L(f+g;[a,b]) \geq \int_a^b f + \int_a^b g - 2\eps. \end{equation*}
    Letting \(\eps \to 0\) and using the inequality \(L(f+g;[a,b]) \leq U(f+g;[a,b])\) (proved in Corollary 9.9), the sandwich rule then yields that
    \begin{equation*} U(f+g;[a,b]) = L(f+g;[a,b]) = \int_a^b f + \int_a^b g\text{.} \end{equation*}
    (b)
    Deduce how the Riemann integral respects pointwise inequalities.
    Solution.
    Corollary 9.14. Inequality rule for the Riemann integral.
    Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(f,g : [a,b] \to \R\) be functions which are Riemann integrable on \([a,b]\text{.}\) Suppose that \(f \leq g\) everywhere on \([a,b]\text{.}\) Then
    \begin{equation*} \int_a^bf\leq \int_a^bg. \end{equation*}
    Proof.
    By the addition and scaling rules (Theorem 9.13) the function \(g-f\) is Riemann integrable. Using the addition and scaling rules, and the definition of the Riemann integral, we have
    \begin{equation*} \int_a^bg - \int_a^bf =\int_a^b(g-f) \geq L(g-f;a,b) = (b-a)\inf_{[a,b]} (g-f). \end{equation*}
    Since \(g-f\geq 0\) on \([a,b]\text{,}\) we have
    \begin{equation*} (b-a)\inf_{[a,b]} (g-f) \geq 0. \end{equation*}
    This yields the required inequality.

    - 9.15. Integrating on a sub-interval.

    State and prove how a Riemann integrable function is still Riemann integrable on any closed subinterval of its domain.
    Solution.
    Proposition 9.16.
    Let \(a,b\in \R\) with \(a\lt b\) and let \(f :[a,b]\to \R\) be Riemann integrable. Then for any \(x\in [a,b]\) we have that \(f\) is Riemann integrable on both \([a,x]\) and \([x,b]\) with
    \begin{equation*} \int_a^b f = \int_a^x f+ \int_x^b f. \end{equation*}
    Proof.
    Since increasing an infimum gives a non-lower bound, there exist a subdivision \(a= x_0 \lt \dots \lt x_N = b\) such that
    \begin{equation*} U(f;x_0, \dots, x_N) \leq \int_a^bf+\eps\text{.} \end{equation*}
    Since decreasing a supremum gives a non-upper bound, there exist a subdivision \(a= y_0 \lt \dots \lt y_M = b\) such that
    \begin{equation*} L(f;y_0, \dots, y_M) \geq \int_a^bf-\eps\text{.} \end{equation*}
    Let \(a= z_0 \lt \dots \lt z_T = b\) denote a common refinement of \((x_0, \dots, x_N)\) and \((y_0, \dots, y_M)\) which also contains the point \(x = z_m\) for some \(m\text{.}\) Then monotonicty of Riemann sums under refinements (Proposition 9.8) gives that
    \begin{equation*} 0\leq U(f;z_0,\dots, z_T) - L(f;z_0, \dots, z_T) \leq 2\eps.\text{.} \end{equation*}
    Since \(a= z_0 \lt \dots \lt z_m = x\) is a subdivision of \([a,x]\) we have that
    \begin{align*} 0\amp \leq U(f; [a,x])-L(f;[a,x])\leq U(f;z_0,\dots, z_m) - L(f;z_0, \dots, z_m) \leq\\ \amp U(f;z_0,\dots, z_m) - L(f;z_0, \dots, z_m) + U(f;z_m,\dots, z_T) - L(f;z_m, \dots, z_T) \\ \amp = U(f;z_0,\dots, z_T) - L(f;z_0, \dots, z_T) \leq 2\eps\text{.} \end{align*}
    Letting \(\eps \to 0\) then establishes that \(U(f;[a,x])= L(f;[a,x])\text{,}\) so that \(f\) is Riemann integrable on \([a,x]\text{.}\)
    A similar argument to that given above shows that \(f\) is Riemann integrable on \([x,b]\text{.}\)
    For the addition formula, we may use the same subdivision \(a= z_0 \lt \dots \lt z_T = b\) constructed above to see that
    \begin{align*} \int_a^xf + \int_x^bf \amp \leq U(f;z_0,\dots, z_m)+U(f;z_m,\dots, z_T)\\ \amp = U(f; z_0, \dots, z_T) \leq \int_a^b f + \eps\text{.} \end{align*}
    Similarly
    \begin{equation*} \int_a^x f+\int_x^bf \geq \int_a^b f-\eps\text{.} \end{equation*}
    Again letting \(\eps \to 0\) then gives the addition formula.

    - 9.17. True/false workshop quiz.

    In your workshop, you may be asked to vote on whether you think the following are is true (thumbs up) or false (thumbs down). For each question, your workshop tutor will ask one of you to briefly justify your vote. This justification can involve giving a counterexample (if false) or outlining which definition/theorem the statement follows from (if true).
    (a)
    Consider the function
    \begin{equation*} f(x):= \begin{cases} x \amp \text{ if } x\leq 0,\\ 0 \amp \text{ if } 0 \lt x \lt 1,\\ x-1 \amp \text{ if } x \geq 1. \end{cases} \end{equation*}
    Then \(f\) has a local maximum at \(x=1/2\text{.}\)
    (b)
    If a function \(f\) has a local minimum at \(x_0\text{,}\) then \(f'(x_0) = 0\text{.}\)
    (c)
    If a function \(f:\R\to\R\) has a maximum at \(x_0\text{,}\) then for any \(x\lt x_0\) we have
    \begin{equation*} \frac{f(x)-f(x_0)}{x-x_0} \geq 0\text{.} \end{equation*}
    (d)
    There exists a function \(f:[0,1]\to \R\) which is differentiable on \((0,1)\) and has \(f(0) = f(1)\text{,}\) yet the derivative of \(f\) is never zero on \((0,1)\text{.}\)
    (e)
    It follows from the mean value theorem that if \(f:\R\to \R\) is differentiable with \(|f'(x)|\leq 100\) for all \(x\text{,}\) then for all \(x_1,x_2 \in \R\) we have
    \begin{equation*} |f(x_1)-f(x_2)|\leq 100|x_1-x_2|\text{.} \end{equation*}
    (f)
    If a function \(f:[a,b]\to \R\) is Riemann integrable then for any subdivision \(a=x_0\lt \dots\lt x_N=b\) we have
    \begin{equation*} L(f;x_0, \dots, x_N) = U(f;x_0, \dots, x_N)\text{.} \end{equation*}
    (g)
    For any bounded function \(f:[a,b]\to \R\) we have
    \begin{equation*} U(f;a, \tfrac{a+b}{2}, b) \leq U(f;a,b)\text{.} \end{equation*}
    (h)
    For any bounded function \(f:[a,b]\to \R\) we have
    \begin{equation*} U(f;[a,b]) \leq L(f;[a,b])\text{.} \end{equation*}
    (i)
    If \(f:[a,b]\to \R\) is a monotone function then
    \begin{equation*} \int_a^b f = f(b)-f(a)\text{.} \end{equation*}
    (j)
    If \(f:[a,b]\to \R\) is a Riemann integrable then the function
    \begin{equation*} F(x) := \int_a^x f \end{equation*}
    is defined at every \(x \in [a,b]\text{.}\)
