    Subsection 7.1 Self-test questions for lecture 13

    - 7.1. Differentiability.

    Define what it means for a real-valued function on an open interval to be differentiable at a point and define the derivative at that point. State what it means for such a function to be differentiable.
    Solution.
    Definition 7.2. Differentiable.
    Let \(f : (a,b)\to \R\) and \(x_0 \in (a,b)\text{.}\) We say that \(f\) is differentiable at \(x_0\) if there exists \(y_0 \in \R\) such that
    \begin{equation*} \frac{f(x)-f(x_0)}{x-x_0} \to y_0 \quad \text{as}\quad x\to x_0 \quad \text{through} \quad (a,b)\setminus\set{x_0}. \end{equation*}
    Should such a value of \(y_0\) exist, one can check that it is unique, and we call it the derivative of \(f\) at \(x_0\text{,}\) denoted \(f'(x_0)\text{.}\)
    We say that \(f : (a,b)\to \R\) is differentiable if it is differentiable at every point \(x_0 \in (a,b)\text{.}\)
    Example 7.3. Constant functions are differentiable.
    Let \(f : (a,b) \to \R\text{.}\) Suppose that \(f\) is constant. In other words, there exists \(c \in \R\) such that for all \(x \in (a,b)\) we have \(f(x) = c\text{.}\) Then for \(x \in (a,b)\setminus\set{x_0}\) we have
    \begin{equation*} \frac{f(x)-f(x_0)}{x-x_0} = \frac{c-c}{x-x_0} = 0 \to 0 \quad \text{as} \quad x \to x_0\quad\text{through}\quad (a,b)\setminus\set{x_0}. \end{equation*}
    Hence \(f\) is differentiable on \((a,b)\text{.}\) Moreover \(f' = 0\) on \((a,b)\text{.}\)
    Example 7.4. The identity function is differentiable.
    Let \(f : (a,b) \to \R\) be given by \(f(x):=x\) for all \(x \in (a,b)\text{.}\) Then for \(x_0 \in (a,b)\) and \(x\in(a,b)\setminus\set{ x_0}\) we have
    \begin{equation*} \frac{f(x)-f(x_0)}{x-x_0} = \frac{x-x_0}{x-x_0} = 1 \to 1 \quad \text{as} \quad x \to x_0\quad\text{through}\quad (a,b)\setminus\set{x_0}. \end{equation*}
    Hence \(f\) is differentiable on \((a,b)\text{.}\) Moreover \(f' =1\) on \((a,b)\text{.}\)
    Example 7.5. The reciprocal function is differentiable.
    Let \(f : (0,\infty) \to \R\) be given by \(f(x):=1/x\) for all \(x \in (0,\infty)\text{.}\) Then for \(x_0 \in (0,\infty)\) and \(x\in(0,\infty)\setminus\set{ x_0}\) we have
    \begin{equation*} \frac{f(x)-f(x_0)}{x-x_0} = \frac{\recip{x}-\recip{x_0}}{x-x_0} = -\recip{xx_0} \to -\recip{x_0^2} \quad \text{as} \quad x \to x_0\quad\text{through}\quad (0,\infty)\setminus\set{x_0}. \end{equation*}
    Hence \(f\) is differentiable on \((0, \infty)\text{.}\) Moreover \(f'(x) =-1/x^2\) on \((0, \infty)\text{.}\) One can also check that \(1/x\) is differentiable on \((-\infty, 0)\) with derivative \(-1/x^2\text{.}\)

    - 7.6. Differentiable implies continuous.

    State and prove how a function which is differentiable at a point is continuous at that point.
    Solution.
    Proposition 7.7. Differentiable implies continuous.
    Let \(f : (a,b) \to \R\) be differentiable at \(x_0 \in (a,b)\text{.}\) Then \(f\) is continuous at \(x_0\text{.}\)
    Proof.
    By the epsilon-delta characterisation of a finite limit at a finite point (see Proposition 5.17) we are required to verify that
    \begin{align} (\forall \eps \gt 0)(\exists \delta_1 \amp= \delta_1(x_0,\eps) \gt 0)(\forall x \in (a,b))\notag\\ \amp \bigsqbrac{\ |x-x_0| \leq \delta_1 \implies |f(x)-f(x_0)| \leq \eps\ }.\tag{7.1} \end{align}
    By the definition of differentiability (see Definition 7.2) we know that
    \begin{align*} (\forall \eps \gt 0)\amp(\exists \delta_2 = \delta_2(x_0,\eps) \gt 0) (\forall x \in (a,b))\\ \amp \sqbrac{\ 0\lt |x-x_0| \leq \delta_2 \implies \abs{\frac{f(x)-f(x_0)}{x-x_0}-f'(x_0)} \leq \eps\ }. \end{align*}
    In particular, if \(x \in (a,b) \) and \(0\lt|x-x_0| \leq \delta_2\) then
    \begin{equation*} \abs{f(x)-f(x_0)} \leq \eps \abs{x-x_0} + \abs{f'(x_0)}|x-x_0|. \end{equation*}
    We therefore obtain (7.1) on taking
    \begin{equation*} \delta_1(x_0,\eps) := \min\set{\delta_2(x_0,\eps),\frac{\eps}{\eps+\abs{f'(x_0)}}}. \end{equation*}
    Example 7.8. A continuous function which is not differentiable.
    The function \(x \mapsto |x|\) is an example of a function which is continuous at a point, but not differentiable there (namely at \(x_0 = 0\)). Hence continuity does not imply differentiability. To see that this function is not differentiable at \(x_0=0\text{,}\) consider the sequences \((1/n)_{n=1}^\infty\) and \((-1/n)_{n=1}^\infty\text{.}\) Both are sequences in the set \(\R\setminus\set{0}\) and both converge to 0, yet
    \begin{equation*} \frac{|1/n|-|0|}{(1/n)-0} = 1 \to 1 \quad\text{as}\quad n \to \infty \end{equation*}
    whilst
    \begin{equation*} \frac{|-1/n|-|0|}{(-1/n)-0} = -1 \to 1 \quad\text{as}\quad n \to \infty\text{.} \end{equation*}
    It follows that \(x \mapsto |x|\) does not satisfy the definition (Definition 7.2) of being differentiable at \(x_0 = 0\text{,}\) since if \(y_0\) were the derivative then we get the contradiction that \(-1 = y_0= 1\text{.}\)

    - 7.9. Addition and product rules.

    State and prove both the addition and product rules for differentiation.
    Solution.
    Proposition 7.10. Algebra of differentiation.
    Let \(f,g: (a,b)\to \R\) be differentiable at \(x_0 \in (a,b)\text{.}\) Then the sum \(f+g\) and product \(f\cdot g\) are both differentiable at \(x_0\text{.}\) Moreover, their derivatives satisfy the identities
    \begin{align*} (f+g)'(x_0) \amp = f'(x_0) + g'(x_0), \amp \text{(addition rule)}\\ (f\cdot g)'(x_0) \amp = f'(x_0)g(x_0) + f(x_0)g'(x_0).\amp \text{(product rule)} \end{align*}
    Proof.
    Let \((a_n)_n\) be a sequence in \((a,b)\setminus\set{x_0}\) with \(a_n \to x_0\text{.}\) Then the addition rule follows from the addition rule for sequence limits, since
    \begin{align*} \lim_{n \to \infty} \amp \frac{(f+g)(a_n)-(f+g)(x_0)}{a_n-x_0}\\ \amp = \lim_{n\to \infty} \frac{f(a_n)-f(x_0)}{a_n-x_0}+ \lim_{n \to \infty} \frac{g(a_n)-g(x_0)}{x-x_0}. \end{align*}
    For the product rule, we can use the identity
    \begin{align*} \amp f(a_n)g(a_n) - f(x_0)g(x_0) = \sqbrac{f(a_n)-f(x_0)}g(a_n) + f(x_0)\sqbrac{g(a_n)-g(x_0)}\\ \amp =\sqbrac{f(a_n)-f(x_0)}\sqbrac{g(a_n)-g(x_0)} + \sqbrac{f(a_n)-f(x_0)}g(x_0)\\ \amp + f(x_0)\sqbrac{g(a_n)-g(x_0)} \end{align*}
    By the multiplication rule for sequence limits (Proposition 2.17), we have
    \begin{gather*} \lim_{n \to \infty}\frac{\sqbrac{f(a_n)-f(x_0)}g(x_0)}{a_n-x_0} = f'(x_0)g(x_0),\\ \lim_{n \to \infty}\frac{f(x_0)\sqbrac{g(a_n)-g(x_0)}}{a_n-x_0} = f(x_0)g'(x_0). \end{gather*}
    Similarly
    \begin{align*} \amp \lim_{n\to\infty}\frac{\sqbrac{f(a_n)-f(x_0)}\sqbrac{g(a_n)-g(x_0)}}{a_n-x_0}\\ \amp = \lim_{n\to\infty}\frac{\sqbrac{f(a_n)-f(x_0)}}{a_n-x_0}\lim_{n\to\infty}\frac{\sqbrac{g(a_n)-g(x_0)}}{a_n-x_0}\lim_{n\to\infty}(a_n-x_0)\\ \amp = f'(x_0)\cdot g'(x_0)\cdot 0 = 0. \end{align*}

    Subsection 7.2 Self-test questions for lecture 14

    - 7.11. Chain rule.

    State the chain rule for differentiation and prove it in the situation in which the innermost function is injective (the general case is non-examinable).
    Solution.
    Theorem 7.12. Chain rule.
    Let \(f : (a,b) \to (c,d)\) and \(g: (c,d) \to \R\text{.}\) Suppose that \(f\) is differentiable at \(x_0\) and that \(g\) is differentiable at \(f(x_0)\text{.}\) Then the composition \(g\circ f\) is differentiable at \(x_0\text{.}\) Moreover the derivative satisfies the identity
    \begin{equation*} (g\circ f)'(x_0) = g'\bigbrac{f(x_0)} \cdot f'(x_0). \end{equation*}
    Proof in the case that \(f\) is injective.
    Let \((a_n)_{n=1}^\infty\) be a sequence in \((a,b) \setminus\set{x_0}\) with \(a_n \to x_0\text{.}\) We wish to prove that
    \begin{equation*} \lim_{n\to \infty} \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{a_n-x_0} = g'\bigbrac{f(x_0)}f'(x_0). \end{equation*}
    Since \(f\) is injective, we have that \(f(a_n) - f(x_0)\neq 0\) for all \(n\text{.}\) Therefore
    \begin{equation*} \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{a_n-x_0} = \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{f(a_n)-f(x_0)}\cdot \frac{f(a_n)-f(x_0)}{a_n-x_0} . \end{equation*}
    Since differentiability implies continuity (Proposition 7.7), we have
    \begin{equation*} \lim_{n \to \infty} f(a_n) = f(x_0). \end{equation*}
    Thus, since \(g\) is differentiable at \(f(x_0)\text{,}\) we have
    \begin{equation*} \lim_{n\to \infty} \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{f(a_n)-f(x_0)} = g'\bigbrac{f(x_0)}. \end{equation*}
    We may therefore use the multiplication rule for sequential limits (Proposition 2.17) to deduce that
    \begin{align*} \lim_{n \to \infty} \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{a_n-x_0} \amp = \lim_{n\to\infty}\brac{ \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{f(a_n)-f(x_0)}\cdot \frac{f(a_n)-f(x_0)}{a_n-x_0}}\\ \amp = \lim_{n\to\infty}\frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{f(a_n)-f(x_0)}\lim_{n\to \infty} \frac{f(a_n)-f(x_0)}{a_n-x_0}\\ \amp = g'\bigbrac{f(x_0)}f'(x_0). \end{align*}
    Proof of the general case (non-examinable).
    Let \((a_n)_{n=1}^\infty\) be a sequence in \((a,b) \setminus\set{x_0}\) with \(a_n \to x_0\text{.}\) We wish to prove that
    \begin{equation} \lim_{n\to \infty} \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{a_n-x_0} = g'\bigbrac{f(x_0)}f'(x_0).\tag{7.2} \end{equation}
    We have
    \begin{equation} \frac{g\bigbrac{f( a_n)} - g\bigbrac{f(x_0)}}{ a_n-x_0} = \begin{cases} 0, \amp \text{if } f( a_n) = f(x_0);\\ \frac{g\brac{f( a_n)}-g\brac{f(x_0)}}{f( a_n)-f(x_0)}\cdot \frac{f( a_n)-f(x_0)}{ a_n-x_0}, \amp \text{otherwise.} \end{cases}\tag{7.3} \end{equation}
    First, let us suppose that \(f(a_n) = f(x_0)\) for only finitely many values of \(n\text{.}\) Then there exists \(N_0\in \N\) such that for all \(n \geq N_0\) we have \(f(a_n)\neq f(x_0)\text{.}\) Hence for all \(n \geq N_0\) we have
    \begin{equation*} \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{a_n-x_0} = \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{f(a_n)-f(x_0)}\cdot \frac{f(a_n)-f(x_0)}{a_n-x_0} . \end{equation*}
    Since differentiability implies continuity (Proposition 7.7), we have
    \begin{equation*} \lim_{n \to \infty} f(a_n) = f(x_0). \end{equation*}
    Thus
    \begin{equation} \lim_{n\to \infty} \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{f(a_n)-f(x_0)} = g'\bigbrac{f(x_0)}.\tag{7.4} \end{equation}
    We may therefore use the multiplication rule for sequential limits (Proposition 2.17) to deduce that
    \begin{align*} \lim_{n \to \infty} \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{a_n-x_0} \amp = \lim_{n\to\infty}\brac{ \frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{f(a_n)-f(x_0)}\cdot \frac{f(a_n)-f(x_0)}{a_n-x_0}}\\ \amp = \lim_{n\to\infty}\frac{g\bigbrac{f(a_n)}-g\bigbrac{f(x_0)}}{f(a_n)-f(x_0)}\lim_{n\to \infty} \frac{f(a_n)-f(x_0)}{a_n-x_0}\\ \amp = g'\bigbrac{f(x_0)}f'(x_0). \end{align*}
    Next, let us suppose that \(f(a_n) = f(x_0)\) for infinitely many values of \(n\text{.}\) Write \(k_1 \lt k_2 \lt \dots\) for those indices for which \(f(x_{k_n}) = f(x_0)\text{.}\) Write \(\ell_1 \lt \ell_2 \lt \dots\) for the (possibly finite) remaining indices with \(f(x_{l_n}) \neq f(x_0)\text{.}\) The sequence
    \begin{equation} n\mapsto \frac{g\bigbrac{f(x_{\ell_n})}-g\bigbrac{f(x_0)}}{f(x_{\ell_n})-f(x_0)}\tag{7.5} \end{equation}
    is either a finite sequence, or converges as in the proof of (7.4). In either case, the sequence is bounded, so there exists \(M \in \R\) such that for all \(\ell_n\) we have
    \begin{equation*} \abs{\frac{g\bigbrac{f(x_{\ell_n})}-g\bigbrac{f(x_0)}}{f(x_{\ell_n})-f(x_0)} } \leq M. \end{equation*}
    Combining this with (7.3), for any \(n \in \N\) we have
    \begin{equation} \abs{\frac{g\bigbrac{f( a_n)} - g\bigbrac{f(x_0)}}{ a_n-x_0}} \leq M\abs{\frac{f(x_{n}) - f(x_0)}{x_{n} - x_0}}.\tag{7.6} \end{equation}
    Since \(x_{k_n} \to x_0\) we have
    \begin{equation*} f'(x_0) = \lim_{n \to \infty} \frac{f(x_{k_n}) - f(x_0)}{x_{k_n} - x_0} = 0. \end{equation*}
    So it follows that
    \begin{equation*} \lim_{n \to \infty} \frac{f(x_{n}) - f(x_0)}{x_{n} - x_0} = f'(x_0) = 0. \end{equation*}
    Combining this with (7.6) gives
    \begin{equation*} \lim_{n\to \infty}\abs{\frac{g\bigbrac{f( a_n)} - g\bigbrac{f(x_0)}}{ a_n-x_0}} = 0. \end{equation*}
    Hence (7.2) also follows in this case.

    - 7.13. Quotient rule.

    State and prove the quotient rule for differentiation.
    Solution.
    Corollary 7.14. Quotient rule.
    Let \(f,g : (a,b) \to \R\) both be differentiable at \(x_0 \in D\text{.}\) Suppose that \(g(x_0) \neq 0\text{.}\) Then the quotient \(f/g\) is defined on \((x_0-\delta, x_0+\delta)\) for some \(\delta \gt 0\) and is differentiable at \(x_0\text{.}\) Moreover the derivative satisfies the identity
    \begin{equation} (f/g)'(x_0) = \frac{f'(x_0)g(x_0)-f(x_0)g'(x_0)}{g(x_0)^2}.\tag{7.7} \end{equation}
    Proof.
    Since differentiability implies continuity (Proposition 7.7), the function \(g\) is continuous at \(x_0\text{.}\) Since \(g(x_0) \neq 0\text{,}\) and non-zero points of continuity yield a non-zero neighbourhood (Proposition 5.19), there exists \(\delta \gt0\) such that \(g(x) \neq 0\) for all \(x \in (a,b)\cap (x_0-\delta, x_0+\delta)\text{.}\) Decreasing \(\delta\) if necessary, we may assume that \((x_0-\delta, x_0+\delta)\subset (a,b)\text{.}\) Hence the quotient \(x \mapsto f(x)/g(x)\) is defined on \((x_0-\delta, x_0+\delta)\text{.}\) Composing the reciprocal function \(y \mapsto 1/y\) with \(g\text{,}\) the chain rule (Theorem 7.12) and Example 7.5 together give that the function
    \begin{equation*} x\mapsto \recip{g(x)} \end{equation*}
    is differentiable at \(x_0\text{.}\) Moreover, the derivative satisfies
    \begin{equation*} \brac{\recip{g(x_0)}}' = -\frac{g'(x_0)}{g(x_0)^2}. \end{equation*}
    Hence by the product rule (Proposition 7.10) the quotient \(f/g = f \cdot\brac{1/g}\) is differentiable at \(x_0\) with derivative
    \begin{equation*} \frac{f'(x_0)}{g(x_0)} - \frac{f(x_0)g'(x_0)}{g(x_0)^2}. \end{equation*}
    This completes the proof of the quotient rule.

    - 7.15. The inverse function theorem.

    (a)
    State and prove how differentiable functions with continuous inverse have a differentiable inverse.
    Solution.
    Theorem 7.16. Inverse function theorem.
    Let \(f: (a,b) \to (c,d)\) be a bijective function and let \(x_0 \in (a,b)\text{.}\) Suppose that \(f^{-1}\) is continuous at \(y_0:=f(x_0)\) and that \(f\) is differentiable at \(x_0\) with
    \begin{equation*} f'(x_0) \neq 0. \end{equation*}
    Then \(f^{-1}\) is differentiable at \(y_0\) with derivative
    \begin{equation} \brac{f^{-1}}'(y_0) = \recip{f'(x_0)}.\tag{7.8} \end{equation}
    Proof.
    We are required to prove that if \((y_n)_{n=1}^\infty\) is a sequence in \((c,d)\setminus\set{y_0}\) with \(y_n \to y_0\text{,}\) then
    \begin{equation*} \frac{f^{-1}(y_n) - f^{-1}(y_0)}{y_n-y_0} \to \recip{f'(x_0)}. \end{equation*}
    Since \(f\) is bijective, for each \(y_n\in (c,d)\setminus\set{y_0}\) there exists \(x_n \in (a,b)\setminus\set{x_0}\) such that \(f(x_n) = y_n\text{.}\) Thus \((x_n)_{n=1}^\infty\) is a sequence in \((a,b)\setminus\set{x_0}\text{.}\) Since \(f^{-1}\) is continuous at \(y_0\) we have
    \begin{equation*} x_n = f^{-1}(y_n) \to f^{-1}(y_0) = x_0. \end{equation*}
    Since \(f\) is differentiable at \(x_0\text{,}\) it follows that
    \begin{equation*} \frac{f(x_n)-f(x_0)}{x_n-x_0} \to f'(x_0). \end{equation*}
    Since \(f'(x_0) \neq 0\text{,}\) it follows from the reciprocal rule for limits (Proposition 2.21) that
    \begin{equation*} \frac{f^{-1}(y_n)-f^{-1}(y_0)}{y_n-y_0} = \frac{x_n-x_0}{f(x_n)-f(x_0)}\to \recip{f'(x_0)}. \end{equation*}

    - 7.17. Differentiability of power series.

    (a)
    State how power series are differentiable within their radius of convergence (the proof is non-examinable).
    Solution.
    Theorem 7.18. Power series are differentiable within their radius of convergence.
    Let \((a_n)_{n=0}^\infty\) be a sequence of real numbers. Let \(R \in [0, \infty]\) denote the radius of convergence (see Theorem 5.4) of the corresponding power series:
    \begin{equation} f(x) := \sum_{n =0}^\infty a_n x^n.\tag{7.9} \end{equation}
    Then the function \(f : (-R, R) \to \R\) given by (7.9) is differentiable. Moreover, for any \(x_0 \in (-R, R)\) we have the formula
    \begin{equation} f'(x_0) = \sum_{n = 1}^\infty na_n x_0^{n-1}\tag{7.10} \end{equation}
    where the right-hand side of (7.10) is absolutely convergent.
    Proof (non-examinable).
    Let us first show that the right-hand side of (7.10) is absolutely convergent. Let \(r\) be a real number satisfying \(|x_0| \lt r \lt R\text{.}\) Such a number exists since \(|x_0| \lt R\text{,}\) for instance we could set
    \begin{equation*} r:=\frac{|x_0|+R}{2}\text{.} \end{equation*}
    We claim that
    \begin{equation} \sum_{n=2}^\infty n^2 |a_n| r^{n-2} \lt \infty.\tag{7.11} \end{equation}
    To see this, let \(r_1\) be a real satisfying \(r \lt r_1 \lt R\text{.}\) Then
    \begin{equation*} \sum_{n=2}^\infty n^2 |a_n| r^{n-2} \leq r^{-2}\sum_{n=2}^\infty n^2 (r/r_1)^n |a_n| r_1^n. \end{equation*}
    Since exponentials beat polynomials (Proposition 3.14), we have that \(n^2 (r/r_1)^n \to 0\text{,}\) in particular the sequence \(n^2 (r/r_1)^n\) is bounded. Hence there exists \(C\) such that
    \begin{equation*} \sum_{n=2}^\infty n^2 |a_n| r^{n-2} \leq r^{-2}\sum_{n=2}^\infty C|a_n| r_1^n \leq r^{-2}C\sum_{n=0}^\infty |a_n|r_1^n. \end{equation*}
    The latter is finite, since the series (7.9) converges absolutely for \(|x|\lt R\text{,}\) in particular for \(x=r_1\text{.}\) This establishes the claim (7.11). It follows, by the comparison test (Proposition 4.15), that we obtain absolute convergence of
    \begin{equation*} \sum_{n=1}^\infty n a_n x^{n-1}. \end{equation*}
    With the above facts on absolute convergence established, we are able to prove (7.10). By linearity of series (Proposition 4.13), for \(x \in [-r,r]\) with \(x\neq x_0\) we have
    \begin{equation*} \frac{f(x) - f(x_0)}{x-x_0} - \sum_{n=1}^\infty na_n x_0^{n-1} = \recip{x-x_0}\sum_{n=2}^\infty a_n \sqbrac{x^n-x_0^n-(x-x_0)nx_0^{n-1}}\text{.} \end{equation*}
    For \(n\in \N\) we have the identity
    \begin{equation*} x^n-x_0^n = (x-x_0)\brac{x^{n-1} + x^{n-2}x_0 + \dots + xx_0^{n-2} + x_0^{n-1}}\text{.} \end{equation*}
    Hence for \(x,x_0 \in [-r,r]\) and \(k \in \N\cup\set{0}\) we obtain the bound
    \begin{equation*} |x^k-x_0^k| \leq |x-x_0|kr^{k-1}\text{.} \end{equation*}
    Hence for \(x,x_0 \in [-r,r]\) and \(n\in \N\) we have
    \begin{align*} \abs{x^n-x_0^n-(x-x_0)nx_0^{n-1}} \amp \leq \sum_{k=0}^{n-1} \abs{x^{k}x_0^{n-1-k} - x_0^{n-1}} \\ \amp = \sum_{k=0}^{n-1} \abs{x^k-x_0^k}|x_0|^{n-1-k} \\ \amp \leq \sum_{k=0}^{n-1} k|x-x_0|r^{n-2} \leq n^2 |x-x_0| r^{n-2}\text{.} \end{align*}
    Therefore
    \begin{equation*} \abs{\frac{f(x) - f(x_0)}{x-x_0} - \sum_{n=1}^\infty na_n x_0^{n-1}} \leq \abs{x-x_0}\sum_{n=2}^\infty n^2|a_n|r^{n-2}\text{.} \end{equation*}
    On taking
    \begin{equation*} \delta(x_0, \eps) := \min\set{\frac{\eps}{\sum_{n=2}^\infty n^2|a_n|r^{n-2}}, r-|x_0|} \gt 0 \end{equation*}
    and \(y_0 := \sum_{n=1}^\infty n a_n x_0^{n-1}\text{,}\) we deduce that
    \begin{align*} (\forall \eps \gt 0)\amp(\exists \delta \gt 0)(\forall x \in (-R, R)\setminus\set{x_0})\\ \amp \sqbrac{ |x-x_0| \leq \delta \implies \abs{\tfrac{f(x)-f(x_0)}{x-x_0} - y_0}\leq \eps}\text{.} \end{align*}
    So we have established the definition of differentiability (Definition 7.2) via the epsilon-delta characterisation (Proposition 5.17) of a finite limit at a finite point.

    Subsection 7.3 Workshop questions

    - 7.19. True/false workshop quiz.

    In your workshop, you will vote on whether you think each of the following statements is true (thumbs up) or false (thumbs down). For each question, your workshop tutor will ask one of you to briefly justify your vote. This justification can involve giving a counterexample (if false) or outlining which definition/theorem the statement follows from (if true).
    (a)
    Let \(f: \R\to\R\) and suppose that \(f(x)\to y_0\) as \(x\to x_0\) through \(\R\text{.}\) Then it is also the case that \(f(x)\to y_0\) as \(x\to x_0\) through \((0,\infty)\text{.}\)
    (b)
    Let \(f: \R\to\R\) and suppose that \(|f(x)|\to |y_0|\) as \(x\to x_0\) through \(\R\text{.}\) Then it is also the case that \(f(x)\to y_0\) as \(x\to x_0\) through \(\R\text{.}\)
    (c)
    Let \(f: \R\to\R\) and \(x_0,y_0\in \R\text{.}\) Suppose that \(f(x)\to y_0\) as \(x\to x_0\) through \(\R\text{.}\) Then it is also the case that \(f(10x)\to y_0\) as \(x\to x_0/10\) through \(\R\text{.}\)
    (d)
    All polynomials are uniformly continuous.
    (e)
    A uniformly continuous function on a closed interval is also continuous.
    (f)
    By the intermediate value theorem, a continuous function \(f:[a,b]\to\R\) attains all of the values between \(f(a)\) and \(f(b)\text{.}\)
    (g)
    By the intermediate value theorem, an increasing function \(f:[a,b]\to\R\) attains all of the values between \(f(a)\) and \(f(b)\text{.}\)
    (h)
    By the intermediate value theorem, a function \(f:\R\to\R\) which is increasing and continuous has \(f(\R)=\R\text{,}\) so that it attains all real values.
    (i)
    By the extreme value theorem, a continuous function \(f:\R\to\R\) attains its infimum.
    (j)
    Suppose that for \(f:\R\to\R\) and \(x_0,y_0\in \R\) we have
    \begin{equation*} \frac{f(x_0+1/n)-f(x_0)}{(x_0+1/n)-x_0)} \to y_0 \quad\text{as}\quad n \to \infty. \end{equation*}
    Then \(f\) is differentiable at \(x_0\) with derivative \(y_0\text{.}\)
    (k)
    If \(f:[a,b]\to\R\) is differentiable, then \(f\) attains all of the values between \(f(a)\) and \(f(b)\text{.}\)
    (l)
    If a function is differentiable, then its derivative is continuous.
    (m)
    The set of differentiable functions \(f:(a,b)\to \R\) form a vector space.
    (n)
    The composition of two differentiable functions is also differentiable.
    (o)
    If a bijective function \(f:(a,b)\to(c,d)\) has continuous inverse and \(f'(x_0)\neq 0\) then \(1/f\) is differentiable at \(x_0\text{.}\)
    (p)
    If a bijective function \(f:(a,b)\to(c,d)\) has continuous inverse and \(f'(x_0)\neq 0\) then \(f^{-1}\) is differentiable at \(x_0\text{.}\)
    (q)
    If a power series in \(x\) converges absolutely at \(x\text{,}\) then it is differentiable at \(x\text{.}\)

    - 7.21. Exam-style self-test question for workshop.

    Solution.
