Proposition 2.17. Multiplication rule.
Let \((a_n)_{n=1}^\infty\) and \((b_n)_{n =1}^\infty\) be convergent sequences of real numbers. Then
\begin{equation*} \lim_{n \to \infty} (a_n\cdot b_n) = \brac{\lim_{n \to \infty} a_n}\cdot \brac{\lim_{n \to \infty} b_n}. \end{equation*}
Non-examinable proof.
Let \(\displaystyle a : = \lim_{n \to \infty}a_n\) and \(\displaystyle b := \lim_{n \to \infty}b_n\text{.}\) By definition of the limit (Definition 1.23), we wish to prove that
\begin{equation*} \forall \eps \gt 0\ \exists N = N(\eps)\ \forall n \geq N \text{ we have } |ab-a_nb_n| \leq \eps. \end{equation*}
Since the limits \(\displaystyle \lim_{n \to \infty} a_n\) and \(\displaystyle \lim_{n \to \infty} b_n\) both exist and are finite, we know that
\begin{align} \amp\forall \eps \gt 0\ \exists N_1 = N_1(\eps)\ \forall n \geq N_1 \text{ we have } \abs{a - a_n} \leq \eps,\tag{2.6}\\ \amp\forall \eps \gt 0\ \exists N_2 = N_2(\eps)\ \forall n \geq N_2 \text{ we have } \abs{b - b_n} \leq \eps .\tag{2.7} \end{align}
We observe the key identity
\begin{equation*} ab - a_nb_n = ab-a_nb +a_nb-a_nb_n = (a-a_n)b + a_n(b - b_n). \end{equation*}
Since convergent sequences are bounded (Proposition 1.30), there exists \(M\in\R\) such that for all \(n\) we have \(|a_n| \leq M\text{.}\) Thus if \(|a - a_n| \leq \eps\) and \(|b - b_n| \leq \eps\) then
\begin{equation*} |ab - a_nb_n| \leq |a-a_n||b| + M|b - b_n| \leq (|b|+M)\eps. \end{equation*}
Writing \(N_1(\eps)\) and \(N_2(\eps)\) for the functions appearing in (2.6) and (2.7), set
\begin{equation*} N(\eps) := \max\set{N_1\brac{\tfrac{\eps}{|b|+M}}, N_2\brac{\tfrac{\eps}{|b|+M}}}. \end{equation*}
Then we have proved that
\begin{equation*} \forall \eps \gt 0\ \exists N = N(\eps)\ \forall n \geq N \text{ we have } \abs{ab - a_nb_n} \leq \brac{|b|+M}\frac{\eps}{|b|+M} = \eps . \end{equation*}
Thus \(a_nb_n\to ab\text{.}\)
