Case 2 (bounded above).
In this case, the set \(\set{a_n: n \in \N}\) is bounded above and contains a real number, hence has a real supremum \(a = \sup_{n \in \N} a_n \in \R\) by the completeness principle (see Fact 1.6). We claim that \(a_n \to a\text{.}\) According to the definition of a real limit (Definition 1.11), we must verify that (1.3) holds. To this end, let \(\eps \gt 0\) be arbitrary. Since decreasing the supremum gives a non-upper bound (Proposition 1.7) there exists \(a_N \in \set{a_n : n \in \N}\) such that \(a_N \gt a - \eps\text{.}\) Since the sequence is increasing, if \(n \geq N\) then \(a_n \gt a -\eps\text{.}\) As \(a = \sup_n a_n\) is an upper bound, we deduce that for all \(n \geq N\) we have
\begin{equation*} a-\eps \leq a_n \leq a. \end{equation*}
In particular
\begin{equation*} n \geq N \implies |a-a_n| \leq \eps. \end{equation*}
Therefore \(a_n \to a\) by definition (see (1.3)), completing our proof.
