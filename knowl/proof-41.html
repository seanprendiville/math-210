Proof

Notice that the set

\begin{equation*} \set{x \in \R : \sum_{n=0}^\infty a_n x^n \text{ converges}} \end{equation*}

is non-empty, as it contains \(x=0\text{.}\) If this set is not bounded above, set \(R:=\infty\text{.}\) If this set is bounded above, we can use the completeness principle to take \(R\) to be its supremum.

If \(|x| \gt R\) then \(a_n x^n \not \to 0\text{,}\) hence by the \(n\)th term test (Proposition 4.23) the series \(\sum_n a_n x^n\) does not converge.

Suppose that \(|x| \lt R\text{.}\) Since decreasing a supremum gives a non upper bound (Lemma 1.9), there exists \(r \in (|x|, R]\) for which \(|a_n| r^n \to 0\text{.}\) The sequence \(|a_n| r^n\) must be bounded, since convergence implies boundedness (Proposition 1.30). Hence there exists \(M>0\) such that

\begin{equation*} |a_n x^n| = |a_n| r^n \brac{|x|/r}^n \leq M|x|^n/r^n. \end{equation*}

Since \(|x|/r \lt 1\text{,}\) the geometric series \(\sum_n M(|x|/r)^n\) is absolutely convergent (Example 4.3). Hence by the comparison test (Proposition 4.15) the series \(\sum_n a_n x^n\) is absolutely convergent.

For uniqueness, suppose that \(R_1 \lt R_2\) both satisfy items Item 4.24.1 and Item 4.24.2. Setting \(r := \recip 2(R_1+R_2)\) we have \(R_1 \lt r \lt R_2\text{.}\) Hence by item Item 4.24.1 we have \(\sum_n |a_n|r^n \lt \infty\) and by item Item 4.24.2 we have \(\sum_n |a_n|r^n = \infty\text{,}\) a contradiction.
