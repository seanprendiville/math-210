Proof
Notice that the set
\begin{equation*} \set{|x| : \sum_{n=0}^\infty a_n x^n \text{ converges}} \end{equation*}
is non-empty, as it contains \(x=0\text{.}\) If this set is not bounded above, set \(R:=\infty\text{.}\) If this set is bounded above, we can use the completeness principle to take \(R\) to be its supremum.
If \(\sum_n a_n x^n\) converges then (by definition) \(|x|\leq R\text{.}\) Taking the contrapositive of this implication, we see that if \(|x| \gt R\) then \(\sum_n a_n x^n\) does not converge.
Suppose that \(|x| \lt R\text{.}\) Since decreasing a supremum gives a non upper bound (Lemma 1.9), there exists \(|y| \gt |x|\) such that \(\sum_n a_n y^n\) converges. Hence, by the \(n\)th term test (Proposition 4.23), we have \(|a_n| |y|^n \to 0\) as \(n\to \infty\text{.}\) In particular, as convergence implies boundedness (Proposition 1.30), there exists \(M\) such that
\begin{equation*} |a_n| |y|^n \leq M \qquad (\forall n \in \N)\text{.} \end{equation*}
It follows that for all \(n\in \N\) we have
\begin{equation*} |a_n x^n| = |a_n| |y|^n \brac{|x|/|y|}^n \leq M\brac{|x|/|y|}^n. \end{equation*}
Since \(|x|/|y| \lt 1\text{,}\) the geometric series \(\sum_n (|x|/|y|)^n\) is convergent (Example 4.3). Hence by the comparison test (Proposition 4.15) the series \(\sum_n a_n x^n\) is absolutely convergent.
