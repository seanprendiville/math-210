Proof

We prove the result assuming that \(f(a) \leq y \leq f(b)\text{;}\) if \(f(a) \geq y \geq f(b)\) then we reduce to the former case on considering \(-f\text{.}\) Set

\begin{equation*} x := \sup\set{t \in [a,b] : f(t) \leq y}. \end{equation*}

The above supremum exists because the set is non-empty (it contains \(a\)) and is bounded above by \(b\text{.}\) In particular, \(a \leq x \leq b\text{.}\) We claim that \(f(x) = y\text{.}\)

We first observe that for any \(n\in \N\) the number \(x-1/n\) is smaller than the supremum, so cannot be an upper bound. Hence there exists \(t_n \in [a,b]\) with \(f(t_n) \leq y\) and \(x-1/n \lt t_n \leq x\text{.}\) By the sandwich rule we have \(t_n \to x\) as \(n\to\infty\text{,}\) so by continuity and the inequality rule, we get that \(f(x)= \lim_{n\to \infty} f(t_n) \leq y\text{.}\)

It remains to show that \(f(x_0) \geq y\text{.}\) By way of contradiction, suppose that \(f(x_0) \lt y\text{.}\) Then \(x_0 \lt b\) (since \(f(b) \geq y \)). Continuous functions which are positive at a point are positive on a neighbourhood of that point (see PropositionÂ 5.20). Hence there exists \(\delta \gt 0\) such that for all \(x\in [a,b]\cap(x_0-\delta, x_0+\delta)\) we have \(y-f(x) \gt 0\text{.}\) On setting

\begin{equation*} \eps := \min\set{b-x_0, \delta/2} \gt 0, \end{equation*}

we obtain \(x+\eps \in [a,b]\) with \(f(x+\eps) \leq y\text{.}\) Yet, since \(x+\eps\) exceeds \(x\text{,}\) this contradicts the definition of \(x\) as the least upper bound of the set of \(t \in [a,b]\) with \(f(t) \leq y\text{.}\)
