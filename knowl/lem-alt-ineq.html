Lemma 4.19.
For any decreasing sequence \(b_1 \geq b_2 \geq \dots \geq b_n\geq 0\) of non-negative reals, with finitely many terms, we have the inequalities
\begin{equation} 0 \leq b_1 - b_2 + \dots + (-1)^{n+1} b_n \leq b_1.\tag{4.3} \end{equation}
Proof of LemmaÂ 4.19.
We prove this by induction on \(n\text{,}\) the number of terms in the sequence. The basis case, when \(n=1\text{,}\) follows from the inequalities
\begin{equation*} 0 \leq b_1 \leq b_1. \end{equation*}
For the inductive step, we may suppose that
\begin{equation*} 0 \leq b_2 -b_3 + \dots + (-1)^n b_{n} \leq b_2. \end{equation*}
It follows that
\begin{equation*} b_1 - b_2 + \dots + (-1)^{n+1} b_n = b_1- \brac{b_2 -b_3+ \dots + (-1)^{n} b_n} \geq b_1-b_2 \geq 0 \end{equation*}
and that
\begin{equation*} b_1 - b_2 + \dots + (-1)^{n+1} b_n = b_1- \brac{b_2 -b_3+ \dots + (-1)^{n} b_n} \leq b_1- 0= b_1. \end{equation*}
