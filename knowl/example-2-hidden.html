  • \(\sup (-\infty, 1) = 1.\)
    Proof.

    Notice that this example shows that the supremum does not always lie in the set. When the supremum lies in the set we call it the maximum (see Definition 1.11).

  • \(\sup \emptyset\) does not exist.

    Proof.
    This means that we can only take the supremum of a non-empty set.

  • If \(S\subset \R\) is not bounded above (for instance \(S=\R\)) then \(S\) has no upper bound, in particular no least upper bound. Thus \(\sup S\) does not exist.

    This means that we can only take the supremum of a set which is bounded above.

    The completeness principle (Theorem 1.15) says that if \(S\) is non-empty and bounded above, then \(\sup S\) exists.
