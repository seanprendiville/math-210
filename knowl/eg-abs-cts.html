Example 5.16. The absolute value function is continuous.
By the triangle inequality, for any \(x, x_0 \in \R\) we have
\begin{equation*} |x-x_0| \leq \eps \implies ||x|-|x_0|| \leq \eps. \end{equation*}
Hence (5.5) holds for the function \(x \mapsto |x|\) on taking \(\delta(x_0, \eps) := \eps\text{.}\)
