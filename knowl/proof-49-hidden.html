Let us suppose that \(x_0\) is a local maximum for \(f\text{;}\) the argument for the local minimum is similar. By definition of a local maximum (see Definition 7.7) there exists \(\delta \gt 0\) such that for all \(x \in (x_0-\delta, x_0+\delta)\) we have \(f(x_0) \geq f(x)\text{.}\) Hence
\begin{equation*} \frac{f(x)-f(x_0)}{x-x_0} \begin{cases} \geq 0 \amp \text{ if } x_0-\delta \lt x \lt x_0,\\ \leq 0 \amp \text{ if } x_0 \lt x \lt x_0+\delta.\end{cases} \end{equation*}
Setting \(a_n := x_0-\trecip{n+1}\delta\text{,}\) the sequential characterisation of the derivative (Proposition 6.23) and the inequality rule (Proposition 2.2) give that
\begin{equation*} f'(x_0) = \lim_{n\to \infty}\frac{f(a_n)-f(x_0)}{a_n-x_0} \geq 0. \end{equation*}
Similarly, setting \(b_0:= x_0+\trecip{n+1}\delta\) gives
\begin{equation*} f'(x_0) = \lim_{n\to \infty}\frac{f(b_n)-f(x_0)}{a_n-x_0} \leq 0. \end{equation*}
Thus \(f'(x_0) = 0\text{.}\)
