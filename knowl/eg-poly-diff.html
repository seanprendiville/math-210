Example 7.5. Polynomials and rational functions are differentiable.
Since the identity function \(x \mapsto x\) is differentiable on \(\R\text{,}\) the product rule gives that \(x \mapsto x^2\) is differentiable on \(\R\text{.}\) Using induction and the product rule, it follows that the for any \(n \in \N\) the function \(x \mapsto x^n = x\cdot x^{n-1}\) is differentiable on \(\R\) with derivative
\begin{equation*} (x^n)' = nx^{n-1}. \end{equation*}
Since constant functions \(x \mapsto c\) are differentiable on \(\R\text{,}\) with zero derivative, it follows from the product rule that the function \(x \mapsto c x^n\) is differentiable on \(\R\) with derivative
\begin{equation*} (cx^n)' = nc x^{n-1}. \end{equation*}
By the addition rule for differentiation (Proposition 7.2) it follows that any polynomial
\begin{equation*} x \mapsto c_0 + c_1 x + \dots + c_n x^n \end{equation*}
is differentiable on \(\R\text{,}\) with derivative
\begin{equation*} c_1 + 2c_2x + \dots + n c_n x^{n-1}. \end{equation*}
By the quotient rule for differentiation (Corollary 7.4) we deduce that any rational function (see Definition 6.5) is differentiable on its domain of definition.
