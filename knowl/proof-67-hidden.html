We first employ the scaling rule for suprema/infima (Lemma 9.2) to deduce that
\begin{equation*} U(cf;x_0,\dots, x_N) = \begin{cases} c U(f;x_0,\dots, x_N) \amp \text{ if } c \geq 0,\\ c L(f;x_0,\dots, x_N) \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Similarly
\begin{equation*} L(cf;x_0,\dots, x_N) = \begin{cases} c L(f;x_0,\dots, x_N) \amp \text{ if } c \geq 0,\\ c U(f;x_0,\dots, x_N) \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Employing the scaling rule (Lemma 9.2) a second time, as we take sup/inf over all subdivisions, we obtain
\begin{equation*} U(cf;[a,b]) = \begin{cases} c U(f;[a,b]) \amp \text{ if } c \geq 0,\\ c L(f;[a,b]) \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Similarly
\begin{equation*} L(cf;[a,b]) = \begin{cases} c L(f;[a,b]) \amp \text{ if } c \geq 0,\\ c U(f;[a,b]) \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Regardless of whether \(c \geq 0\) or \(c \lt 0\text{,}\) the fact that \(U(f;[a,b]) = L(f;[a,b])\) gives that
\begin{equation*} U(cf;[a,b]) = L(cf;[a,b]) = c\int_a^b f. \end{equation*}
