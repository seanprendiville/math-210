Lemma 9.13. Scaling rule for suprema and infima.
Let \(f : S \to \R\) be a bounded function on an arbitrary non-empty set \(S\text{.}\) Let \(c \in \R\text{.}\) Then:
\begin{equation*} \sup_{S} \brac{cf} = \begin{cases} \displaystyle c \sup_{ S} f \amp \text{ if } c \geq 0, \\ \displaystyle c \inf_{ S} f \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Similarly
\begin{equation*} \inf_{ S} \brac{cf} = \begin{cases} \displaystyle c \inf_{ S} f \amp \text{ if } c \geq 0, \\ \displaystyle c \sup_{ S} f \amp \text{ if } c \lt 0. \end{cases} \end{equation*}
Proof.
If \(c \gt 0\) then both claims follows from the fact that
\begin{equation*} \set{M : M \text{ is an upper bound for } cf} = \set{cM : M \text{ is an upper bound for } f}, \end{equation*}
and similarly for lower bounds.
For \(c \lt 0\) we use the result for \(c\gt0\) as follows
\begin{equation*} \sup_{S} \brac{cf} = \sup_{S} \brac{-(-c)f} = -\inf_S \brac{(-c) f} = -(-c)\inf_S f = c\inf_S f, \end{equation*}
and similarly for the infimum.
