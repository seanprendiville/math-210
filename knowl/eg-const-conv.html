Example 1.25. Constant sequences are convergent.
Fix a real number \(a \in \R\text{,}\) say \(a = 100\text{.}\) For each \(n \in \N\) set \(a_n := a\text{.}\) Then we claim that \(a_n \to a\text{.}\) To prove this, we must verify that (1.5) holds. To this end, fix \(\eps \gt 0\text{.}\) We must find \(N = N(\eps)\) such that
\begin{equation} n \geq N(\eps)\implies |a_n-a| \leq \eps.\tag{1.7} \end{equation}
For this particular sequence, we have \(|a_n-a| = |a-a| = 0\) for all \(n\text{.}\) Hence (1.7) holds on taking \(N(\eps) :=1\text{.}\)
