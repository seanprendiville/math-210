Definition 9.9. Positive and negative parts.
Given a real number \(x \in \R\) define the positive part by
\begin{equation*} x^+ := \max\set{0, x}. \end{equation*}
Using this notation, define the negative part by
\begin{equation*} x^-:= -\min\set{x,0} = |\min\set{x,0}| = (-x)^+. \end{equation*}
One may check that
\begin{equation} x = x^+ - x^- \quad\text{and}\quad |x| = x^+ + x^-.\tag{9.1} \end{equation}
