Proposition 2.21. Reciprocal rule.
Let \((a_n)_{n=1}^\infty\) be a sequence of real numbers which converge to a non-zero limit. Then
\begin{equation} \lim_{n \to \infty} \brac{\frac{1}{a_n}} = \frac{1}{\displaystyle \lim_{n\to \infty} a_n}. \tag{2.8} \end{equation}
Remark 2.22. No need to assume all terms non-zero.
Strictly speaking, (2.8) only makes sense if we know that \(a_n \neq 0\) for all \(n \in \N\text{,}\) so that the reciprocal \(1/a_n\) is well-defined for all values of \(n\text{.}\) However, the limiting value of a sequence does not depend on a finite number of initial terms: if you alter a finite number of terms of a sequence, you do not change the value of the limit. When the limit is non-zero, PropositionÂ 2.19 tells us that \(a_n = 0\) for only a finite number of \(n\text{,}\) so we may discard a finite number of initial terms in order for the expression (2.8) to make sense.
Non-examinable proof.
