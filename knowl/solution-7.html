Solution 1.10.b.1.
Given a set \(S \subset D\text{,}\) we write
\begin{equation*} \max_S f = \max_{x\in S} f(x) := \max\set{f(x) : x \in S}, \end{equation*}
provided that this quantity exists (i.e. provided that the set \(\set{f(x) : x \in S}\) has a supremum which belongs to the set).
Similarly, we write
\begin{equation*} \min_S f = \min_{x\in S} f(x) := \min\set{f(x) : x \in S}, \end{equation*}
provided that this quantity exists.
