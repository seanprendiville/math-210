Definition 5.12. Sequential definition of function limit.
Given a real-valued function \(f\) on a real domain, we write
\begin{equation*} f(x) \to y \quad \text{as} \quad x \to x_0 \end{equation*}
if, for any sequence \((a_n)_n\) in the domain 2  of \(f\) we have the implication
\begin{equation} a_n \to x_0 \quad \implies \quad f(a_n) \to y.\tag{5.3} \end{equation}
If there is only one such value of \(y\text{,}\) then we write
\begin{equation*} y = \lim_{x\to x_0} f(x) \end{equation*}
