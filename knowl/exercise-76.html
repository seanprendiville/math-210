- 9.17. True/false workshop quiz.

In your workshop, you will vote on whether you think each of the following statements is true (thumbs up) or false (thumbs down). For each question, your workshop tutor will ask one of you to briefly justify your vote. This justification can involve giving a counterexample (if false) or outlining which definition/theorem the statement follows from (if true).
(a)
Consider the function
\begin{equation*} f(x):= \begin{cases} x \amp \text{ if } x\leq 0,\\ 0 \amp \text{ if } 0 \lt x \lt 1,\\ x-1 \amp \text{ if } x \geq 1. \end{cases} \end{equation*}
Then \(f\) has a local maximum at \(x=1/2\text{.}\)
(b)
If a function \(f\) has a local minimum at \(x_0\text{,}\) then \(f'(x_0) = 0\text{.}\)
(c)
If a function \(f:\R\to\R\) has a maximum at \(x_0\text{,}\) then for any \(x\lt x_0\) we have
\begin{equation*} \frac{f(x)-f(x_0)}{x-x_0} \geq 0\text{.} \end{equation*}
(d)
There exists a function \(f:[0,1]\to \R\) which is differentiable on \((0,1)\) and has \(f(0) = f(1)\text{,}\) yet the derivative of \(f\) is never zero on \((0,1)\text{.}\)
(e)
It follows from the mean value theorem that if \(f:\R\to \R\) is differentiable with \(|f'(x)|\leq 100\) for all \(x\text{,}\) then for all \(x_1,x_2 \in \R\) we have
\begin{equation*} |f(x_1)-f(x_2)|\leq 100|x_1-x_2|\text{.} \end{equation*}
(f)
If a function \(f:[a,b]\to \R\) is Riemann integrable then for any subdivision \(a=x_0\lt \dots\lt x_N=b\) we have
\begin{equation*} L(f;x_0, \dots, x_N) = U(f;x_0, \dots, x_N)\text{.} \end{equation*}
(g)
For any bounded function \(f:[a,b]\to \R\) we have
\begin{equation*} U(f;a, \tfrac{a+b}{2}, b) \leq U(f;a,b)\text{.} \end{equation*}
(h)
For any bounded function \(f:[a,b]\to \R\) we have
\begin{equation*} U(f;[a,b]) \leq L(f;[a,b])\text{.} \end{equation*}
(i)
If \(f:[a,b]\to \R\) is a monotone function then
\begin{equation*} \int_a^b f = f(b)-f(a)\text{.} \end{equation*}
(j)
If \(f:[a,b]\to \R\) is a Riemann integrable then the function
\begin{equation*} F(x) := \int_a^x f \end{equation*}
is defined at every \(x \in [a,b]\text{.}\)
