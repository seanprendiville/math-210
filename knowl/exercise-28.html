- 3.30. Exam-style self-test question for workshop.

(a)
Justify why \(\sup_{n\in \N} a_n\) exists.
Solution.
This is a special case of - 1.14.
(b)
Writing \(a:= \sup_{n\in \N} a_n\text{,}\) justify why, for any \(\eps \gt 0\text{,}\) there exists \(N= N(\eps)\) such that \(a_N \gt a-\eps\text{.}\)
Solution.
This is a special case of Task 1.5.c.
(c)
Hence prove that for any \(\eps \gt 0\) there exists \(N= N(\eps)\) such that for all \(n\geq N\) we have
\begin{equation*} a-\eps\lt a_n \leq a\text{.} \end{equation*}
Solution.
This uses the previous part and Task 2.1.a.
(d)
Using the previous part, show how the sequence \((a_n)_n\) satisfies the definition of converging to a real limit.
Solution.
Re-arrange the inequalities from the previous part and use Task 1.22.a.
