Example 3.4. An unbounded sequence which does not tend to \(\infty\).
Set \(a_n:=(-1)^n n\) for each \(n\text{.}\) The sequence has no upper bound, since for any \(M\in \R\text{,}\) taking \(n\gt M\) gives an even positive integer \(2n\) satisfying
\begin{equation*} a_{2n} = (-1)^{2n}2n= 2n\gt n \gt M\text{.} \end{equation*}
We claim that it is not the case that \(a_n\to\infty\) as \(n\to\infty\text{.}\) To establish this we must show that (3.1) does not hold, i.e. that
\begin{equation*} (\exists M \in \R)(\forall N \in \N)(\exists n = n(N) \geq N)[a_n \lt M]\text{.} \end{equation*}
To this end, set \(M:=1\) and let \(N\in \N\) be arbitrary. Then taking \(n=n(N):= 2N+1\) we have
\begin{equation*} a_n = -(2N+1) \lt 1 = M\text{,} \end{equation*}
as required.
