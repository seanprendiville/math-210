Use logical proof to establish properties of the above concepts, for example:

  • Prove that every differentiable function on an open interval is continuous.

  • Prove that every continuous function on a closed interval is uniformly continuous.

  • Show that if \(S \subset \R\) is non-empty and bounded then

    \begin{equation*} (\forall n \in \N)(\exists a_n \in S)\sqbrac{a_n + \tfrac{1}{n} \geq \sup S }. \end{equation*}
