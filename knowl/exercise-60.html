- 7.19. True/false workshop quiz.

In your workshop, you will vote on whether you think each of the following statements is true (thumbs up) or false (thumbs down). For each question, your workshop tutor will ask one of you to briefly justify your vote. This justification can involve giving a counterexample (if false) or outlining which definition/theorem the statement follows from (if true).
(a)
Let \(f: \R\to\R\) and suppose that \(f(x)\to y_0\) as \(x\to x_0\) through \(\R\text{.}\) Then it is also the case that \(f(x)\to y_0\) as \(x\to x_0\) through \((0,\infty)\text{.}\)
(b)
Let \(f: \R\to\R\) and suppose that \(|f(x)|\to |y_0|\) as \(x\to x_0\) through \(\R\text{.}\) Then it is also the case that \(f(x)\to y_0\) as \(x\to x_0\) through \(\R\text{.}\)
(c)
Let \(f: \R\to\R\) and \(x_0,y_0\in \R\text{.}\) Suppose that \(f(x)\to y_0\) as \(x\to x_0\) through \(\R\text{.}\) Then it is also the case that \(f(10x)\to y_0\) as \(x\to x_0/10\) through \(\R\text{.}\)
(d)
All polynomials are uniformly continuous.
(e)
A uniformly continuous function on a closed interval is also continuous.
(f)
By the intermediate value theorem, a continuous function \(f:[a,b]\to\R\) attains all of the values between \(f(a)\) and \(f(b)\text{.}\)
(g)
By the intermediate value theorem, an increasing function \(f:[a,b]\to\R\) attains all of the values between \(f(a)\) and \(f(b)\text{.}\)
(h)
By the intermediate value theorem, a function \(f:\R\to\R\) which is increasing and continuous has \(f(\R)=\R\text{,}\) so that it attains all real values.
(i)
By the extreme value theorem, a continuous function \(f:\R\to\R\) attains its infimum.
(j)
Suppose that for \(f:\R\to\R\) and \(x_0,y_0\in \R\) we have
\begin{equation*} \frac{f(x_0+1/n)-f(x_0)}{(x_0+1/n)-x_0)} \to y_0 \quad\text{as}\quad n \to \infty. \end{equation*}
Then \(f\) is differentiable at \(x_0\) with derivative \(y_0\text{.}\)
(k)
If \(f:[a,b]\to\R\) is differentiable, then \(f\) attains all of the values between \(f(a)\) and \(f(b)\text{.}\)
(l)
If a function is differentiable, then its derivative is continuous.
(m)
The set of differentiable functions \(f:(a,b)\to \R\) form a vector space.
(n)
The composition of two differentiable functions is also differentiable.
(o)
If a bijective function \(f:(a,b)\to(c,d)\) has continuous inverse and \(f'(x_0)\neq 0\) then \(1/f\) is differentiable at \(x_0\text{.}\)
(p)
If a bijective function \(f:(a,b)\to(c,d)\) has continuous inverse and \(f'(x_0)\neq 0\) then \(f^{-1}\) is differentiable at \(x_0\text{.}\)
(q)
If a power series in \(x\) converges absolutely at \(x\text{,}\) then it is differentiable at \(x\text{.}\)
