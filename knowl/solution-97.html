Solution 8.11.c.1.

Since \(\log\) is injective on \((0,\infty)\text{,}\) we have
\begin{equation*} a^x = b \iff \log(a^x) = \log(b). \end{equation*}
By definition we have \(a^x := \exp(x\log a)\text{.}\) Since \(\log = \exp^{-1}\) we deduce that
\begin{equation*} a^x = b \iff x\log a = \log b. \end{equation*}
Since \(a \neq 1\text{,}\) we have \(\log a\neq0\text{,}\) thus we may divide by \(\log a\) to conclude that
\begin{equation*} a^x = b \iff x = \frac{\log b}{\log a}. \end{equation*}
By definition \(\log_a b = \log b / \log a\text{,}\) establishing the claim.
