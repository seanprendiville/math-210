Task 3.1.a.

Define what it means for a sequence of real numbers to tend to positive infinity.
Solution.
Definition 3.2. Tending to \(\pm\infty\).
Given a sequence of real numbers \((a_n)_{n=1}^\infty\) we write
\begin{equation*} a_n \to \infty\text{ as } n \to \infty\text{,} \end{equation*}
or
\begin{equation*} \lim_{n\to \infty} a_n = \infty\text{,} \end{equation*}
if
\begin{equation} (\forall M >0)( \exists N = N(M))( \forall n \geq N)[a_n \geq M].\tag{3.1} \end{equation}
We write
\begin{equation*} a_n \to -\infty\text{ as } n \to \infty\text{,} \end{equation*}
if
\begin{equation*} (\forall M >0)( \exists N =N(M))( \forall n \geq N)[a_n \leq -M]. \end{equation*}
Example 3.3. A sequence tending to \(\infty\).
Set \(a_n: = n\) for each \(n \in \N\text{.}\) Then we claim that \(a_n \to \infty\) as \(n \to \infty\text{.}\) To prove this, we must verify that (3.1) holds. To this end, fix \(M \gt 0\text{.}\) We must find \(N = N(M) \in \N\) such that
\begin{equation} n \geq N(M)\implies a_n\geq M\text{.}\tag{3.2} \end{equation}
Since \(a_n =n \geq N\text{,}\) we may take \(N(M) :=\ceil{M}\) to yield (3.2).
Example 3.4. An unbounded sequence which does not tend to \(\infty\).
Set \(a_n:=(-1)^n n\) for each \(n\text{.}\) The sequence has no upper bound, since for any \(M\in \R\text{,}\) taking \(n:=\max\set{2\ceil{M},2}\) gives an even positive integer satisfying
\begin{equation*} a_n = (-1)^{n}n= n\gt M\text{.} \end{equation*}
We claim that it is not the case that \(a_n\to\infty\) as \(n\to\infty\text{.}\) To establish this we must show that (3.1) does not hold, i.e. that
\begin{equation*} (\exists M >0)(\forall N \in \N)(\exists n = n(N) \geq N)[a_n \lt M]\text{.} \end{equation*}
To this end, set \(M:=1\gt0\) and let \(N\in \N\) be arbitrary. Then taking \(n=n(N):= 2N+1\) we have
\begin{equation*} a_n = -(2N+1) \lt 1 = M\text{,} \end{equation*}
as required.
