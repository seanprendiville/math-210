Proof
By the multiplication rule for limits (Proposition 2.17) we have that
\begin{equation*} \brac{\sum_{n=0}^\infty a_n }\brac{\sum_{n=0}^\infty b_n} = \lim_{N\to \infty } \brac{\brac{\sum_{n=0}^N a_n}\brac{\sum_{n=0}^N b_n}}. \end{equation*}
One can check (using the addition rule for limits Proposition 2.14), that it therefore suffices to show that
\begin{equation} \brac{\sum_{n=0}^N a_n}\brac{\sum_{n=0}^N b_n} - \sum_{n=0}^N \brac{\sum_{r+s = n} a_rb_s} \to 0 \quad \text{ as }\quad N \to \infty \text{.}\tag{4.5} \end{equation}
Our proof of (4.5) starts by multiplying out the partial sums and grouping the terms \(a_rb_s\) according to the value of \(r+s\text{,}\) as follows
\begin{align*} \brac{\sum_{n=0}^N a_n}\brac{\sum_{n=0}^N b_n} \amp = \sum_{0\leq r,s\leq N} a_rb_s = \sum_{n=0}^{2N} \ \sum_{\substack{0 \leq r, s \leq N\\ r+s = n}} a_rb_s \\ \amp = \sum_{n=0}^{N}\ \sum_{r+s = n} a_rb_s + \sum_{n=N+1}^{2N} \ \sum_{\substack{0 \leq r, s \leq N\\ r+s = n}} a_rb_s\text{.} \end{align*}
To prove (4.5) it therefore suffices to prove that
\begin{equation*} \sum_{n=N+1}^{2N} \ \sum_{\substack{0 \leq r, s \leq N\\ r+s = n}} a_rb_s \to 0 \quad\text{ as } \quad N \to \infty. \end{equation*}
We observe that if \(0 \leq r, s \leq N\) and \(r+s = n\) for some \(n \in (N, 2N]\) then either \(r \geq N/2\) or \(s \geq N/2\text{.}\) Hence, on writing
\begin{equation*} A:=\sum_{n=0}^\infty |a_n| \quad \text{and}\quad B:=\sum_{n=0}^\infty |b_n|, \end{equation*}
the triangle inequality and the inequality rule (Proposition 2.5) together give that
\begin{align*} \abs{\sum_{n=N+1}^{2N} \ \sum_{\substack{0 \leq r, s \leq N\\ r+s = n}} a_rb_s} \amp \leq \sum_{r = \ceil{\frac{N}{2}}}^N |a_r|\sum_{s=0}^N |b_s|+ \sum_{r=0}^N |a_r|\sum_{s = \ceil{\frac{N}{2}}}^N |b_s|\\ \amp \leq B\sum_{r = \ceil{\frac{N}{2}}}^\infty |a_r|+ A\sum_{s = \ceil{\frac{N}{2}}}^\infty |b_s|\text{.} \end{align*}
By absolute convergence we have
\begin{equation*} \sum_{r =N}^\infty |a_r| = \sum_{r=0}^\infty|a_r| - \sum_{n=0}^{N-1} |a_r| \to 0 \text{ as } N \to \infty. \end{equation*}
Since convergence is preserved on passing to a subsequence (Proposition 3.20) we deduce that
\begin{equation*} \sum_{r = \ceil{\frac{N}{2}}}^\infty |a_r| \to 0 \text{ as } N \to \infty. \end{equation*}
Therefore, by the multiplication rule for limits (Proposition 2.17) and using the fact that \(B = \sum_s |b_s|\) is a fixed finite real number, we have
\begin{equation*} B\sum_{r = \ceil{\frac{N}{2}}}^\infty |a_r| \to 0 \text{ as } N \to \infty. \end{equation*}
The proof is complete on employing a similar argument to show that
\begin{equation*} A\sum_{s = \ceil{\frac{N}{2}}}^\infty |b_s| \to 0 \text{ as } N \to \infty. \end{equation*}
