- 1.5. Supremum and infimum.

(a)
Define what it means to be a supremum or infimum for a set of real numbers.
Solution.
Definition 1.6. Supremum.
\(M \in \R\) is a supremum for \(S \subset \R\) if \(M\) is an upper bound for \(S\) and any other upper bound \(\tilde M\) for \(S\) is larger, so that \(\tilde M \geq M\text{.}\)
In other words, \(M\) is the least upper bound for \(S\text{,}\) and we denote this quantity \(\sup S\) (if it exists).
Definition 1.7. Infimum.
\(M \in \R\) is an infimum for \(S \subset \R\) if \(M\) is a lower bound for \(S\) and any other lower bound \(\tilde M\) for \(S\) is smaller, so that \(\tilde M \leq M\text{.}\)
In other words, \(M\) is the greatest lower bound for \(S\text{,}\) and we denote this quantity \(\inf S\) (if it exists).
Example 1.8.
  • \(\sup\set{1,2,3} = 3.\)
    Proof.
    3 is an upper bound for \(\set{1,2,3}\text{.}\) We must show it is the least upper bound. Suppose that \(M\) is another upper bound for \(\set{1,2,3}\text{.}\) Then by definition of being an upper bound and the fact that 3 lies in our set, we have that \(3 \leq M\text{.}\) Thus 3 is smaller than any other upper bound, thereby satisfies the definition (Definition 1.6) of being a supremum.
  • \(\sup (-\infty, 1) = 1.\)
    Proof.
    By definition \((-\infty, 1) = \set{x\in \R : x \lt 1}\text{,}\) so 1 is certainly an upper bound for the set. It remains to show that 1 is the smallest upper bound. To this end, suppose that \(M\) is another upper bound for \((-\infty, 1)\text{.}\) Notice that if \(M \lt 1\text{,}\) then any element strictly between \(M\) and 1 (for instance the midpoint \((M+1)/2\) will be an element of \((-\infty, 1)\) which exceeds the upper bound \(M\) (a contradiction). Therefore we must have \(M\geq 1\text{,}\) so that 1 is the least upper bound (as claimed).
    Notice that this example shows that the supremum does not always lie in the set. When the supremum lies in the set we call it the maximum (see Definition 1.11).
  • \(\sup \emptyset\) does not exist.
    Proof.
    Every \(M\in \R\) is an upper bound for \(S = \emptyset\text{,}\) since the negation (1.2) does not hold (we say that a statement of the form \((\forall x \in \emptyset)[\dots]\) is true "vacuously", since the negation cannot hold due to there being no elements in the empty set). No real \(M\) can be the least upper bound for the empty set, since \(M-1\) is a smaller upper bound.
    This means that we can only take the supremum of a non-empty set.
  • If \(S\subset \R\) is not bounded above (for instance \(S=\R\)) then \(S\) has no upper bound, in particular no least upper bound. Thus \(\sup S\) does not exist.
    This means that we can only take the supremum of a set which is bounded above.
    The completeness principle (Theorem 1.15) says that if \(S\) is non-empty and bounded above, then \(\sup S\) exists.
(b)
Given a function \(f:D\to\R\text{,}\) define the notation \(\sup_S f\) and \(\inf_S f\) (when these quantities exist).
Solution.
Given a function \(f:D\to \R\) and a set \(S\subset D\) we write
\begin{equation*} \sup_S f = \sup_{x\in S} f(x) := \sup\set{f(x) : x \in S}. \end{equation*}
This supremum exists if and only if \(f\) is bounded above on \(S\) and \(S\) is non-empty.
We write
\begin{equation*} \inf_S f = \inf_{x\in S} f(x) := \inf\set{f(x) : x \in S}. \end{equation*}
This infimum exists if and only if \(f\) is bounded below on \(S\) and \(S\) is non-empty.
(c)
State and prove how decreasing a supremum gives a non-upper bound.
Solution.
Lemma 1.9. Decreasing a sup gives a non-upper bound.
Let \(S \subset \R\) be non-empty and bounded above. Then
\begin{equation*} (\forall \eps > 0)(\exists x \in S)[x> \sup S-\eps]. \end{equation*}
Proof.
Let \(\eps\gt0\) be arbitrary; we are required to find \(x\in S\) such that \(x\gt \sup S-\eps\text{.}\) Aiming for a contradiction, let us suppose that this does not hold, so that for all \(x\in S\) we have \(x\leq \sup S-\eps\text{.}\) But then \(\sup S-\eps\) is an upper bound for \(S\) which is strictly smaller than \(\sup S\text{.}\) Since \(\sup S\) is the smallest upper bound for \(S\text{,}\) we have obtained our desired contradiction.
