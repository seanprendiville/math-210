Proof
If \(x_0 \in S\) then we have that
\begin{equation*} f(x_0) \leq \sup_{ S} f\quad \text{and} \quad g(x_0) \leq \sup_{S} g. \end{equation*}
Hence the sum
\begin{equation*} \sup_{S} f +\sup_{ S} g \end{equation*}
is an upper bound for the set
\begin{equation*} \set{f(x_0) + g(x_0) : x_0 \in S}. \end{equation*}
By definition of the supremum as the least upper bound (DefinitionÂ 1.5) we deduce the subadditivity of the supremum. The argument for the infimum is analogous.
