Task 8.10.a.
State Taylor’s theorem (the proof is non-examinable).
Solution.
Theorem 8.11. Taylor’s theorem.
Let \(f : (a,b) \to \R\) be a function which is \(n\)-times differentiable and let \(x_0, x \in (a,b)\text{.}\) Then
\begin{align*} \Bigl|f(x) - \amp\bigl[f(x_0) + (x-x_0)f'(x_0) + \dots + \trecip{(n-1)!} (x-x_0)^{n-1} f^{(n-1)}(x_0)\bigr]\Bigr|\\ \amp \leq \trecip{n!}|x-x_0|^n\sup\set{|f^{(n)}(t)| : t \text{ is between } x \text{ and } x_0}. \end{align*}
The following proof of Taylor’s theorem is not examinable.
We begin by proving a lemma.
Lemma 8.12. Key estimate for Taylor’s theorem.
Let \(f : (a, b) \to \R\) be a differentiable function. Let \(c \in [0, \infty)\text{,}\) \(n \in \set{0, 1, 2, \dots}\) and \(x_0,x \in (a,b)\) with \(x_0 \leq x\text{.}\) Suppose that for all \(t \in [x_0,x]\) we have
\begin{equation*} |f'(t)| \leq c(t-x_0)^n. \end{equation*}
Suppose in addition that \(f(x_0) = 0\text{.}\) Then for all \(t\in [x_0,x]\) we have
\begin{equation*} |f(t)| \leq \trecip{n+1} c (t-x_0)^{n+1}\text{.} \end{equation*}
Proof.
We are assuming that for all \(t \in [x_0,x]\) we have
\begin{equation*} -c(t-x_0)^n \leq f'(t) \leq c(t-x_0)^n. \end{equation*}
Since functions with non-negative derivative are increasing (Corollary 8.9), it follows that the functions
\begin{equation*} f(t) + \trecip{n+1}c(t-x_0)^{n+1} \qquad \text{and} \qquad \trecip{n+1}c(t-x_0)^{n+1} -f(t) \end{equation*}
are increasing as \(t\) ranges over \([x_0,x]\text{.}\) Both of these functions evaluate to zero at \(t=x_0\text{,}\) hence for all \(t\in [x_0,x]\) we have
\begin{equation*} f(t) + \trecip{n+1}c(t-x_0)^{n+1}\geq 0 \geq f(t) - \trecip{n+1}c(t-x_0)^{n+1}. \end{equation*}
Combining inequalities we deduce that for any \(t\in [x_0,x]\) we have
\begin{equation*} -\trecip{n+1} c (t-x_0)^{n+1}\leq f(t) \leq \trecip{n+1} c (t-x_0)^{n+1}. \end{equation*}
This completes our proof.
We are now in a position to prove Taylor’s theorem.
Proof of Taylor’s theorem.
We may assume that \(x\geq x_0\text{,}\) for if \(x\leq x_0\) we obtain the result on applying the following argument to \(g: (-b, -a) \to \R\) given by \(g(x):=f(-x)\text{.}\)
Define the following auxiliary function
\begin{align*} \amp F(x) := f(x)-\\ \amp \sqbrac{f(x_0) + (x-x_0)f'(x_0) + \dots + \trecip{(n-1)!} (x-x_0)^{n-1} f^{(n-1)}(x_0)}\text{.} \end{align*}
Then \(F\) is \(n\) times differentiable on \((a,b)\text{.}\) Moreover, for each \(0 \leq m \lt n\) we have
\begin{equation*} F^{(m)}(x_0) = 0\text{,} \end{equation*}
whilst for \(m=n\) we have
\begin{equation*} F^{(n)}(t) = f^{(n)}(t) \quad \text{for all } t \in (a,b). \end{equation*}
Letting \(t\in [x_0,x]\text{,}\) the mean value theorem (Theorem 8.7) gives some \(x_t\) between \(x_0\) and \(t\) such that
\begin{equation*} |F^{(n-1)}(t)| \leq (t-x_0) \abs{f^{(n)}(x_t)} . \end{equation*}
Write
\begin{equation*} c := \sup\set{|f^{(n)}(t)| : t \text{ is between } x \text{ and } x_0}\text{.} \end{equation*}
Then we have shown that for all \(t \in [x_0,x]\) we have
\begin{equation*} \abs{F^{(n-1)}(t)} \leq c(t-x_0). \end{equation*}
As a consequence of Lemma 8.12, we deduce that for \(t \in [x_0,x]\) we have
\begin{equation*} \abs{F^{(n-2)}(t)} \leq \trecip{2}c(t-x_0)^2. \end{equation*}
Re-applying Lemma 8.12, for \(t \in [x_0,x]\) we have
\begin{equation*} \abs{F^{(n-3)}(t)} \leq \trecip{3!}c(t-x_0)^3. \end{equation*}
Iteratively applying Lemma 8.12, for \(t \in [x_0,x]\) we obtain
\begin{equation*} \abs{F(t)} \leq \trecip{n!}c(t-x_0)^n. \end{equation*}
