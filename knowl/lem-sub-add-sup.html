Lemma 9.14. Subadditivity of sup and inf.
\(f,g : S \to \R\)\(S\text{.}\)
\begin{align*} \sup_{ S} \brac{f + g}\amp \leq \sup_{ S} f +\sup_{ S} g;\\ \inf_{ S} \brac{f + g} \amp \geq \inf_{ S} f +\inf_{ S} g. \end{align*}
Proof.
If \(x_0 \in S\) then we have that
\begin{equation*} f(x_0) \leq \sup_{ S} f\quad \text{and} \quad g(x_0) \leq \sup_{S} g. \end{equation*}
Hence the sum
\begin{equation*} \sup_{S} f +\sup_{ S} g \end{equation*}
is an upper bound for the set
\begin{equation*} \set{f(x_0) + g(x_0) : x_0 \in S}. \end{equation*}
By definition of the supremum as the least upper bound (DefinitionÂ 1.6) we deduce the subadditivity of the supremum. The argument for the infimum is analogous.
