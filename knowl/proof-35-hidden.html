The identity function \(x \mapsto x\) is continuous, since denoting this function by \(f\text{,}\) we have \(a_n \to x_0\) implies \(f(a_n) \to f(x_0)\text{.}\) By the multiplication rule (Proposition 2.10) it follows that \(x^2\) is continuous. Iterating this argument, we see that \(x^3 = x\cdot x^2\) is continuous. It follows from induction that \(x^n\) is continuous for any \(n \in \N\text{.}\) Since constant functions are continuous (Example 5.14), it again follows from the multiplication rule that \(c_n x^n\) is continuous for any \(c_n \in \R\text{.}\) By the addition rule (Proposition 2.8), the sum
\begin{equation*} c_0 + c_1x+ \dots + c_n x^n \end{equation*}
is continuous. Hence polynomials are continuous.
