- 8.8. Monotonicity and the sign of the derivative.

Prove how a function with non-negative derivative is increasing.
Solution.
Corollary 8.9.
Let \(f : [a,b] \to \R\) be a continuous function which is differentiable on \((a,b)\text{,}\) with \(a \lt b\text{.}\)
  1. If \((\forall x \in (a,b))[f'(x) \geq 0]\) then \(f\) is increasing.
  2. If \((\forall x \in (a,b))[f'(x) \gt 0]\) then \(f\) is strictly increasing.
  3. If \((\forall x \in (a,b))[f'(x) \leq 0]\) then \(f\) is decreasing.
  4. If \((\forall x \in (a,b))[f'(x) \lt 0]\) then \(f\) is strictly decreasing.
  5. If \((\forall x \in (a,b))[f'(x) = 0]\) then \(f\) is constant.
Proof.
Let us first suppose that \((\forall x \in (a,b))[f'(x) \geq 0]\text{.}\) Let \(x_1, x_2 \in [a,b]\) with \(x_1 \lt x_2\text{.}\) Our task is to prove that \(f(x_1) \leq f(x_2)\text{.}\) Applying the mean value theorem to the function \(f: [x_1, x_2] \to \R\text{,}\) there exists \(x_0 \in (x_1, x_2)\) such that
\begin{equation*} f(x_2)-f(x_1) = f'(x_0)(x_2-x_1) \geq 0, \end{equation*}
as required. Notice that if \((\forall x \in (a,b))[f'(x) \gt 0]\) then \(f(x_2) -f(x_1) = f'(x_0)(x_2-x_1) \gt 0\text{,}\) which implies that the function is strictly increasing. The conclusions involving decreasing functions follow on applying the increasing results to \(-f\text{.}\) If \((\forall x \in (a,b))[f'(x) = 0]\) then \(f\) is both increasing and decreasing, hence for all \(x \in [a,b]\) we have \(f(x) = f(b)\text{.}\) Thus in the latter case \(f\) is constant.
