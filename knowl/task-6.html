Task 1.10.a.
Define the maximum and minimum for a set of real numbers (when these quantities exist).
Solution.
Definition 1.11. Maximum.
We say that a set of real numbers \(S \subset \R\) has a maximum if \(\sup S\) exists and \(\sup S \in S\text{.}\) In this situation, we write \(\max S\) for \(\sup S\) and call it the maximum.
Definition 1.12. Minimum.
We say that a set of real numbers \(S \subset \R\) has a minimum if \(\inf S\) exists and \(\inf S \in S\text{.}\) In this situation, we write \(\min S\) for \(\inf S\) and call it the minimum.
Example 1.13.
  • \(\max \set{1,2,3} = 3\text{.}\)
  • \(\max (-\infty, 1)\) does not exist.
  • For any finite non-empty set, both \(\max S\) and \(\min S\) exist.
