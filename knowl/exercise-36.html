- 4.17. Alternating series test.

State the "alternating series test" for convergence of infinite series (the proof is non-examinable).
Solution.
Proposition 4.18. Alternating series test.
Let \((a_n)_n\) denote a decreasing sequence of non-negative reals, so that \(a_n \geq a_{n+1} \geq 0\) for all \(n\text{.}\) Suppose that \(a_n \to 0\text{.}\) Then the series
\begin{equation*} a_1-a_2+a_3-a_4 + \dots = \sum_{n=1}^\infty (-1)^{n+1} a_n \end{equation*}
converges.
Proof (NON-EXAMINABLE).
Before proving the alternating series test, we prove the following key inequality, whose statement and proof are both NON-EXAMINABLE.
Lemma 4.19.
For any decreasing sequence \(b_1 \geq b_2 \geq \dots \geq b_n\geq 0\) of non-negative reals, with finitely many terms, we have the inequalities
\begin{equation} 0 \leq b_1 - b_2 + \dots + (-1)^{n+1} b_n \leq b_1.\tag{4.3} \end{equation}
Proof of Lemma 4.19.
We prove this by induction on \(n\text{,}\) the number of terms in the sequence. The basis case, when \(n=1\text{,}\) follows from the inequalities
\begin{equation*} 0 \leq b_1 \leq b_1. \end{equation*}
For the inductive step, we may suppose that
\begin{equation*} 0 \leq b_2 -b_3 + \dots + (-1)^n b_{n} \leq b_2. \end{equation*}
It follows that
\begin{equation*} b_1 - b_2 + \dots + (-1)^{n+1} b_n = b_1- \brac{b_2 -b_3+ \dots + (-1)^{n} b_n} \geq b_1-b_2 \geq 0 \end{equation*}
and that
\begin{equation*} b_1 - b_2 + \dots + (-1)^{n+1} b_n = b_1- \brac{b_2 -b_3+ \dots + (-1)^{n} b_n} \leq b_1- 0= b_1. \end{equation*}
Proof of alternating series test (Proposition 4.18).
Applying the inequality (4.3), for any \(N \gt M\) we have
\begin{align*} \abs{\sum_{n=1}^N(-1)^{n+1} a_n -\sum_{n=1}^M(-1)^{n+1} a_n } \amp= \abs{ \sum_{n=M+1}^N (-1)^{n+1} a_n}\\ \amp = a_{M+1} - a_{M+2} + \dots + (-1)^{N-M+1} a_N\\ \amp \leq a_{M+1}. \end{align*}
Since \(a_{n} \to 0\) as \(n \to \infty\) there exists \(N_0 = N_0(\eps)\) such that for all \(n \geq N_0(\eps)\) we have \(a_{n} \leq \eps\text{.}\) In particular, if \(N \gt M \geq N_0(\eps)\) then
\begin{equation*} \abs{\sum_{n=1}^N(-1)^{n+1} a_n -\sum_{n=1}^M(-1)^{n+1} a_n } \leq a_{M+1} \leq \eps. \end{equation*}
It follows that the sequence of partial sums
\begin{equation*} \sum_{n=1}^N (-1)^{n+1} a_n \end{equation*}
is a Cauchy sequence (Definition 3.27), hence convergent (see Proposition 3.28).
Example 4.20. An example of applying the alternating series test.
We claim that the series
\begin{equation*} \sum_{n=1}^\infty \frac{(-1)^{n+1}}{n} \end{equation*}
converges, even though it is not absolutely convergent. To establish this we employ the alternating series test (Proposition 4.18) with \(a_n := \frac{1}{n}\text{,}\) so that it suffices to establish that the sequence \((a_n)_n\) is non-negative, decreasing and converges to 0 (we leave proving this to the reader).
Example 4.21. An example of an alternating series which converges, but whose terms are not decreasing in absolute value.
Consider the sequence
\begin{equation*} a_n := \begin{cases} 2^{-n} \amp \text{if } n \text{ odd};\\ 3^{-n} \amp \text{if } n \text{ even}. \end{cases} \end{equation*}
Then the sequence \((a_n)_{n=1}^\infty\) is not decreasing, yet the corresponding alternating series converges, since by addition of convergent series (Proposition 4.13) we have
\begin{equation*} \sum_{n=1}^\infty (-1)^{n+1} a_n = \sum_{m=1}^\infty 2^{-(2m-1)} - \sum_{m=1}^\infty 3^{2m} = \frac{2}{3}-\frac{1}{8}. \end{equation*}
