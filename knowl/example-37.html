Example 7.9. A function which is once but not twice differentiable.
Define \(f : \R \to \R\) at \(x \in \R\) by \(f(x) := \trecip{2}x|x|\text{.}\) One can check that this function is differentiable on \(\R\text{.}\) Moreover, for all \(x \in \R\) we have (check this!)
\begin{equation*} f'(x) = |x|. \end{equation*}
