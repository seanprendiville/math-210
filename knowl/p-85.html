Next, let us suppose that \(f(x_n) = f(x_0)\) for infinitely many values of \(n\text{.}\) Write \(k_1 \lt k_2 \lt \dots\) for those indices for which \(f(x_{k_n}) = f(x_0)\text{.}\) Write \(\ell_1 \lt \ell_2 \lt \dots\) for the (possibly finite) remaining indices with \(f(x_{l_n}) \neq f(x_0)\text{.}\) The sequence

\begin{equation} n\mapsto \frac{g\bigbrac{f(x_{\ell_n})}-g\bigbrac{f(x_0)}}{f(x_{\ell_n})-f(x_0)}\tag{7.4} \end{equation}

is either a finite sequence, or converges as in the proof of (7.3). In either case, the sequence is bounded, so there exists \(M \in \R\) such that for all \(\ell_n\) we have

\begin{equation*} \abs{\frac{g\bigbrac{f(x_{\ell_n})}-g\bigbrac{f(x_0)}}{f(x_{\ell_n})-f(x_0)} } \leq M. \end{equation*}

Combining this with (7.2), for any \(n \in \N\) we have

\begin{equation} \abs{\frac{g\bigbrac{f( x_n)} - g\bigbrac{f(x_0)}}{ x_n-x_0}} \leq M\abs{\frac{f(x_{n}) - f(x_0)}{x_{n} - x_0}}.\tag{7.5} \end{equation}

Since \(x_{k_n} \to x_0\) we have

\begin{equation*} f'(x_0) = \lim_{n \to \infty} \frac{f(x_{k_n}) - f(x_0)}{x_{k_n} - x_0} = 0. \end{equation*}

So it follows that

\begin{equation*} \lim_{n \to \infty} \frac{f(x_{n}) - f(x_0)}{x_{n} - x_0} = f'(x_0) = 0. \end{equation*}

Combining this with (7.5) gives

\begin{equation*} \lim_{n\to \infty}\abs{\frac{g\bigbrac{f( x_n)} - g\bigbrac{f(x_0)}}{ x_n-x_0}} = 0. \end{equation*}

Hence (7.1) also follows in this case.
