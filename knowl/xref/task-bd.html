Task 1.1.b.
Define what it means for a set/function to be bounded above, bounded below and bounded.
Solution.
Definition 1.4. Bounded above/below.
We say that a set \(S\subset\R\) is bounded above if it has an upper bound, that is
\begin{equation} (\exists M \in \R)(\forall x \in S)[x \leq M].\tag{1.3} \end{equation}
Similarly, a function \(f : D \to \R\) is bounded above if it has an upper bound, that is
\begin{equation*} (\exists M \in \R)(\forall x \in D)[f(x) \leq M]. \end{equation*}
A set/function is bounded below if it has a lower bound.
A set/function is bounded if it is both bounded above and bounded below. For sets, this is equivalent to:
\begin{equation} (\exists M \in \R)(\forall x \in S)[|x| \leq M].\tag{1.4} \end{equation}
For functions, this is equivalent to:
\begin{equation*} (\exists M \in \R)(\forall x \in D)[|f(x)| \leq M]. \end{equation*}
