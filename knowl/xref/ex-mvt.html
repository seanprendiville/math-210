- 8.6. The mean value theorem.

State and prove the mean value theorem.
Solution.
Theorem 8.7. Mean value theorem.
Let \(f : [a,b] \to \R\) be a continuous function which is differentiable on \((a,b)\text{,}\) with \(a \lt b\text{.}\) Then there exists \(x_0 \in (a,b)\) such that
\begin{equation} f(b) - f(a) = f'(x_0)(b-a).\tag{8.1} \end{equation}
Proof.
We deduce this result from Rolle’s theorem (Theorem 8.5). The idea of the proof is to try and find a continuous function \(g:[a,b] \to \R\) which is differentiable on \((a,b)\) with \(g(a) = g(b)\) and such that the identity \(g'(x_0) = 0\) implies (8.1). To this end set
\begin{equation*} g(x) :=f(x) - \frac{f(b)-f(a)}{b-a} (x-a), \qquad (x \in [a,b]). \end{equation*}
This is continuous on \([a,b]\) since constant functions are continuous (Example 5.22), the identity function is continuous (Example 5.23) and so is the sum/product of continuous functions (Proposition 6.2). The function \(g\) is differentiable on \((a,b)\) since constant functions are differentiable (Example 7.3), the identity function is differentiable (Example 7.4) and so is the sum/product of differentiable functions (Proposition 6.2). We have \(g(a) = f(a)\) and \(g(b) = f(a)\text{.}\) We may therefore apply Rolle’s theorem to deduce the existence of \(x_0 \in (a,b)\) such that \(g'(x_0) = 0\text{.}\) Employing the sum/product rules for differentiation, we have that
\begin{equation*} g'(x_0) = f'(x_0) -\frac{f(b)-f(a)}{b-a}. \end{equation*}
The result follows on re-arranging the identity
\begin{equation*} 0 = f'(x_0) -\frac{f(b)-f(a)}{b-a}. \end{equation*}
