Proposition 1.30. Convergent sequences are bounded.
Suppose that \((a_n)\) is a convergent sequence of reals. Then \((a_n)\) is a bounded sequence.
Proof.
By definition of a bounded function (Definition 1.2) we must show that
\begin{equation*} (\exists M \in \R)(\forall n \in \N)[|a_n| \leq M]. \end{equation*}
Since the sequence is convergent, there exists \(a \in \R\) such that \(a_n \to a\text{.}\) Taking \(\eps :=1\) in the definition of the limit (Definition 1.23), there exists \(N = N(1)\) such that for all \(n \geq N\) we have \(|a_n - a| \leq 1\text{.}\) Therefore, when \(n \geq N\text{,}\) the triangle inequality gives
\begin{equation*} |a_n| = |a_n - a + a| \leq |a_n-a| + |a| \leq 1 + |a|. \end{equation*}
Rounding up if necessary, we may assume that \(N \in \N\text{,}\) so we can set
\begin{equation*} M:= \max\set{|a_1|, \dots, |a_{N}|, 1+|a|}. \end{equation*}
Then for all \(n \in \N\) we have \(|a_n| \leq M\text{.}\)
