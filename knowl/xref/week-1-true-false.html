- 1.31. True/false workshop quiz.

In your workshop, you will vote on whether you think each of the following statements is true (thumbs up) or false (thumbs down). For each question, your workshop tutor will ask one of you to briefly justify your vote. This justification can involve giving a counterexample (if false) or outlining which definition/theorem the statement follows from (if true).
(a)
If \(M\in \R\) is not an upper bound for \(S\subset \R\) then
\begin{equation*} (\forall x \in S)[x > M]. \end{equation*}
(b)
There exists a set for which 0 is both an upper bound and lower bound.
(c)
If a function \(f: \R \to \R\) is bounded above, then it is also bounded below.
(d)
\(M\in \R\) is a supremum for \(S\subset \R\) if it is smaller than any other upper bound.
(e)
If \(f: [0,1] \to \R\) is given by \(f(x) = 2x\) then \(\sup_{[0,1]} f= 2\text{.}\)
(f)
If 0 is an infimum for \(S\subset \R\text{,}\) then \(S\) contains an element smaller than \(0.00000001\text{.}\)
(g)
A sequence which is bounded above has a maximum.
(h)
One consequence of the completeness principle is that a non-empty set of negative real numbers has a supremum.
(i)
If a set has a supremum then it is bounded.
(j)
The function \(f(x) = x^2\) is a sequence.
(k)
The natural numbers are a sequence.
(l)
If a sequence converges to 0, then all of the terms lie in the neighbourhood \([-0.01, 0.01]\text{.}\)
(m)
If a sequence \((a_n)_{n=1}^\infty\) converges to \(a\in \R\text{,}\) then
\begin{equation*} (\exists N \in \N)(\forall \eps \gt 0)(\forall n \geq N)[|a_n-a|\leq \eps]. \end{equation*}
(n)
The sequence \(a_n:=\frac{n}{n+1}\) converges to 1 as \(n \to\infty\text{.}\)
(o)
If a sequence converges then it has a supremum.
