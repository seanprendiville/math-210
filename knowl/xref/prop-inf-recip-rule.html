Proposition 3.11. Infinite reciprocal rule.
Let \((a_n)_{n=1}^\infty\) be a sequence of positive real numbers. Suppose that \(\displaystyle \lim_{n \to \infty} a_n\) exists. Then
\begin{equation*} \lim_{n \to \infty} \brac{\frac{1}{a_n}} = \begin{cases} \infty \amp \text{if }\quad \lim_{n\to \infty} a_n = 0,\\ 0 \amp \text{if }\quad \lim_{n\to \infty} a_n = \infty. \end{cases} \end{equation*}
Proof.
Let \(\displaystyle a : = \lim_{n \to \infty}a_n\text{.}\)
Case 1 (\(a = 0\)).
Fix \(M \in \R\) and set \(\eps := \recip{1+|M|} \gt 0\text{.}\) Then by Definition 1.23 there exists \(N = N(\eps)\) such that for all \(n \geq N\) we have \(a_n = |a_n| \leq \eps\text{.}\) Hence if \(n \geq N(\eps)\) then
\begin{equation*} \trecip{a_n} \geq \trecip{\eps} \geq M. \end{equation*}
It follows that \(1/a_n \to \infty\) according to Definition 3.2.
Case 2 (\(a=\infty\)).
Fix \(\eps \gt 0\) and set \(M := 1/\eps\text{.}\) Then by Definition 3.2 there exists \(N(M) \in \N\) such that for all \(n \geq N(M)\) we have \(a_n \geq M\text{.}\) Hence for all \(n \geq N(M)\) we have \(|1/a_n| = 1/a_n \leq 1/M = \eps\text{.}\) It follows that \(1/a_n \to 0\) according to Definition 1.23.
