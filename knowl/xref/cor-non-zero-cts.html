Proposition 5.19. Inequalities spread around a limit.
Let \(f : D \to \R\) be a real-valued function on a real domain \(D \subset \R\) and let \(x_0 \in [-\infty, \infty]\text{.}\) Suppose that there exists \(y\) such that
\begin{equation*} f(x)\to y\quad\text{as}\quad x\to x_0 \quad\text{through}\quad D. \end{equation*}
If \(y\gt z\) then there exists a neighbourhood
 2 
If \(x_0\in\R\) then \(N\) is a neighbourhood of \(x_0\) if \(N=[x_0-\eps, x_0+\eps]\) for some \(\eps \gt 0\text{.}\) If \(x_0 = +\infty\) then \(N\) is a neighbourhood if \(N=[M, \infty)\) for some \(M\in \R\text{.}\) If \(x_0 = -\infty\) then \(N\) is a neighbourhood if \(N=(-\infty, M]\) for some \(M\in \R\text{.}\)
\(N\) of \(x_0\) such that for all \(x \in D\cap N\) we have \(f(x) \gt z \text{.}\) (An analogous result hold if \(y \lt z\text{,}\) with inequalities reversed.)
Proof.
We prove the case in which \(x_0, y, z \in \R\) and leave the infinite cases to the reader.
Take \(\eps := \frac{y-z}{2}\) in (5.4). Then by the triangle inequality, for \(x \in D\) satisfying \(|x-x_0| \leq \delta = \delta(\eps)\) we have
\begin{align*} f(x) = y + f(x) - y \geq y- |f(x)-y| \amp \geq y - \eps\\ \amp = z + \eps \gt z. \end{align*}
