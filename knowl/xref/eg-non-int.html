Example 10.3.
The function \(1_{\Q \cap [0,1]}\) is not Riemann integrable on \([0,1]\text{.}\)
As in Example 5.26, the key observation in establishing our claim is to note that every real number \(x_0\) has a sequence of rationals \(q_n \in \Q\) such that \(q_n \to x_0\text{.}\) For instance, one may take
\begin{equation*} q_n := \frac{\floor{nx_0}}{n} \in \left(x_0 -\frac{1}{n}, x_0\right]\text{,} \end{equation*}
and employ the two-sided sandwich rule (Proposition 2.10).
One consequence of this observation is that any non-empty open interval contains a rational number, since one may approximate the mid-point by a rational. It follows that if \(0 \leq x \lt y \leq 1\) then \((x,y)\) contains a rational \(q\in \Q\) and \((x+\sqrt{2},y+\sqrt{2})\) also contains a rational \(\tilde q \in \Q\text{.}\) Hence \((x,y)\) contains both a rational \(q\) and an irrational \(q-\sqrt{2}\text{.}\) It follows that
\begin{equation*} \sup_{[x,y]} 1_{\Q\cap [0,1]} = 1 \quad \text{and}\quad \inf_{[x,y]} 1_{\Q\cap [0,1]} = 0. \end{equation*}
Hence for any subdivision \(0 = x_0 \lt \dots \lt x_N = 1\) we have
\begin{equation*} U(f;x_0,\dots, x_N) = \sum_{n=1}^N (x_n-x_{n-1}) \sup_{[x_{n-1}, x_n]} 1_{\Q\cap [0,1]} = \sum_{n=1}^N (x_n-x_{n-1}) = 1. \end{equation*}
Thus \(U(f;[0,1]) = 1\text{.}\) Similarly
\begin{equation*} L(f;[0,1]) = 0. \end{equation*}
Therefore the function \(1_{\Q\cap [0,1]}\) is not Riemann integrable.
