Proposition 3.8.
If \((a_n)_{n=1}^\infty\) is a sequence of real numbers with \(a_n\to \infty\) as \(n \to \infty\text{,}\) then for any \(c>0\) we have \(ca_n\to \infty\) as \(n\to \infty\text{.}\)
Proof.
By definition of tending to infinity (DefinitionÂ 3.2), given \(M\in \R\) we must find \(N = N(M)\) such that if \(n\geq N(M)\) then \(ca_n\geq M\text{.}\) Since \(c\gt 0\text{,}\) the latter is equivalent to \(a_n\geq M/c\text{.}\) Since \(a_n\to\infty\text{,}\) we can find \(N_1 = N_1(M/c)\) such that if \(n\geq N_1(M/c)\) then \(a_n \geq M/c\text{.}\) Hence the proof is complete on setting \(N(M) := N_1(M/c)\text{.}\)
