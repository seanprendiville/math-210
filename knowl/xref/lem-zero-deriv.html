Proposition 8.3. The derivative is zero at a local max/min.
Suppose that \(f : (a,b) \to \R\) is differentiable at \(x_0 \in (a,b)\text{.}\) Suppose that \(x_0\) is a local maximum or local minimum for \(f\text{.}\) Then \(f'(x_0) = 0\text{.}\)
Proof.
Let us suppose that \(x_0\) is a local maximum for \(f\text{;}\) the argument for the local minimum is similar. By definition of a local maximum (see Definition 8.2) there exists \(\delta \gt 0\) such that for all \(x \in (a,b)\cap [x_0-\delta, x_0+\delta]\) we have \(f(x_0) \geq f(x)\text{.}\) Decreasing \(\delta \gt 0\) if necessary
 1 
For instance, take replace \(\delta\) with \(\min\set{\delta, (x_0 - a)/2, (b-x_0)/2}\text{.}\)
, we may assume that
\begin{equation*} [x_0-\delta, x_0+\delta] \subset (a,b)\text{.} \end{equation*}
Hence
\begin{equation*} \frac{f(x)-f(x_0)}{x-x_0} \begin{cases} \geq 0 \amp \text{ if } x_0-\delta \leq x \lt x_0,\\ \leq 0 \amp \text{ if } x_0 \lt x \leq x_0+\delta.\end{cases} \end{equation*}
Setting \(a_n := x_0-\trecip{n}\delta\text{,}\) the sequential definition of the limit Definition 5.12, the definition of the derivative (Definition 7.2) and the inequality rule (Proposition 2.5) give that
\begin{equation*} f'(x_0) = \lim_{n\to \infty}\frac{f(a_n)-f(x_0)}{a_n-x_0} \geq 0. \end{equation*}
Similarly, setting \(b_0:= x_0+\trecip{n}\delta\) gives
\begin{equation*} f'(x_0) = \lim_{n\to \infty}\frac{f(b_n)-f(x_0)}{a_n-x_0} \leq 0. \end{equation*}
Thus \(f'(x_0) = 0\text{.}\)
