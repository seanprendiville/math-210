- 1.34. Exam-style self-test question for workshop.

(a)
Define what it means for a sequence \((a_n)_{n=1}^\infty\) to be bounded.
Solution.
This combines the definition of a sequence as a function (Task 1.20.a) with what it means for a function to be a bounded (Task 1.1.b).
(b)
Suppose that \(a_n\to 1\) as \(n\to\infty\text{.}\) Prove that for infinitely many values of \(n\) we have \(a_n \leq 2.\)
Solution.
This appears to be an "unseen" question, but really it is a special case of Task 1.29.a. In particular, arguing exactly as in the first half of the proof of Proposition 1.30 with \(a=1\text{,}\) we find that there exists \(N \in \N\) such that for all \(n\geq N\) we have \(a_n \leq 2\text{.}\)
