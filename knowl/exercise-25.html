- 3.23. Erdős-Szekeres and Bolzano-Weierstrass.

(a)

State the Erdős-Szekeres theorem (the proof is non-examinable).
Solution.
Theorem 3.24. Erdős-Szekeres.
Every sequence of real numbers has a monotonic subsequence.
Non-examinable proof.
A useful trick in proving this theorem is to define a notion that we call being a "viewpoint":
We say that \(k \in \N\) is a viewpoint 2  if \((\forall n \geq k)[a_k\geq a_n]\text{.}\) If there are infinitely many viewpoints \(k_1 \lt k_2 \lt k_3 \lt \dots\) then the subsequence \((a_{k_n})_{n=1}^\infty\) is decreasing. We may therefore suppose that there are only finitely many viewpoints. In particular, there exists \(k_0\geq 0\) such that no \(n \gt k_0\) is a viewpoint, so that there exists a higher point \(a_m\) to the right of \(a_n\text{.}\) More precisely:
\begin{equation*} (\forall n \gt k_0)(\exists m(n) \gt n)[ a_{m(n)} \gt a_n ]. \end{equation*}
Let us recursively define \(k_n\) as follows: \(k_1 := k_0+1\) and if \(k_n\) has been defined then we set \(k_{n+1} := m(k_n)\text{.}\) Then \(k_1 \lt k_2 \lt \dots\) is strictly increasing and for each \(n\in \N\) we have
\begin{equation*} a_{k_{n+1}} = a_{m(k_n)} \gt a_{k_n}. \end{equation*}
Hence \((a_{k_n})_{n=1}^\infty\) is an increasing sequence.

(b)

State and prove the Bolzano-Weierstrass theorem as a corollary.
Solution.
Corollary 3.25. Bolzano-Weierstrass theorem.
Every bounded sequence of real numbers has a convergent subsequence.
Proof.
Let \((a_n)_{n =1}^\infty\) be a bounded sequence of real numbers. By the definition of convergence (Definition 1.23), we are required to demonstrate the existence of a real number \(a\) and a subsequence \((a_{k_n})_{n = 1}^\infty\) such that \(a_{k_n} \to a\) as \(n \to \infty\text{.}\) By the Erdős-Szekeres theorem (Theorem 3.24) there exists a monotonic subsequence \((a_{k_n})_{n = 1}^\infty\text{.}\) Since this subsequence is bounded and monotonic, it tends to a real number \(a \in \R\) by the monotone convergence principle (Proposition 2.3).
