In this week's workshop you will be asked to write-up (under exam conditions) two of the following:

  • This question is a variant of Task 7.1.a. Let \(f,g :\R\to\R\) be functions which are both differentiable at 0 with \(f'(0)=1\) and \(g'(0)=0\text{.}\) Without using the product rule for differentiation, prove that the product \(fg\) satisfies the definition of differentiability at 0, with \((fg)'(0) = g(0)\text{.}\)

  • This question is a variant of Task 7.6.e. Let \(f,g:[0,1]\to \R\) be continuous functions which are differentiable on \((a,b)\text{.}\) Suppose that \(f'(x) \geq g'(x)\) for all \(x\in[0,1]\text{.}\) Use the mean value theorem to prove that \(f(1) + g(0) \geq f(0) + g(1)\text{.}\)

  • This question is a variant of Task 7.12.a. Let \(f:\R\to\R\) be a function which is \(n\)-times differentiable and suppose that

    \begin{equation*} f(0) = f'(0) = \dots = f^{(n-1)}(0)=0\text{.} \end{equation*}
    Without using the statement of Taylor's theorem, but by modifying the proof, prove that for all \(x\gt 0\) there exists \(x_n \in (0,x)\) such that
    \begin{equation*} \abs{f(x)} \leq x^n \abs{f^{(n)}(x_n)}. \end{equation*}
