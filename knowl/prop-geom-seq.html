Proposition 3.16.
Let \(x \in \R\text{.}\) Then
\begin{equation*} \lim_{n\to \infty} x^n = \begin{cases} \infty \amp \text{ if } x \gt 1;\\ 1 \amp \text{ if } x = 1;\\ 0 \amp \text{ if } -1 \lt x \lt 1;\\ \text{does not exist} \amp \text{ if } x \leq -1. \end{cases} \end{equation*}
