Proposition 9.19. Indicator of an interval.
Let \(a, b\in \R\) with \(a\lt b\text{.}\) Let \(c, d \in [a,b]\) with \(c \leq d\text{.}\) For any interval
\begin{equation*} I \in \set{(c,d), [c,d), (c,d], [c,d]} \end{equation*}
we have
\begin{equation*} \int_a^b 1_I = d- c = \mathrm{Length}(I). \end{equation*}
Proof.
We prove the result when \(a \lt c\lt d\lt b\text{,}\) the remaining cases being similar. We may also assume that \(I = (c,d)\text{,}\) since we can change the value of a function on a finite set of points without affecting the integral (PropositionÂ 9.18). Consider the subdivision \(a \lt c \lt d \lt b\text{.}\) Then
\begin{equation*} U(1_{(c,d)}; a,c,d,b) = (c-a)\cdot 0 + (d-c)\cdot 1 + (b-d)\cdot 0 = d-c . \end{equation*}
Similarly
\begin{equation*} L(1_{(c,d)}; a,c,d,b) = d-c. \end{equation*}
It therefore follows that \(1_{(c,d)}\) is Riemann integrable on \([a,b]\) with integral \(d-c\text{.}\)
