Remark 1.17.
The completeness principle is equivalent to the following statement involving the infimum:
Corollary 1.18.
Suppose that a set of real numbers \(S\subset \R\) is
  • non-empty
  • bounded below,
then \(S\) has an infimum, \(\inf S\in \R\text{.}\)
In other words, any set with a lower bound has a greatest lower bound (provided that the set is non-empty).
The second statement can be derived from the first by reflecting the set through zero to obtain \(-S:=\set{-x: x\in S}\text{,}\) then observing that \(\sup(-S) = -\inf S\text{.}\)
