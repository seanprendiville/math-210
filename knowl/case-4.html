Case 2 (\(0\lt a \lt \infty\)).
We wish to prove that
\begin{equation} \forall \eps \gt 0\ \exists N = N(\eps)\ \forall n \geq N \text{ we have } |a^{-1}-a_n^{-1}| \leq \eps.\tag{2.9} \end{equation}
We observe the key identity
\begin{equation*} a^{-1} - a_n^{-1} = \frac{a_n-a}{aa_n}. \end{equation*}
By definition of the limit (DefinitionÂ 1.11), we know that
\begin{equation*} \forall \eps \gt 0\ \exists N_1 = N_1(\eps)\ \forall n \geq N_1 \text{ we have } \abs{a - a_n} \leq \eps. \end{equation*}
By the triangle inequality
\begin{equation*} |a_n| = |a-(a-a_n)| \geq |a| - |a-a_n|. \end{equation*}
Hence if \(|a-a_n| \leq |a|/2\) then \(|a_n| \geq |a|/2 \gt 0\text{.}\) Setting \(N_0 := N_1(|a|/2)\text{,}\) it follows that for all \(n \geq N_0\) we have \(|a_n| \geq |a|/2 \gt 0\text{,}\) in particular we have
\begin{equation*} \abs{a^{-1} - a_n^{-1}} = \frac{|a_n-a|}{|aa_n|} \leq \frac{|a_n-a|}{|a|^2/2}\text{.} \end{equation*}
Hence if \(n \geq N_0\) and \(|a - a_n| \leq \eps\) then
\begin{equation*} \abs{a^{-1} - a_n^{-1}} = \frac{|a_n-a|}{|aa_n|} \leq \frac{\eps}{|a|^2/2} = \frac{2\eps}{|a|^2}. \end{equation*}
We therefore obtain (2.9) on setting
\begin{equation*} N(\eps) := \max\set{N_0, N_1\brac{\trecip{2}|a|^{2}\eps}}. \end{equation*}
