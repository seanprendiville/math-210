Example 6.25. Constant functions are differentiable.
Example 6.26. The identity function is differentiable.
Example 6.27. The reciprocal function is differentiable.
Since differentiable functions are continuous, any example of a non-continuous function is also an example of a non-differentiable function. For instance, the function Example 5.17 is discontinuous at \(x=0\text{,}\) so is not differentiable at \(x=0\text{.}\) Similarly, Example 5.18 is discontinuous everywhere, so is also nowhere differentiable.
