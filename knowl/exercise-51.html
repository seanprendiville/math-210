- 6.10. The intermediate value theorem.

(a)
State and prove the intermediate value theorem.
Solution.
Theorem 6.11. Intermediate value theorem.
Let \(f : [a,b] \to \R\) be a continuous function on a closed interval \([a,b] \subset \R\text{.}\) For any real number \(y\) between 2  \(f(a)\) and \(f(b)\text{,}\) there exists \(x \in [a,b]\) such that \(f(x) = y\text{.}\)
Proof.
We prove the result assuming that \(f(a) \leq y \leq f(b)\text{;}\) if \(f(a) \geq y \geq f(b)\) then we reduce to the former case on considering \(-f\text{.}\) Set
\begin{equation*} x := \sup\set{t \in [a,b] : f(t) \leq y}. \end{equation*}
The above supremum exists because the set is non-empty (it contains \(a\)) and is bounded above by \(b\text{.}\) In particular, \(a \leq x \leq b\text{.}\) We claim that \(f(x) = y\text{.}\)
If it is not the case that \(f(x)\leq y\text{,}\) then \(f(x)\gt y\text{.}\) It follows that the function \(g(t):=f(t)-y\) is continuous on \([a,b]\) and has \(g(x) \gt 0\text{.}\) Since positivity spreads around a positive limit, there exists \(\delta \gt 0\) such that for all \(t\in [a,b]\cap[x-\delta, x+\delta]\) we have \(f(t)-y = g(t) \gt 0\text{.}\) Since decreasing a supremum gives a non-upper bound, there exists \(t\gt x-\delta\) with \(f(t) \leq y\text{.}\) But then \(t \in [a,b]\cap[x-\delta, x+\delta]\text{,}\) and so we obtain the contradiction that
\begin{equation*} y \geq f(t) \gt y\text{.} \end{equation*}
It remains to show that \(f(x) \geq y\text{.}\) By way of contradiction, suppose that \(f(x) \lt y\text{.}\) It follows that the function \(h(t):=y-f(t)\) is continuous on \([a,b]\) and has \(h(x) \gt 0\text{.}\) Since positivity spreads around a positive limit, there exists \(\delta \gt 0\) such that for all \(t\in [a,b]\cap[x-\delta, x+\delta]\) we have \(y- f(t) = h(t) \gt 0\text{.}\) Notice that if \(b\leq x+\delta\text{,}\) then \(b \in [a,b]\cap[x-\delta, x+\delta]\) and hence \(f(b)\lt y\text{,}\) which is a contradiction. Thus \(b\gt x-\delta\text{.}\) In particular, \(x+\delta \in [a,b]\cap[x-\delta, x+\delta]\) and so \(f(x+\delta) \lt y\text{.}\) However the latter contradicts the fact that \(x\) is an upper bound for the set of \(t\in [a,b]\) satisfying \(f(t)\leq y\text{.}\) Thus \(f(x)\geq y\text{,}\) as required.
(b)
Use the intermediate value theorem to state and prove the existence of positive \(n\)th roots.
Solution.
Corollary 6.12. Existence of roots.
Let \(n \in \N\text{.}\) Let \(a \in [0, \infty)\text{.}\) Then there exists a unique \(x \in [0, \infty)\) such that
\begin{equation*} x^n = a. \end{equation*}
Proof.
The function \(x \mapsto x^n\) is continuous on the closed interval \([0, 1+a]\text{,}\) by Example 6.4. One can check that
\begin{equation*} 0^n \leq a \leq (1+a)^n. \end{equation*}
Hence by the intermediate value theorem (Theorem 6.11), there exists \(x \in [0, 1+a]\) such that \(x^n = a\text{.}\) For uniqueness, suppose that \(\tilde x \in [0,\infty)\) also satisfies \(\tilde x^n = a\text{.}\) Factorising, we have
\begin{equation*} (x-\tilde x)(x^{n-1} + x^{n-2}\tilde x + \dots + x \tilde x^{n-2} + \tilde x^{n-1}) = x^n - \tilde x^n = 0. \end{equation*}
If \(x \neq \tilde x\) then the factor \(x^{n-1} + \dots + \tilde x^{n-1}\) must be zero. Since \(x, \tilde x \geq 0\text{,}\) the only way this can happen is if \(x = \tilde x\text{.}\)
Definition 6.13. \(n\)th root function.
Let \(n \in \N\text{.}\) We define the function \(\sqrt[n]{} : [0, \infty) \to [0, \infty)\) by setting \(\sqrt[n]{a}\) to be the unique non-negative real number \(x \in [0, \infty)\) such that \(x^n= a\text{.}\) The existence and uniqueness of \(x\) are guaranteed by Corollary 6.12.
