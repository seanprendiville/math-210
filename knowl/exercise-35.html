- 4.14. The comparison test.

State and prove the "comparison test" for convergence of infinite series.
Solution.
Proposition 4.15. Comparison test.
Let \((a_n)_{n=1}^\infty\) and \((b_n)_{n=1}^\infty\) be sequences of real numbers. Suppose that there exists a constant \(C\) such that for all \(n\) we have \(|a_n| \leq Cb_n\text{.}\) If
\begin{equation*} \sum_{n=1}^\infty b_n \lt \infty. \end{equation*}
Then \(\displaystyle \sum_{n=1}^\infty a_n\) converges absolutely.
Proof.
Notice that we must have \(b_n \geq 0\) for all \(n\text{.}\) Hence for any \(N\) we have
\begin{equation*} \sum_{n=1}^N |a_n| \leq \sum_{n=1}^N Cb_n \leq C\sum_{n=1}^\infty b_n. \end{equation*}
Therefore the sequence of partial sums \(\displaystyle \sum_{n=1}^N |a_n|\) is an increasing sequence bounded above by the real number \(\displaystyle C \sum_{n=1}^\infty b_n\text{.}\) Hence, by the monotone convergence principle (Proposition 2.3), the sequence of partial sums \(\displaystyle \sum_{n=1}^N |a_n|\) converges to a real number. This proves absolute convergence (and hence convergence by Proposition 4.11).
Example 4.16. An example of applying the comparison test.
We claim that
\begin{equation*} \sum_{n=1}^\infty \frac{100n}{3^n} \lt \infty. \end{equation*}
To establish this we employ the comparison test (Proposition 4.15) with \(a_n := \frac{100n}{3^n}\text{,}\) \(b_n := \brac{\frac{2}{3}}^n\) and \(C:= \frac{50}{3}\text{,}\) so that it suffices to establish the inequality
\begin{equation} \frac{100n}{3^n} \leq 50 \times \brac{\frac{2}{3}}^n \qquad (n \in \N).\tag{4.2} \end{equation}
To this end, we first observe that for \(n\geq 1\) we have
\begin{equation*} a_{n+1} = \frac{100(n+1)}{3^{n+1}} \leq \frac{100(n+n)}{3\times3^{n}} = \frac{2}{3}\times \frac{100n}{3^{n}} = \frac{2}{3} a_n. \end{equation*}
It follows that for all \(n\) we have
\begin{equation*} a_n \leq \frac{2}{3} a_{n-1} \leq \brac{\frac{2}{3}}^2 a_{n-2} \leq \dots \leq \brac{\frac{2}{3}}^{n-1} a_{1} = 50 \brac{\frac{2}{3}}^{n}. \end{equation*}
The claimed inequality (4.2) follows.
