Definition 3.2. Tending to \(\pm\infty\).
Given a sequence of real numbers \((a_n)_{n=1}^\infty\) we write
\begin{equation*} a_n \to \infty\text{ as } n \to \infty\text{,} \end{equation*}
or
\begin{equation*} \lim_{n\to \infty} a_n = \infty\text{,} \end{equation*}
if
\begin{equation} (\forall M >0)( \exists N = N(M))( \forall n \geq N)[a_n \geq M].\tag{3.1} \end{equation}
We write
\begin{equation*} a_n \to -\infty\text{ as } n \to \infty\text{,} \end{equation*}
if
\begin{equation*} (\forall M >0)( \exists N =N(M))( \forall n \geq N)[a_n \leq -M]. \end{equation*}
