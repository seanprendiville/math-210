Notation
  • "\(P \iff Q\)": "the proposition \(P\) holds if and only if the proposition \(Q\) holds".
  • "\(\neg P\)": "the proposition \(P\) does not hold".
  • "\(P \implies Q\)": "it is not the case that the proposition \(P\) holds and the proposition \(Q\) does not hold", equivalently \(\neg(P\& \neg Q)\text{.}\)
  • "\(x \in A\)": "\(x\) is an element of the set \(A\)".
  • "\(\forall x \in A\)": "for all \(x \in A\)".
  • "\(\exists x \in A\)": "there exists \(x \in A\)".
  • \(\N := \set{1,2, 3, \dots}\text{.}\)
  • \(\Z := \set{\dots, -3,-2,-1,0,1,2, 3, \dots}\text{.}\)
  • Non-negative integers \(:= \set{0,1,2,3, \dots}\text{.}\)
  • \(\Q := \set{\frac{a}{b} : a \in \Z,\ q\in \N}\text{.}\)
  • \(\R := \text{the set of real numbers}\text{.}\)
  • Non-negative reals \(:= \set{x \in \R : x \geq 0}\text{.}\)
  • Positive reals \(:= \set{x \in \R : x \gt 0}\text{.}\)
  • Negative reals \(:= \set{x \in \R : x \lt 0}\text{.}\)
  • \(A\cup B := \set{x : x \in A \text{ or } x\in B}\text{.}\)
  • \(A\cap B := \set{x : x \in A \text{ and } x\in B}\text{.}\)
  • \(A\setminus B := \set{x : x \in A \text{ and } x\notin B}\text{.}\)
  • \(\displaystyle\bigcup_{i=1}^n A_i := \set{x : x \in A_i \text{ for some } i =1, \dots, n}\text{.}\)
  • \(\displaystyle\bigcap_{i=1}^n A_i := \set{x : x \in A_i \text{ for all } i =1, \dots, n}\text{.}\)
  • \(\displaystyle(a,b) := \set{x \in \R : a \lt x \lt b}\text{.}\)
  • \(\displaystyle(a,b] := \set{x \in \R : a \lt x \leq b}\text{.}\)
  • \(\displaystyle[a,b) := \set{x \in \R : a \leq x \lt b}\text{.}\)
  • \(\displaystyle[a,b] := \set{x \in \R : a \leq x \leq b}\text{.}\)
