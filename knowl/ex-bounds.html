- 1.1. Upper and lower bounds.

(a)
Define what it means to be an upper/lower bound for a set/function.
Solution.
Definition 1.2. Upper/lower bound.
We say that \(M \in \R\) is an upper bound for the set \(S\subset \R\) if
\begin{equation} (\forall x \in S)[x \leq M].\tag{1.1} \end{equation}
\(M\) is a lower bound for \(S\) if
\begin{equation*} (\forall x \in S)[x \geq M]. \end{equation*}
\(M\) is an upper/lower bound for the function \(f:D\to \R\) if it is an upper/lower bound for the image set
\begin{equation*} f(D) := \set{f(x) : x \in D}. \end{equation*}
(The notation "\(:=\)" should be read as "defined to equal".)
Example 1.3.
  • If \(S =\set{1,2}\) then 3 is an upper bound for \(S\text{.}\)
    Proof.
    We must check that for every \(x\in S\) we have \(x \leq 3\text{.}\) The only elements of \(S\) are 1 and 2, and both of the inequalities \(1\leq 3\) and \(2\leq 3\) hold. We have therefore verified that the defining condition (1.1) of being an upper bound holds.
  • If \(S= \set{0,1}\) then 1 is an upper bound for \(S\text{.}\)
    Proof.
    We have \(0\leq1\) and \(1\leq 1\text{.}\) So for all \(x \in S\) we have \(x \leq 1\text{.}\)
  • If \(S= (0, \infty)\) then 0 is a lower bound for \(S\text{.}\)
    Proof.
    If \(x \in S\) then by definition \(x \geq 0\text{.}\)
  • If \(S= (-\infty,0)\) then \(-1\) is not an upper bound for \(S\text{.}\)
    Proof.
    Negating the definition (1.1) of being an upper bound, we must show that for \(M = -1\) we have
    \begin{equation} (\exists x \in S)[x \gt M].\tag{1.2} \end{equation}
    This holds since, for instance, \(x:= -1/2 \in S \) satisfies \(x > -1\text{.}\)
(b)
Define what it means for a set/function to be bounded above, bounded below and bounded.
Solution.
Definition 1.4. Bounded above/below.
We say that a set \(S\subset\R\) is bounded above if it has an upper bound, that is
\begin{equation} (\exists M \in \R)(\forall x \in S)[x \leq M].\tag{1.3} \end{equation}
Similarly, a function \(f : D \to \R\) is bounded above if it has an upper bound, that is
\begin{equation*} (\exists M \in \R)(\forall x \in D)[f(x) \leq M]. \end{equation*}
A set/function is bounded below if it has a lower bound.
A set/function is bounded if it is both bounded above and bounded below. For sets, this is equivalent to:
\begin{equation} (\exists M \in \R)(\forall x \in S)[|x| \leq M].\tag{1.4} \end{equation}
For functions, this is equivalent to:
\begin{equation*} (\exists M \in \R)(\forall x \in D)[|f(x)| \leq M]. \end{equation*}
