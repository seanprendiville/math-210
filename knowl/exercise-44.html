- 5.20. Continuity.

Define what it means for a real-valued function on a real domain to be continuous at a point and what it means for the function to be continuous.
Solution.
Definition 5.21. Continuous.
Let \(f : D \to \R\) be a real valued function on a real domain \(D \subset \R\text{.}\) Let \(x_0\in D\text{.}\) We say that \(f : D \to \R\) is continuous at \(x_0\) if \(f(x) \to f(x_0)\) as \(x \to x_0\) through \(D\text{.}\) We say that \(f : D \to \R\) is continuous on the set \(D_0\subset D\) if it is continuous at all \(x_0 \in D_0\) (when \(D_0 = D\) we simply say "\(f\) is continuous").
Example 5.22. Constant functions are continuous.
Let \(f : D \to \R\) with \(D \subset \R\text{.}\) Suppose that \(f\) is constant. In other words, there exists \(c \in \R\) such that for all \(x \in D\) we have \(f(x) = c\text{.}\) Then for any sequence \((a_n)_n\) in \(D\) we have \(f(a_n) \to c\text{,}\) by Example 1.25. Hence \(f\) is continuous on \(D\text{.}\)
Example 5.23. The identity function is continuous.
Let \(D \subset \R\text{.}\) Define \(f : D \to \R\) by \(f(x) := x\) for all \(x \in D\text{.}\) Then for any sequence \((a_n)_n\) in \(D\) with \(a_n \to x_0\) we have \(f(a_n) = a_n \to x_0 = f(x_0)\text{.}\) Hence \(f\) is continuous on \(D\text{.}\)
Example 5.24. A function discontinuous at a point.
Consider the function \(f : \R \to \R\) given by
\begin{equation*} f(x) := \begin{cases} 0 \amp \text{ if } x \leq 0;\\ 1\amp \text{ if } x \gt0. \end{cases} \end{equation*}
We claim that this function is not continuous at \(x_0 = 0\text{.}\) In order to have \(f(x) \to f(0)\) as \(x \to 0\) we need \(f(a_n) \to 0\) for all sequences \((a_n)_n\) for which \(a_n \to 0\text{.}\) Yet \(1/n \to 0\) and \(f(1/n) \to 1\text{.}\)
Example 5.25. The absolute value function is continuous.
By the triangle inequality, for any \(x, x_0 \in \R\) we have
\begin{equation*} |x-x_0| \leq \eps \implies ||x|-|x_0|| \leq \eps. \end{equation*}
Hence (5.4) holds for the function \(x \mapsto |x|\) on taking \(\delta(x_0, \eps) := \eps\text{.}\)
Example 5.26. A function discontinuous everywhere.
Consider the function \(f : \R \to \R\) given by
\begin{equation*} f(x) := \begin{cases} 1 \amp \text{ if } x \in \Q;\\ 0\amp \text{ if } x \in \R \setminus\Q. \end{cases} \end{equation*}
We claim that this function is not continuous at any \(x_0 \in \R\text{.}\) Notice, however, that the restrictions 3  \(f|_{\Q}\) and \(f|_{\R\setminus\Q}\) are both continuous, since they are constant functions. The key observation in establishing our claim is to note that every real number \(x_0\) has a sequence of rationals \(q_n \in \Q\) such that \(q_n \to x_0\text{.}\) For instance, one may take
\begin{equation*} q_n := \frac{\floor{nx_0}}{n} \in \left(x_0 -\frac{1}{n}, x_0\right]\text{,} \end{equation*}
and employ the two-sided sandwich rule (Proposition 2.10). Let us use this to show that our function is not continuous at \(x_0\text{.}\) First suppose that \(x_0\in \R\setminus\Q\text{.}\) It follows that \(f(q_n) = 1 \not\to 0 = f(x_0)\text{.}\) Next suppose that \(x_0 \in \Q\text{.}\) This time, let \(q_n \in \Q\) be a sequence of rationals such that \(q_n \to x_0+ \sqrt{2}\text{.}\) Notice that for any \(n \in \N\) we have \(q_n-\sqrt{2} \notin \Q\text{,}\) otherwise \(\sqrt{2} \in \Q\) (a contradiction). Also \(q_n -\sqrt{2} \to x_0\text{.}\) Yet \(f(q_n-\sqrt{2}) = 0 \not\to 1 = f(x_0)\text{.}\)
