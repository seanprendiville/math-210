Given \(M \gt 0\text{,}\) we would like to find \(N=N(M)\) such that if \(n \geq N(M)\) then \(a_n+b_n \geq M\text{.}\)

We know both of the following hold:

\begin{equation*} (\forall M \gt 0)(\exists N_1 = N_1(M))(\forall n\geq N_1)[a_n\geq M]\text{,} \end{equation*}
\begin{equation*} (\exists M_1 \in \R)(\forall n \in \N)[b_n \geq M_1]. \end{equation*}

Thus for all \(n\in \N\) we have \(a_n + b_n \geq a_n + M_1\text{,}\) so that if \(n \geq N_1(M-M_1)\) we have \(a_n+b_n \geq (M-M_1) + M_1 = M.\) Thus the result follows on taking \(N(M):=N_1(M-M_1)\text{.}\)

(Notice that this implicitly assumes that \(M\gt M_1\text{.}\) The proof can be easily modified if this is not the case. Try it!)
