Solution 1.20.d.1.
Some examples of sequences include:
  • The enumeration of the even numbers in increasing order
    \begin{equation*} (2n)_{n=1}^\infty = (2, 4, 6, 8, \dots)\text{.} \end{equation*}
  • An enumeration of the even numbers in a different order
    \begin{equation*} (4,2,8,6,12,10, \dots)\text{.} \end{equation*}
    Notice that this is a different sequence to our first example, since they have different 1st term. The order of a sequence matters.
  • A constant sequence, consisting of the same real number repeated at every term, e.g.
    \begin{equation*} (\pi, \pi, \pi, \dots)\text{,} \end{equation*}
    so that \(a_n = \pi\) for all \(n \in \N\text{.}\)
  • The sequence of values of the \(\sin\) function evaluated at positive integers
    \begin{equation*} (\sin(n))_{n=1}^\infty = (\sin(1), \sin(2), \sin(3), \dots)\text{.} \end{equation*}
Some non-examples of sequences include:
  • The sine function \(\sin:\R\to\R\text{.}\) The problem is that the domain of this function is not a countable set.
  • The set of even numbers
    \begin{equation*} \set{2, 4, 6, \dots} = \set{4,2,8,6,12,10,\dots}\text{.} \end{equation*}
    The problem here is that there is no ordering associated with a set. It is not clear what the \(n\)th term of a set is.
  • The set
    \begin{equation*} \set{\pi, \pi, \pi, \dots} = \set{\pi}\text{.} \end{equation*}
    For a sequence, different terms may take the same value. It makes sense to say that the value \(\pi\) occurs infinitely often in a sequence. However, there are no repetitions of elements in a set: an element either belongs once or not at all.
