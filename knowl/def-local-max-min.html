Definition 7.7. Local max/min.
Let \(f : (a,b) \to \R\text{.}\) We say that \(x_0 \in (a,b)\) is a local maximum for \(f\) if there exists \(\delta \gt 0\) such that
\begin{equation*} (\forall x \in (x_0-\delta, x_0+\delta)) [f(x) \leq f(x_0)]. \end{equation*}
We say that \(x_0 \in (a,b)\) is a local minimum for \(f\) if there exists \(\delta \gt 0\) such that
\begin{equation*} (\forall x \in (x_0-\delta, x_0+\delta)) [f(x) \geq f(x_0)]. \end{equation*}
