Example 1.25. Constant sequences are convergent.
Fix a real number \(a \in \R\text{,}\) say \(a = 100\text{.}\) For each \(n \in \N\) set \(a_n := a\text{.}\) Then we claim that \(a_n \to a\text{.}\) To prove this, we must verify that (1.5) holds. To this end, fix \(\eps \gt 0\text{.}\) We must find \(N = N(\eps)\) such that
\begin{equation} n \geq N(\eps)\implies |a_n-a| \leq \eps.\tag{1.7} \end{equation}
For this particular sequence, we have \(|a_n-a| = |a-a| = 0\) for all \(n\text{.}\) Hence (1.7) holds on taking \(N(\eps) :=1\text{.}\)
Example 1.26. The reciprocals of the naturals converge to 0.
\(1/n \to 0\) as \(n\to \infty\)
Proof.
We must verify (1.5). Fixing \(\eps \gt 0\text{,}\) this constitutes showing that, from some point onwards, all terms lie in the \(\eps\)-neighbourhood of zero \([-\eps,\eps]\text{.}\) In other words, we want to find \(N = N(\eps)\) such that
\begin{equation*} n \geq N(\eps)\implies 1/n \in [-\eps, \eps]. \end{equation*}
Setting \(N(\eps):= 1/\eps\) we see that if \(n\geq N(\eps) = 1/\eps\) then re-arranging the inequality gives \(1/n \leq \eps\text{,}\) and so \(|\recip{n}-0|\leq \eps\) (as required).
Example 1.27. A non-convergent sequence.
We claim that the sequence \((a_n)_n\) defined by \(a_n:=n\) for all \(n\) does not converge to any real number \(a \in \R\text{.}\) If \(a\in \R\) were the limit of this sequence, then taking \(\eps:=1/3\) in (1.5) gives the existence of \(N\) such that for all \(n\geq N\) we have \(|a-n| \leq 1/3\text{.}\) In particular, by the triangle inequality we have
\begin{equation*} 1 = |(N+1)-N| \leq |(N+1)-a|+|a-N| \leq 2/3. \end{equation*}
This is a contradiction.
Example 1.28. Another non-convergent sequence.
We claim that the sequence
\begin{equation*} ((-1)^n)_{n=1}^\infty = (-1,1,-1,1,\dots) \end{equation*}
does not converge to any real number \(a \in \R\text{.}\) If \(a\in \R\) were the limit of this sequence, then taking \(\eps:=1/2\) in (1.5) gives the existence of \(N\) such that for all \(n\geq N\) we have \(|a-(-1)^n| \leq 1/2\text{.}\) In particular, by the triangle inequality we have
\begin{equation*} 2 = |(-1)^{N+1}-(-1)^N| \leq |(-1)^{N+1}-a|+|a-(-1)^N| \leq 1/2 + 1/2 = 1. \end{equation*}
This is a contradiction.
