- 7.21. Exam-style self-test question for workshop.

Let \(f :\R \to \R\text{.}\)
(a)
Define what it means for the function \(f\) to be differentiable at \(x_0\in \R\text{.}\)
Solution.
This is a version of Definition 7.2.
(b)
Suppose that \(f\) is differentiable at \(x_0\) with \(f'(x_0) = 0\text{.}\) Justify why, for any \(\eps \gt 0\) there exists \(\delta(\eps) \gt 0\) such that if \(0\lt |x-x_0| \leq \delta(\eps)\) then
\begin{equation*} |f(x) - f(x_0)| \leq \eps|x-x_0|. \end{equation*}
Solution.
This follows from combining Definition 7.2 with Proposition 5.17, then re-arranging and using the triangle inequality, as in the proof of Proposition 7.7.
(c)
Using the above, prove that if \(f\) is differentiable at \(x_0\) then it is continuous at \(x_0\text{.}\)
Solution.
This is the proof of Proposition 7.7.
