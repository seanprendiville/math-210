Example 7.8. A continuous function which is not differentiable.
The function \(x \mapsto |x|\) is an example of a function which is continuous at a point, but not differentiable there (namely at \(x_0 = 0\)). Hence continuity does not imply differentiability. To see that this function is not differentiable at \(x_0=0\text{,}\) consider the sequences \((1/n)_{n=1}^\infty\) and \((-1/n)_{n=1}^\infty\text{.}\) Both are sequences in the set \(\R\setminus\set{0}\) and both converge to 0, yet
\begin{equation*} \frac{|1/n|-|0|}{(1/n)-0} = 1 \to 1 \quad\text{as}\quad n \to \infty \end{equation*}
whilst
\begin{equation*} \frac{|-1/n|-|0|}{(-1/n)-0} = -1 \to 1 \quad\text{as}\quad n \to \infty\text{.} \end{equation*}
It follows that \(x \mapsto |x|\) does not satisfy the definition (DefinitionÂ 7.2) of being differentiable at \(x_0 = 0\text{,}\) since if \(y_0\) were the derivative then we get the contradiction that \(-1 = y_0= 1\text{.}\)
