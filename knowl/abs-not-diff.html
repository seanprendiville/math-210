Example 6.28. A continuous function which is not differentiable.
The function \(x \mapsto |x|\) is an example of a function which is continuous at a point, but not differentiable there (namely at \(x_0 = 0\)). Hence continuity does not imply differentiability. To see that this function is not differentiable at \(x_0=0\text{,}\) we must prove the negation of DefinitionÂ 6.22, namely
\begin{align} (\forall y \in \R)(\exists \eps \gt 0)(\forall \delta\gt 0)\amp(\exists x \in \R)\notag\\ \amp\sqbrac{0\lt |x-0| \leq \delta \text{ and } \abs{\frac{|x|-|0|}{x-0}-y} \gt \eps}\text{.}\tag{6.6} \end{align}
Simplifying, we have
\begin{equation*} \frac{|x|-|0|}{x-0} = \begin{cases} 1 \amp \text{ if } x \gt 0,\\ -1\amp \text{ if } x \lt 0.\end{cases} \end{equation*}
Hence, for fixed \(y\in \R\) we let \(\eps := 1/2\) and for any \(\delta > 0\) we let
\begin{equation*} x = x(y,\delta) = \begin{cases} -\delta \amp \text{ if } y \geq 0,\\ \delta \amp \text{ if } y \lt 0.\end{cases} \end{equation*}
Then \(|x-0| = \delta\) and
\begin{equation*} \abs{\frac{|x|-|0|}{x-0}-y} = 1+|y| \geq 1\gt \eps\text{.} \end{equation*}
This yields (6.6)
