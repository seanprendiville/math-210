- 10.9. Uniform convergence.

(a)
Define what it means for a sequence of functions to converge pointwise.
Solution.
Definition 10.10. Pointwise convergence.
For each \(n \in \N\) let \(f_n:S \to \R\) be a real-valued function with domain \(S\) (not necessarily a set of real numbers). Let \(f : S \to \R\text{.}\) We say that the sequence of functions \((f_n)_{n=1}^\infty\) converges pointwise to \(f\) on \(S\) if
\begin{equation*} (\forall x \in S)\sqbrac{\lim_{n\to \infty} f_n(x) = f(x)}. \end{equation*}
(b)
Define what it means for a sequence of functions to converge uniformly.
Solution.
Definition 10.11. Uniform convergence.
For each \(n \in \N\) let \(f_n:S \to \R\) be a real-valued function with domain \(S\) (not necessarily a set of real numbers). Let \(f : S \to \R\text{.}\) We say that the sequence of functions \((f_n)_{n=1}^\infty\) converges uniformly to \(f\) on \(S\) if
\begin{equation*} \lim_{n\to \infty} \sup_{S}| f-f_n| = 0. \end{equation*}
(c)
State and prove how uniform convergence implies pointwise convergence.
Solution.
Proposition 10.12. Uniform convergence implies pointwise convergence.
Suppose that the \(f_n\) converge uniformly to \(f\) on \(S\text{.}\) Then the \(f_n\) converge pointwise to \(f\) on \(S\text{.}\)
Proof.
Fix \(x \in S\text{.}\) Then
\begin{equation*} |f_n(x) - f(x)| \leq \sup_{S} |f_n-f| \to 0 \quad \text{as} \quad n \to \infty. \end{equation*}
Hence by the sandwich rule (PropositionÂ 3.6) we have \(f_n(x) \to f(x)\) as \(n \to \infty\text{.}\)
(d)
Give an example to show that pointwise convergence does not necessarily imply uniform convergence.
Solution.
Example 10.13. Pointwise convergence does not imply uniform convergence.
Define \(f_n : (0, 1) \to \R\) by
\begin{equation*} f_n(x) = 1_{[\frac{1}{n}, 1)}. \end{equation*}
Then the \(f_n\) converge pointwise to \(1_{(0,1)}\text{,}\) but not uniformly.
Proof.
Fix \(x \in (0,1)\text{.}\) Setting \(N = N(x) := \ceil{1/x}\text{,}\) if \(n \geq N\) then \(1/n \leq x\) so
\begin{equation*} f_n(x) = 1 = 1_{(0,1)}(x). \end{equation*}
Thus \(f_n(x) \to 1_{(0,1)}(x)\) as \(n \to \infty\text{.}\) For any \(n \in \N\text{,}\) setting \(x_n := 1/(2n)\) gives \(f_n(x_n) = 0\) and \(1_{(0,1)}(x_n) = 1\text{.}\) Therefore, for any \(n \in \N\) we have
\begin{equation*} \sup_{(0,1)} |f_n-1_{(0,1)}| = 1. \end{equation*}
This supremum does not tend to 0 as \(n \to \infty\text{.}\)
(e)
State and prove how continuity is preserved under uniform convergence.
Solution.
Proposition 10.14. Continuity is preserved under uniform convergence.
Let \(f, f_1, f_2, \dots:D \to \R\) be real-valued functions on a real domain \(D \subset \R\text{.}\) Suppose that the \(f_n\) converge uniformly to \(f\) on \(D\text{.}\) Suppose that each \(f_n\) is continuous at \(x_0 \in D\text{.}\) Then \(f\) is continuous at \(x_0\text{.}\)
Proof.
To show that \(f\) is continuous at \(x_0\) it suffices to establish that
\begin{equation*} (\forall \eps \gt 0)(\exists \delta= \delta(\eps) \gt 0)(\forall x \in D)[|x-x_0|\leq \delta \implies |f(x)-f(x_0)| \leq \eps]. \end{equation*}
Since \(f_n\) converges to \(f\) uniformly, there exists \(n\) for which
\begin{equation*} \sup_D |f_n-f| \leq \trecip{3} \eps. \end{equation*}
Since \(f_n\) is continuous at \(x_0\text{,}\) there exists \(\delta \gt 0\) such that if \(x \in D\) satisfies \(|x-x_0| \leq \delta\) then \(|f_n(x)-f_n(x_0)| \leq \trecip{3}\eps\text{.}\) Hence
\begin{align*} |f(x)-f(x_0)| \amp \leq |f(x)-f_n(x)| + |f_n(x)-f_n(x_0)| + |f_n(x_0) - f(x_0)|\\ \amp \leq \sup_D |f_n-f| + |f_n(x)-f_n(x_0)| + \sup_D |f_n-f|\\ \amp \leq \trecip{3} \eps +\trecip{3} \eps +\trecip{3} \eps = \eps. \end{align*}
(f)
State and prove how Riemann integrability is preserved under uniform convergence.
Solution.
Proposition 10.15. Integrability is preserved under uniform convergence.
Let \(f, f_1, f_2, \dots:[a,b] \to \R\text{.}\) Suppose that the \(f_n\) converge uniformly to \(f\) on \([a,b]\text{.}\) Suppose that each \(f_n\) is Riemann integrable on \([a,b]\text{.}\) Then \(f\) is Riemann integrable on \([a,b]\text{.}\) Furthermore
\begin{equation*} \int_a^b f = \lim_{n\to \infty} \int_a^b f_n. \end{equation*}
Proof.
Let \(\eps \gt 0\text{.}\) By uniform convergence, there exists \(N \in \N\) such that for all \(n \geq N\) and all \(x \in [a,b]\) we have
\begin{equation*} f_n(x)-\eps \leq f(x) \leq f_n(x) + \eps. \end{equation*}
In particular, since \(f_n\) is bounded on \([a,b]\text{,}\) so is \(f\text{.}\) It follows that
\begin{align*} \int_a^bf_n-\eps(b-a) = \int_a^b(f_n-\eps) \leq L(f;[a,b]) \leq U(f; \amp [a,b]) \leq \int_a^b(f_n+\eps)\\ \amp = \int_a^bf_n+\eps(b-a) . \end{align*}
Hence by two applications of the sandwich rule (PropositionÂ 2.10 and PropositionÂ 2.10) we deduce that
\begin{equation*} U(f;[a,b]) \leq \lim_{n \to \infty} \int_a^b f_n \leq L(f;[a,b]). \end{equation*}
The claimed result follows.
