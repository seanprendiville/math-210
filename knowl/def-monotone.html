Definition 2.2. Increasing/decreasing/monotone.

Let \(f : D \to\R\) be a real-valued function with domain \(D\subset \R\text{.}\) We say that \(f\) is increasing on \(D\) if

\begin{equation*} (\forall x, y \in D)[x \leq y \implies f(x) \leq f(y)] \end{equation*}

We say that \(f\) is strictly increasing on \(D\) if

\begin{equation*} (\forall x, y \in D)[x \lt y \implies f(x) \lt f(y)] \end{equation*}

We say that \(f\) is decreasing on \(D\) if

\begin{equation*} (\forall x, y \in D)[x \leq y \implies f(x) \geq f(y)] \end{equation*}

We say that \(f\) is strictly decreasing on \(D\) if

\begin{equation*} (\forall x, y \in D)[x \lt y \implies f(x) \gt f(y)] \end{equation*}

We say that \(f\) is monotone if it is either increasing or decreasing.

Recall that a sequence of reals \((a_n)_{n=1}^\infty\) is a function \(f : \N \to \R\) where \(f(n) := a_n\) for each \(n \in \N\text{.}\) One can check that such a sequence \((a_n)_{n=1}^\infty\) is increasing (as defined above) if and only if

\begin{equation*} (\forall n \in \N)[a_n \leq a_{n+1}]. \end{equation*}

A similar characterisation holds for decreasing sequences.
