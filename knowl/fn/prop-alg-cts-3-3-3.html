Footnote 6.1.
If \(x_0\in\R\) then \(N\) is a neighbourhood of \(x_0\) if \(N=[x_0-\eps, x_0+\eps]\) for some \(\eps \gt 0\text{.}\) If \(x_0 = +\infty\) then \(N\) is a neighbourhood if \(N=[M, \infty)\) for some \(M\in \R\text{.}\) If \(x_0 = -\infty\) then \(N\) is a neighbourhood if \(N=(-\infty, M]\) for some \(M\in \R\text{.}\)
