- 1.32. Exam-style self-test question for workshop.

(a)
Define what it means for 0 to be an infimum for \(S\subset \R\text{.}\)
Solution.
This is a special case of Task 1.5.a, with \(M = 0\text{.}\)
(b)
Justify why a set has a supremum if it contains at least one negative number and no positive numbers.
Solution.
This is a special case of Theorem 1.15.
(c)
Give an example of a set which contains at least one negative number, no positive numbers and which does not have a maximum.
Solution.
See - 1.10.
(d)
Prove that if \(\sup S = 100\) then \(S\) contains a number greater than 99.
Solution.
This is Task 1.5.c with \(\sup S = 100\) and \(\eps = 1\text{.}\)
