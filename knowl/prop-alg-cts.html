Proposition 6.2. Algebra of (finite) function limits.
Let \(f_1, f_2 : D \to \R\) with \(D \subset \R\text{.}\) Let \(x_0 \in [-\infty, \infty]\) and \(y_1,y_2\in \R\text{.}\) Suppose that for \(i=1,2\) we have
\begin{equation} f_i(x) \to y_i \quad \text{as}\quad x \to x_0\quad\text{through}\quad D\text{.}\tag{6.1} \end{equation}
Then
  1. Inequality rule.
    If \(f_1(x) \leq f_2(x)\) for all \(x \in D\text{,}\) then \(y_1\leq y_2\text{.}\)
  2. Addition rule.
    \begin{equation*} f_1(x)+f_2(x) \to y_1+y_2 \quad \text{as}\quad x \to x_0\quad\text{through}\quad D \end{equation*}
  3. Multiplication rule.
    \begin{equation*} f_1(x)f_2(x) \to y_1y_2 \quad \text{as}\quad x \to x_0\quad\text{through}\quad D \end{equation*}
  4. Division rule.
    If \(y_2\neq 0\) then there exists a neighbourhood 1  \(N\) of \(x_0\) such that
    \begin{equation*} \frac{f_1(x)}{f_2(x)} \to \frac{y_1}{y_2} \quad \text{as}\quad x \to x_0\quad\text{through}\quad D\cap N\text{.} \end{equation*}
  5. Composition rule.
    If \(g :f_1(D)\to \R\) satisfies
    \begin{equation*} g(y) \to z \quad \text{as}\quad y \to y_1 \quad\text{through}\quad f_1(D)\text{,} \end{equation*}
    then
    \begin{equation*} g(f_1(x)) \to z \quad \text{as}\quad x \to x_0\quad\text{through}\quad D\text{.} \end{equation*}
  6. Absolute value rule.
    If \(y_1\in \R\) then
    \begin{equation*} |f_1(x)| \to |y_1| \quad \text{as}\quad x \to x_0\quad\text{through}\quad D \end{equation*}
  7. Min/max rule.
    If \(y_1,y_2\in \R\) then
    \begin{equation*} \min\set{f_1(x),f_2(x)} \to \min\set{y_1,y_2} \quad \text{as}\quad x \to x_0\quad\text{through}\quad D\text{,} \end{equation*}
    \begin{equation*} \max\set{f_1(x),f_2(x)} \to \max\set{y_1,y_2} \quad \text{as}\quad x \to x_0\quad\text{through}\quad D\text{.} \end{equation*}
Proof.
We first prove the result for the division rule, the arguments for the inequality rule, addition rule and the multiplication rule being similar (and easier, hence left to the reader).
Since inequalities spread around a limit (Proposition 5.19), there exists a neighbourhood \(N\) of \(x_0\) such that for all \(x\in D\cap N\) we have \(f_2(x)\neq 0\text{.}\) Let \((a_n)_n\) be a sequence in \(D \cap N\) with \(a_n\to x_0\text{.}\) By definition of function limits (Definition 5.12), we are required to show that \(f_1(a_n)/f_2(a_n)\to y_1/y_2\) as \(n\to\infty\text{.}\) By our assumption (6.1) and the definition of function limit (Definition 5.12), we have \(f_i(a_n)\to y_i\) for both \(i=1,2\text{.}\) Hence combining the multiplication rule (Proposition 2.17) and reciprocal rule (Proposition 2.21) for sequential limits, we obtain
\begin{equation*} \lim_{n\to \infty}\frac{f_1(a_n)}{f_2(a_n)} = \lim_{n\to \infty}f_1(a_n)\lim_{n\to \infty}\frac{1}{f_2(a_n)}= y_1\frac{1}{y_2} \text{,} \end{equation*}
as required.
Next we prove the composition rule. Let \((a_n)_n\) be a sequence in \(D\) which tends to \(x_0\) through \(D\text{.}\) By definition of the function limit (Definition 5.12), we are required to prove that \(g( f_1(a_n))\to z\) as \(n\to\infty\text{.}\) By our assumption (6.1) and the definition of function limit (Definition 5.12), we have \(f_1(a_n)\to y_1\text{.}\) Since \(g\) tends to \(z\) as \(y\) tends to \(y_1\) through \(f_1(D)\text{,}\) it follows that \(g \brac{f_1(a_n)}\to z\) as \(n\to\infty\text{,}\) as required.
The absolute value rule follows from the composition rule, taking \(g\) to be the absolute value function \(g(y) := |y|\text{,}\) which is continuous (Example 5.25), hence \(g(y)\to |y_1|\) as \(y\to y_1\) through \(f_1(D)\text{.}\)
For the minimum rule, we first observe that the function \(g(x) := \min\set{x,0}\) is continuous on \(\R\text{.}\) To prove this one can establish the inequality \(|g(x_1)-g(x_2)|\leq |x_1-x_2|\) then (e.g.) take \(\delta(\eps) = \eps\) in the epsilon-delta characterisation of continuity. By the addition and multiplication rules, we have that
\begin{equation*} f_1(x) - f_2(x) \to y_1-y_2 \quad \text{as} \quad x\to x_0\quad\text{through}\quad D. \end{equation*}
Hence, using the composition rule with \(g\text{,}\) we see that
\begin{equation*} \min\set{f_1(x) - f_2(x),0} \to \min\set{y_1-y_2,0} \quad \text{as} \quad x\to x_0\quad\text{through}\quad D. \end{equation*}
Using the identity
\begin{equation*} \min\set{a,b} = \min{a-b,0}+b\text{,} \end{equation*}
together with the addition/multiplication rules, we deduce the minimum rule. The maximum rule follows from a similar argument.
