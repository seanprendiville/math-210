Definition 1.6. Supremum.

\(M \in \R\) is a supremum for \(S \subset \R\) if \(M\) is an upper bound for \(S\) and any any other upper bound \(\tilde M\) for \(S\) is larger, so that \(\tilde M \geq M\text{.}\)

In other words, \(M\) is the least upper bound for \(S\text{,}\) and we denote this quantity \(\sup S\) (if it exists).

Definition 1.7. Infimum.

\(M \in \R\) is an infimum for \(S \subset \R\) if \(M\) is a lower bound for \(S\) and any any other lower bound \(\tilde M\) for \(S\) is smaller, so that \(\tilde M \leq M\text{.}\)

In other words, \(M\) is the greatest lower bound for \(S\text{,}\) and we denote this quantity \(\inf S\) (if it exists).
