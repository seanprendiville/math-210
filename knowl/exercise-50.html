- 6.5. Uniform continuity.

(a)
Define what it means for a real-valued function on a real domain to be uniformly continuous.
Solution.
Definition 6.6. Uniformly continuous.
Let \(f : D \to \R\) be a real valued function on a real domain \(D \subset \R\text{.}\) We say that \(f\) is uniformly continuous on \(D\) if
\begin{align} (\forall \eps \gt 0)(\exists \delta = \delta(\eps) \gt 0)\amp (\forall x_1,x_2 \in D)\notag\\ \amp\bigsqbrac{\ |x_1-x_2| \leq \delta \implies |f(x_1)-f(x_2)| \leq \eps\ }.\tag{6.2} \end{align}
(b)
Justify why uniform continuity implies continuity, but not vice versa.
Solution.
Proposition 6.7. Uniform continuity implies continuity.
Let \(D \subset \R\) and \(f:D\to\R\text{.}\) If \(f\) is uniformly continuous on \(D\text{,}\) then it is also continuous on \(D\text{.}\)
Proof.
By the definition of continuity (Definition 5.21) and the epsilon-delta characterisation of the limit (Proposition 5.17), it suffices to prove that
\begin{align*} (\forall x_0 \in D)(\forall \eps \gt 0)(\exists \delta_1 = \delta_1(x_0,\eps) \gt 0)\amp (\forall x \in D)\\ \amp \bigsqbrac{\ |x-x_0| \leq \delta_1 \implies |f(x)-f(x_0)| \leq \eps\ }. \end{align*}
By the definition of uniform continuity Definition 6.6, we know that
\begin{align*} (\forall \eps \gt 0)(\exists \delta_2 = \delta_2(\eps) \gt 0)\amp (\forall x_1,x_2 \in D)\\ \amp\bigsqbrac{\ |x_1-x_2| \leq \delta_2 \implies |f(x_1)-f(x_2)| \leq \eps\ }. \end{align*}
Fixing \(x_0\in D\) and \(\eps \gt 0\text{,}\) set \(\delta_1(x_0,\eps) := \delta_2(\eps)\text{.}\) Then for \(x\in D\) with \(|x-x_0|\leq \delta_1 = \delta_2\text{,}\) we have \(|f(x)-f(x_0)|\leq \eps\text{,}\) as required.
Example 6.8. Continuity does not imply uniform continuity.
Define \(f : \R \to \R\) at each \(x \in \R\) by \(f(x) := x^2\text{.}\) Then \(f\) is a polynomial, so is continuous on \(\R\) by Example 6.4. By way of contradiction, suppose that \(f\) were uniformly continuous on \(\R\text{.}\) Taking \(\eps := 1\) in the definition of uniform continuity (Definition 6.6), there exists \(\delta \gt 0\) such that if \(|x_1 - x_2| \leq \delta\) then \(|x_1^2 - x_2^2| \leq 1\text{.}\) Setting \(x_2 := x_1 + \delta\text{,}\) we deduce that for any \(x_1 \gt 0\) we have
\begin{equation*} 1 \geq |x_1^2 - x_2^2| = |x_1-x_2||x_1+x_2| = \delta (2x_1+\delta) \geq \delta x_1. \end{equation*}
Taking, say, \(x_1 = 2/\delta\) then yields a contradiction.
The domain of a function is important for uniform continuity: the function \(f(x) = x^2\) is not uniformly continuous on \(\R\text{,}\) yet one can show that it is uniformly continuous on any closed interval \([a,b]\) (see Theorem 6.9).
(c)
State how a continuous function on a closed interval is uniformly continuous (the proof is non-examinable).
Solution.
Theorem 6.9. Continuous on closed interval implies uniformly continuous.
Let \(f : [a,b] \to \R\) be a real valued function on a closed interval \([a,b] \subset \R\text{.}\) Suppose that \(f\) is continuous on \([a,b]\text{.}\) Then \(f\) is uniformly continuous on \([a,b]\text{.}\)
Proof (non-examinable).
By way of contradiction, suppose that \(f\) is not uniformly continuous on \([a,b]\text{.}\) Negating the definition of uniform continuity (6.2), we deduce that
\begin{equation*} (\exists \eps \gt 0)(\forall \delta \gt 0)(\exists x,y \in [a,b]) \bigsqbrac{\ |x-y| \leq \delta \quad \text{and} \quad |f(x)-f(y)| \gt \eps\ }. \end{equation*}
Taking \(\delta := 1/n\text{,}\) we have
\begin{equation*} (\forall n \in \N)(\exists x_n,y_n \in [a,b]) \bigsqbrac{\ |x_n-y_n| \leq \trecip{n} \quad \text{and} \quad |f(x_n)-f(y_n)| \gt \eps\ }. \end{equation*}
By the Bolzano-Weierstrass theorem (Corollary 3.25) there exists \(x_0\in \R\) and \(k(1) \lt k(2) \lt \dots\) such that \(x_{k(n)} \to x_0\) as \(n \to \infty\text{.}\) By the inequality rule for limits (Proposition 2.5) we see that \(x_0 \in [a,b]\text{.}\) Applying the same argument to the sequence \((y_{k(n)})_n\text{,}\) there exists \(y_0\in [a,b]\) and \(\ell_1 \lt \ell_2 \lt \dots\) such that \(y_{k(\ell_n)} \to y_0\) as \(n \to \infty\text{.}\) Since subsequences of convergent sequence tend to the same limit (Proposition 3.20), we also have \(x_{k(\ell_n)} \to x_0\) as \(n \to \infty\text{.}\) Notice that
\begin{align*} |x_0 - y_0| \amp \leq |x_0-x_{k(\ell_n)}| + |x_{k(\ell_n)}- y_{k(\ell_n)}| + |y_{k(\ell_n)} -y_0|\\ \amp \leq |x_0-x_{k(\ell_n)}| + \recip{k(\ell_n)} + |y_{k(\ell_n)} -y_0|\\ \amp \to 0 \quad \text{as}\quad n \to \infty. \end{align*}
Therefore \(x_0 = y_0\text{.}\) Yet continuity of \(f\text{,}\) continuity of the absolute value and the addition rule (Proposition 2.14) together give that
\begin{equation*} \eps \lt |f(x_{k(\ell_n)}) - f(y_{k(\ell_n)})| \to |f(x_0)-f(y_0)| = 0. \end{equation*}
Since this is a contradiction, it follows that \(f\) must indeed be uniformly continuous.
