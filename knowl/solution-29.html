Proposition 2.10. Sandwich rule.
Let \((a_n)_{n=1}^\infty\text{,}\) \((b_n)_{n=1}^\infty\) and \((c_n)_{n=1}^\infty\) be sequences of real numbers. Suppose that for all \(n \in \N\) we have
\begin{equation*} a_n \leq b_n \leq c_n\text{.} \end{equation*}
If \(a_n \to a\) and \(c_n \to a\text{,}\) then \(b_n \to a\text{.}\)
Proof.
Example 2.11. An example of using the sandwich rule.
Remark 2.12.
The conclusion of the sandwich rule still holds if the inequalities \(a_n\leq b_n\leq c_n\) only hold for all but finitely 2  many values of \(n\text{.}\) For instance, the second inequality in
\begin{equation*} 0 \leq 100/n^2 \leq 1/n \end{equation*}
doesn't hold for \(n \lt 100\text{.}\) Yet since these inequalities hold for all \(n\geq 100\text{,}\) we can still use the sandwich rule to conclude that \(100/n^2 \to 0\) as \(n\to\infty\text{.}\)
