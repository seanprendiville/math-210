Definition 5.19. Continuous.
Let \(f : D \to \R\) be a real valued function on a real domain \(D \subset \R\text{.}\) Let \(x_0\in D\text{.}\) We say that \(f : D \to \R\) is continuous at \(x_0\) if \(f(x) \to f(x_0)\) as \(x \to x_0\) through \(D\text{.}\) We say that \(f : D \to \R\) is continuous on the set \(D_0\subset D\) if it is continuous at all \(x_0 \in D_0\) (when \(D_0 = D\) we simply say "\(f\) is continuous").
