Definition 5.14. Continuous.
Let \(f : D \to \R\) be a real valued function on a real domain \(D \subset \R\text{.}\) Let \(x_0\in D\text{.}\) We say that \(f : D \to \R\) is continuous at \(x_0\) if \(f(x) \to f(x_0)\) as \(x \to x_0\text{.}\) We say that \(f : D \to \R\) is continuous if it is continuous at all \(x_0 \in D_0\text{.}\)
