In this week's workshop you will be asked to write-up (under exam conditions) two of the following:

  • The following is a variant of Task 8.22.a. By modifying the proof of Task 8.22.a, prove that if \(f:[0,1]\to\R\) is bounded then there exist subdivisions \(0=a_0 \lt a_1 \lt \dots \lt a_P = 1\) and \(0=b_0 \lt b_1 \lt \dots \lt b_Q = 1\) such that

    \begin{gather*} L(f;[0,1]) - 0.05 \leq L(f;a_0, \dots, a_P),\\ U(f;b_0, \dots, b_Q) \leq U(f;[0,1]) + 0.05. \end{gather*}
    Explain how this implies that, if \(f:[0,1]\to\R\) is Riemann integrable, then there exists a subdivision \(0=x_0 \lt \dots \lt x_N = 1\) such that
    \begin{equation*} |U(f;x_0, \dots, x_N) - L(f;x_0, \dots, x_N)| \leq 0.1. \end{equation*}

  • The following is a variant of Task 9.20.a. Let \(f : [0,1] \to \R\) be a decreasing function. By modifying the proof of Task 9.20.a, prove that \(f\) is Riemann integrable on \([0,1]\text{.}\)

  • The following is a variant of Task 9.20.c. Let \(f : [0,1] \to \R\) be a function satisfying the following

    \begin{equation*} (\forall x_1, x_2 \in [0,1])[|f(x_1)-f(x_2)| \leq 9|x_1-x_2|]\text{.} \end{equation*}
    By modifying the proof of Task 9.20.c, prove that \(f\) is Riemann integrable on \([0,1]\text{.}\)
