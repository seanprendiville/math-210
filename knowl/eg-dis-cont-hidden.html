Consider the function \(f : \R \to \R\) given by
\begin{equation*} f(x) := \begin{cases} 0 \amp \text{ if } x \leq 0;\\ 1\amp \text{ if } x \gt0. \end{cases} \end{equation*}
We claim that this function is not continuous at \(x_0 = 0\text{.}\) In order to have \(f(x) \to f(0)\) as \(x \to 0\) we need \(f(a_n) \to 0\) for all sequences \((a_n)_n\) for which \(a_n \to 0\text{.}\) Yet \(1/n \to 0\) and \(f(1/n) \to 1\text{.}\)
