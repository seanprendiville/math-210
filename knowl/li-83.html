Item.

Construct counterexamples to illustrate why certain assumptions are necessary, for instance:

  • Construct a continuous function on an open interval which is not differentiable.

  • Construct a continuous function on an open interval which is not uniformly continuous.

  • Construct a set \(S \subset \R\) which is non-empty, but for which

    \begin{equation*} (\exists n \in \N)(\forall a \in S)\sqbrac{ a + \tfrac{1}{n} \lt \sup S }. \end{equation*}
