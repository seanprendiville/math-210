    Subsection 6.1 Self-test questions for lecture 11

    - 6.1. Algebra of (finite) function limits.

    (a)
    State and prove how (finite) function limits respect addition, multiplication, division, composition, absolute value and min/max.
    Solution.
    Proposition 6.2. Algebra of (finite) function limits.
    Let \(f_1, f_2 : D \to \R\) with \(D \subset \R\text{.}\) Let \(x_0 \in [-\infty, \infty]\) and \(y_1,y_2\in \R\text{.}\) Suppose that for \(i=1,2\) we have
    \begin{equation} f_i(x) \to y_i \quad \text{as}\quad x \to x_0\quad\text{through}\quad D\text{.}\tag{6.1} \end{equation}
    Then
    1. Addition rule.
      \begin{equation*} f_1(x)+f_2(x) \to y_1+y_2 \quad \text{as}\quad x \to x_0\quad\text{through}\quad D \end{equation*}
    2. Multiplication rule.
      \begin{equation*} f_1(x)f_2(x) \to y_1y_2 \quad \text{as}\quad x \to x_0\quad\text{through}\quad D \end{equation*}
    3. Division rule.
      If \(y_2\neq 0\) then there exists a neighbourhood 1  \(N\) of \(x_0\) such that
      \begin{equation*} \frac{f_1(x)}{f_2(x)} \to \frac{y_1}{y_2} \quad \text{as}\quad x \to x_0\quad\text{through}\quad D\cap N\text{.} \end{equation*}
    4. Composition rule.
      If \(g :f_1(D)\to \R\) satisfies
      \begin{equation*} g(y) \to z \quad \text{as}\quad y \to y_1 \quad\text{through}\quad f_1(D)\text{,} \end{equation*}
      then
      \begin{equation*} g(f_1(x)) \to z \quad \text{as}\quad x \to x_0\quad\text{through}\quad D\text{.} \end{equation*}
    5. Absolute value rule.
      If \(y_1\in \R\) then
      \begin{equation*} |f_1(x)| \to |y_1| \quad \text{as}\quad x \to x_0\quad\text{through}\quad D \end{equation*}
    6. Min/max rule.
      If \(y_1,y_2\in \R\) then
      \begin{equation*} \min\set{f_1(x),f_2(x)} \to \min\set{y_1,y_2} \quad \text{as}\quad x \to x_0\quad\text{through}\quad D\text{,} \end{equation*}
      \begin{equation*} \max\set{f_1(x),f_2(x)} \to \max\set{y_1,y_2} \quad \text{as}\quad x \to x_0\quad\text{through}\quad D\text{.} \end{equation*}
    Proof.
    We first prove the result for the division rule, the arguments for the addition rule and the multiplication rule being similar (and easier, hence left to the reader).
    Since inequalities spread around a limit (Proposition 5.19), there exists a neighbourhood \(N\) of \(x_0\) such that for all \(x\in D\cap N\) we have \(f_2(x)\neq 0\text{.}\) Let \((a_n)_n\) be a sequence in \(D \cap N\) with \(a_n\to x_0\text{.}\) By definition of function limits (Definition 5.12), we are required to show that \(f_1(a_n)/f_2(a_n)\to y_1/y_2\) as \(n\to\infty\text{.}\) By our assumption (6.1) and the definition of function limit (Definition 5.12), we have \(f_i(a_n)\to y_i\) for both \(i=1,2\text{.}\) Hence combining the multiplication rule (Proposition 2.17) and reciprocal rule (Proposition 2.21) for sequential limits, we obtain
    \begin{equation*} \lim_{n\to \infty}\frac{f_1(a_n)}{f_2(a_n)} = \lim_{n\to \infty}f_1(a_n)\lim_{n\to \infty}\frac{1}{f_2(a_n)}= y_1\frac{1}{y_2} \text{,} \end{equation*}
    as required.
    Next we prove the composition rule. Let \((a_n)_n\) be a sequence in \(D\) which tends to \(x_0\) through \(D\text{.}\) By definition of the function limit (Definition 5.12), we are required to prove that \(g( f_1(a_n))\to z\) as \(n\to\infty\text{.}\) By our assumption (6.1) and the definition of function limit (Definition 5.12), we have \(f_1(a_n)\to y_1\text{.}\) Since \(g\) tends to \(z\) as \(y\) tends to \(y_1\) through \(f_1(D)\text{,}\) it follows that \(g \brac{f_1(a_n)}\to z\) as \(n\to\infty\text{,}\) as required.
    The absolute value rule follows from the composition rule, taking \(g\) to be the absolute value function \(g(y) := |y|\text{,}\) which is continuous (Example 5.25), hence \(g(y)\to |y_1|\) as \(y\to y_1\) through \(f_1(D)\text{.}\)
    For the minimum rule, we first observe that the function \(g(x) := \min\set{x,0}\) is continuous on \(\R\text{.}\) To prove this one can establish the inequality \(|g(x_1)-g(x_2)|\leq |x_1-x_2|\) then (e.g.) take \(\delta(\eps) = \eps\) in the epsilon-delta characterisation of continuity. By the addition and multiplication rules, we have that
    \begin{equation*} f_1(x) - f_2(x) \to y_1-y_2 \quad \text{as} \quad x\to x_0\quad\text{through}\quad D. \end{equation*}
    Hence, using the composition rule with \(g\text{,}\) we see that
    \begin{equation*} \min\set{f_1(x) - f_2(x),0} \to \min\set{y_1-y_2,0} \quad \text{as} \quad x\to x_0\quad\text{through}\quad D. \end{equation*}
    Using the identity
    \begin{equation*} \min\set{a,b} = \min{a-b,0}+b\text{,} \end{equation*}
    together with the addition/multiplication rules, we deduce the minimum rule. The maximum rule follows from a similar argument.
    Corollary 6.3.
    Let \(f,g : D\to \R\) and \(h:f(D) \to \R\) all be continuous. Then \(f+g, fg, h\circ f, |f|, \min\set{f,g}, \max\set{f,g}\) are all continuous.
    Example 6.4. Polynomials are continuous.
    Given \(c_0, c_1, \dots, c_n \in \R\) define
    \begin{equation*} f(x) = c_0 + c_1 x + \dots + c_n x^n\text{.} \end{equation*}
    Then the polynomial \(f\) is continuous on \(\R\text{.}\)
    Proof.
    The identity function \(x \mapsto x\) is continuous, since denoting this function by \(f\text{,}\) we have \(a_n \to x_0\) implies \(f(a_n) \to f(x_0)\text{.}\) By the multiplication rule (Proposition 2.17) it follows that \(x^2\) is continuous. Iterating this argument, we see that \(x^3 = x\cdot x^2\) is continuous. It follows from induction that \(x^n\) is continuous for any \(n \in \N\text{.}\) Since constant functions are continuous (Example 5.22), it again follows from the multiplication rule that \(c_n x^n\) is continuous for any \(c_n \in \R\text{.}\) By the addition rule (Proposition 2.14), the sum
    \begin{equation*} c_0 + c_1x+ \dots + c_n x^n \end{equation*}
    is continuous. Hence polynomials are continuous.

    - 6.5. Uniform continuity.

    (a)
    Define what it means for a real-valued function on a real domain to be uniformly continuous.
    Solution.
    Definition 6.6. Uniformly continuous.
    Let \(f : D \to \R\) be a real valued function on a real domain \(D \subset \R\text{.}\) We say that \(f\) is uniformly continuous on \(D\) if
    \begin{align} (\forall \eps \gt 0)(\exists \delta = \delta(\eps) \gt 0)\amp (\forall x_1,x_2 \in D)\notag\\ \amp\bigsqbrac{\ |x_1-x_2| \leq \delta \implies |f(x_1)-f(x_2)| \leq \eps\ }.\tag{6.2} \end{align}
    (b)
    Justify why uniform continuity implies continuity, but not vice versa.
    Solution.
    Proposition 6.7. Uniform continuity implies continuity.
    Let \(D \subset \R\) and \(f:D\to\R\text{.}\) If \(f\) is uniformly continuous on \(D\text{,}\) then it is also continuous on \(D\text{.}\)
    Proof.
    By the definition of continuity (Definition 5.21) and the epsilon-delta characterisation of the limit (Proposition 5.17), it suffices to prove that
    \begin{align*} (\forall x_0 \in D)(\forall \eps \gt 0)(\exists \delta_1 = \delta_1(x_0,\eps) \gt 0)\amp (\forall x \in D)\\ \amp \bigsqbrac{\ |x-x_0| \leq \delta_1 \implies |f(x)-f(x_0)| \leq \eps\ }. \end{align*}
    By the definition of uniform continuity Definition 6.6, we know that
    \begin{align*} (\forall \eps \gt 0)(\exists \delta_2 = \delta_2(\eps) \gt 0)\amp (\forall x_1,x_2 \in D)\\ \amp\bigsqbrac{\ |x_1-x_2| \leq \delta_2 \implies |f(x_1)-f(x_2)| \leq \eps\ }. \end{align*}
    Fixing \(x_0\in D\) and \(\eps \gt 0\text{,}\) set \(\delta_1(x_0,\eps) := \delta_2(\eps)\text{.}\) Then for \(x\in D\) with \(|x-x_0|\leq \delta_1 = \delta_2\text{,}\) we have \(|f(x)-f(x_0)|\leq \eps\text{,}\) as required.
    Example 6.8. Continuity does not imply uniform continuity.
    Define \(f : \R \to \R\) at each \(x \in \R\) by \(f(x) := x^2\text{.}\) Then \(f\) is a polynomial, so is continuous on \(\R\) by Example 6.4. By way of contradiction, suppose that \(f\) were uniformly continuous on \(\R\text{.}\) Taking \(\eps := 1\) in the definition of uniform continuity (Definition 6.6), there exists \(\delta \gt 0\) such that if \(|x_1 - x_2| \leq \delta\) then \(|x_1^2 - x_2^2| \leq 1\text{.}\) Setting \(x_2 := x_1 + \delta\text{,}\) we deduce that for any \(x_1 \gt 0\) we have
    \begin{equation*} 1 \geq |x_1^2 - x_2^2| = |x_1-x_2||x_1+x_2| = \delta (2x_1+\delta) \geq \delta x_1. \end{equation*}
    Taking, say, \(x_1 = 2/\delta\) then yields a contradiction.
    The domain of a function is important for uniform continuity: the function \(f(x) = x^2\) is not uniformly continuous on \(\R\text{,}\) yet one can show that it is uniformly continuous on any closed interval \([a,b]\) (see Theorem 6.9).
    (c)
    State how a continuous function on a closed interval is uniformly continuous (the proof is non-examinable).
    Solution.
    Theorem 6.9. Continuous on closed interval implies uniformly continuous.
    Let \(f : [a,b] \to \R\) be a real valued function on a closed interval \([a,b] \subset \R\text{.}\) Suppose that \(f\) is continuous on \([a,b]\text{.}\) Then \(f\) is uniformly continuous on \([a,b]\text{.}\)
    Proof (non-examinable).
    By way of contradiction, suppose that \(f\) is not uniformly continuous on \([a,b]\text{.}\) Negating the definition of uniform continuity (6.2), we deduce that
    \begin{equation*} (\exists \eps \gt 0)(\forall \delta \gt 0)(\exists x,y \in [a,b]) \bigsqbrac{\ |x-y| \leq \delta \quad \text{and} \quad |f(x)-f(y)| \gt \eps\ }. \end{equation*}
    Taking \(\delta := 1/n\text{,}\) we have
    \begin{equation*} (\forall n \in \N)(\exists x_n,y_n \in [a,b]) \bigsqbrac{\ |x_n-y_n| \leq \trecip{n} \quad \text{and} \quad |f(x_n)-f(y_n)| \gt \eps\ }. \end{equation*}
    By the Bolzano-Weierstrass theorem (Corollary 3.25) there exists \(x_0\in \R\) and \(k(1) \lt k(2) \lt \dots\) such that \(x_{k(n)} \to x_0\) as \(n \to \infty\text{.}\) By the inequality rule for limits (Proposition 2.5) we see that \(x_0 \in [a,b]\text{.}\) Applying the same argument to the sequence \((y_{k(n)})_n\text{,}\) there exists \(y_0\in [a,b]\) and \(\ell_1 \lt \ell_2 \lt \dots\) such that \(y_{k(\ell_n)} \to y_0\) as \(n \to \infty\text{.}\) Since subsequences of convergent sequence tend to the same limit (Proposition 3.20), we also have \(x_{k(\ell_n)} \to x_0\) as \(n \to \infty\text{.}\) Notice that
    \begin{align*} |x_0 - y_0| \amp \leq |x_0-x_{k(\ell_n)}| + |x_{k(\ell_n)}- y_{k(\ell_n)}| + |y_{k(\ell_n)} -y_0|\\ \amp \leq |x_0-x_{k(\ell_n)}| + \recip{k(\ell_n)} + |y_{k(\ell_n)} -y_0|\\ \amp \to 0 \quad \text{as}\quad n \to \infty. \end{align*}
    Therefore \(x_0 = y_0\text{.}\) Yet continuity of \(f\text{,}\) continuity of the absolute value and the addition rule (Proposition 2.14) together give that
    \begin{equation*} \eps \lt |f(x_{k(\ell_n)}) - f(y_{k(\ell_n)})| \to |f(x_0)-f(y_0)| = 0. \end{equation*}
    Since this is a contradiction, it follows that \(f\) must indeed be uniformly continuous.

    Subsection 6.2 Self-test questions for lecture 12

    - 6.10. The intermediate value theorem.

    (a)
    State and prove the intermediate value theorem.
    Solution.
    Theorem 6.11. Intermediate value theorem.
    Let \(f : [a,b] \to \R\) be a continuous function on a closed interval \([a,b] \subset \R\text{.}\) For any real number \(y\) between 2  \(f(a)\) and \(f(b)\text{,}\) there exists \(x \in [a,b]\) such that \(f(x) = y\text{.}\)
    Proof.
    We prove the result assuming that \(f(a) \leq y \leq f(b)\text{;}\) if \(f(a) \geq y \geq f(b)\) then we reduce to the former case on considering \(-f\text{.}\) Set
    \begin{equation*} x := \inf\set{t \in [a,b] : f(t) \geq y}. \end{equation*}
    The above infimum exists because the set is non-empty (it contains \(b\)) and is bounded below by \(a\text{.}\) In particular, \(a \leq x \leq b\text{.}\) We claim that \(f(x) = y\text{.}\)
    We first observe that for any \(n\in \N\) the number \(x+1/n\) is larger than the infimum, so cannot be a lower bound. Hence there exists \(t_n \in [a,b]\) with \(f(t_n) \geq y\) and \(t_n \lt x+1/n \text{.}\) Notice that if \(t_n\lt x\) then we contradict the fact that \(x\) is a lower bound for the set of \(t\) satisfying \(f(t)\geq y\text{.}\) Thus for all \(n\in \N\) we have \(x \leq t_n\leq x+\recip{n}\text{.}\) It follows from the sandwich rule that \(t_n \to x\) as \(n\to\infty\text{,}\) so by continuity and the inequality rule, we get that \(f(x)= \lim_{n\to \infty} f(t_n) \geq y\text{.}\)
    It remains to show that \(f(x) \leq y\text{.}\) By way of contradiction, suppose that \(f(x) \gt y\text{.}\) Since inequalities spread around a point of continuity (Proposition 5.19), there exists \(\delta \gt 0\) such that for all \(t\in [a,b]\cap[x-\delta, x+\delta]\) we have \(f(x) \gt y\text{.}\) Notice that if \(a\geq x-\delta\text{,}\) then \(a \in [a,b]\cap[x-\delta, x+\delta]\) and hence \(f(a)\gt y\text{,}\) which is a contradiction. Thus \(a\lt x-\delta\text{.}\) In particular, \(x-\delta \in [a,b]\cap[x-\delta, x+\delta]\) and so \(f(x-\delta) \gt y\text{.}\) However the latter contradicts the fact that \(x\) is a lower bound for the set of \(t\in [a,b]\) satisfying \(f(t)\geq y\text{.}\) Thus \(f(x)\leq y\text{,}\) as required.
    (b)
    Use the intermediate value theorem to state and prove the existence of positive \(n\)th roots.
    Solution.
    Corollary 6.12. Existence of roots.
    Let \(n \in \N\text{.}\) Let \(a \in [0, \infty)\text{.}\) Then there exists a unique \(x \in [0, \infty)\) such that
    \begin{equation*} x^n = a. \end{equation*}
    Proof.
    The function \(x \mapsto x^n\) is continuous on the closed interval \([0, 1+a]\text{,}\) by Example 6.4. One can check that
    \begin{equation*} 0^n \leq a \leq (1+a)^n. \end{equation*}
    Hence by the intermediate value theorem (Theorem 6.11), there exists \(x \in [0, 1+a]\) such that \(x^n = a\text{.}\) For uniqueness, suppose that \(\tilde x \in [0,\infty)\) also satisfies \(\tilde x^n = a\text{.}\) Factorising, we have
    \begin{equation*} (x-\tilde x)(x^{n-1} + x^{n-2}\tilde x + \dots + x \tilde x^{n-2} + \tilde x^{n-1}) = x^n - \tilde x^n = 0. \end{equation*}
    If \(x \neq \tilde x\) then the factor \(x^{n-1} + \dots + \tilde x^{n-1}\) must be zero. Since \(x, \tilde x \geq 0\text{,}\) the only way this can happen is if \(x = \tilde x\text{.}\)
    Definition 6.13. \(n\)th root function.
    Let \(n \in \N\text{.}\) We define the function \(\sqrt[n]{} : [0, \infty) \to [0, \infty)\) by setting \(\sqrt[n]{a}\) to be the unique non-negative real number \(x \in [0, \infty)\) such that \(x^n= a\text{.}\) The existence and uniqueness of \(x\) are guaranteed by Corollary 6.12.

    - 6.14. The extreme value theorem.

    (a)
    State the extreme value theorem (the proof is non-examinable).
    Solution.
    Theorem 6.15. Extreme value theorem.
    Let \(f : [a,b] \to \R\) be a continuous function. Then there exist \(x_1,x_2 \in [a,b]\) such that
    \begin{equation*} f(x_1) = \inf_{[a,b]} f \quad \text{and} \quad f(x_2) = \sup_{[a,b]} f. \end{equation*}
    In particular, a continuous function on a closed interval is bounded.
    Proof (non-examinable).
    We prove the result for the supremum, the argument for the infimum being similar. Notice that the supremum \(\sup_{[a,b]} f\) is either real or equal to \(\infty\text{.}\) We first discount the latter situation. If \(\sup_{[a,b]} f = \infty\) then for any \(n \in \N\) there exist \(x_n \in [a,b]\) with \(f(x_n) \gt n\) (otherwise \(n\) is a finite upper bound for \(f\)). By the Bolzano-Weierstrass theorem, there exists \(k_1 \lt k_2 \lt \dots\) such that the subsequence \(x_{k_n}\) tends to a real limit \(x_0\) for some \(x_0 \in \R\text{.}\) By the inequality rule (Proposition 2.5) we have \(x_0 \in [a,b]\text{.}\) By continuity (Definition 5.21) and the inequality rule (Proposition 2.5) we have
    \begin{equation*} f(x_0) = \lim_{n \to \infty} f(x_{k_n}) \geq \lim_{n \to \infty} k_n = \infty. \end{equation*}
    This is a contradiction, since \(f(x_0)\) is a real number. It follows that the supremum is finite. Since decreasing a supremum gives a non upper bound (Lemma 1.9), for each \(n \in \N\) there exists \(x_n \in [a,b]\) such that
    \begin{equation*} \brac{\sup_{[a,b]} f}-\recip n \lt f(x_n) \leq \sup_{[a,b]} f. \end{equation*}
    Since the set \([a,b]\) is bounded, the Bolzano-Weierstrass theorem (Corollary 3.25) gives the existence of \(k_1 \lt k_2 \lt k_3 \lt \dots\) such that the subsequence \(x_{k_n}\) converges to some \(x_0 \in [a,b]\text{.}\) Since \(f\) is continuous at \(x_0\text{,}\) the sequential definition of continuity (Definition 5.21) gives that
    \begin{equation*} f(x_0) = \lim_{n\to \infty} f(x_{k_n}). \end{equation*}
    By the sandwich rule (Proposition 2.10), we deduce that
    \begin{equation*} f(x_0) = \sup_{[a,b]} f. \end{equation*}
