    Subsection 8.1 Self-test questions for lecture 15

    - 8.1. Local min/max.

    (a)
    Define a local max/min.
    Solution.
    Definition 8.2. Local max/min.
    Let \(D \subset \R\) and \(f : D \to \R\text{.}\) We say that \(x_0 \in D\) is a local maximum for \(f\) if there exists \(\delta \gt 0\) such that
    \begin{equation*} (\forall x \in D\cap [x_0-\delta, x_0+\delta]) [f(x) \leq f(x_0)]. \end{equation*}
    We say that \(x_0 \in (a,b)\) is a local minimum for \(f\) if there exists \(\delta \gt 0\) such that
    \begin{equation*} (\forall x \in D\cap [x_0-\delta, x_0+\delta]) [f(x) \geq f(x_0)]. \end{equation*}
    (b)
    State and prove how a derivative must be zero at a local max/min.
    Solution.
    Proposition 8.3. The derivative is zero at a local max/min.
    Suppose that \(f : (a,b) \to \R\) is differentiable at \(x_0 \in (a,b)\text{.}\) Suppose that \(x_0\) is a local maximum or local minimum for \(f\text{.}\) Then \(f'(x_0) = 0\text{.}\)
    Proof.
    Let us suppose that \(x_0\) is a local maximum for \(f\text{;}\) the argument for the local minimum is similar. By definition of a local maximum (see Definition 8.2) there exists \(\delta \gt 0\) such that for all \(x \in (a,b)\cap [x_0-\delta, x_0+\delta]\) we have \(f(x_0) \geq f(x)\text{.}\) Decreasing \(\delta \gt 0\) if necessary 1 , we may assume that
    \begin{equation*} [x_0-\delta, x_0+\delta] \subset (a,b)\text{.} \end{equation*}
    Hence
    \begin{equation*} \frac{f(x)-f(x_0)}{x-x_0} \begin{cases} \geq 0 \amp \text{ if } x_0-\delta \leq x \lt x_0,\\ \leq 0 \amp \text{ if } x_0 \lt x \leq x_0+\delta.\end{cases} \end{equation*}
    Setting \(a_n := x_0-\trecip{n}\delta\text{,}\) the sequential definition of the limit Definition 5.12, the definition of the derivative (Definition 7.2) and the inequality rule (Proposition 2.5) give that
    \begin{equation*} f'(x_0) = \lim_{n\to \infty}\frac{f(a_n)-f(x_0)}{a_n-x_0} \geq 0. \end{equation*}
    Similarly, setting \(b_0:= x_0+\trecip{n}\delta\) gives
    \begin{equation*} f'(x_0) = \lim_{n\to \infty}\frac{f(b_n)-f(x_0)}{a_n-x_0} \leq 0. \end{equation*}
    Thus \(f'(x_0) = 0\text{.}\)

    - 8.4. Rolle’s theorem.

    State and prove Rolle’s theorem.
    Solution.
    Theorem 8.5. Rolle’s theorem.
    Let \(f : [a,b] \to \R\) be a continuous function which is differentiable on \((a,b)\text{,}\) with \(a \lt b\text{.}\) Suppose that \(f(a) = f(b)\text{.}\) Then there exists \(x_0 \in (a,b)\) such that \(f'(x_0) = 0\text{.}\)
    Proof.
    By the extreme value theorem (Theorem 6.15), there exists \(x_1,x_2 \in [a,b]\) which are minima and maxima (respectively) for \(f\) on \([a,b]\text{.}\) In particular \(x_1\) is a local minimum and \(x_2\) is a local maximum. If \(x_1 \in (a,b)\) or \(x_2 \in (a,b)\text{,}\) then it follows that \(f'(x_1) = 0\) or \(f'(x_2) = 0\) since local minima/maxima have zero derivative at points of differentiability (Proposition 8.3). We may therefore assume that \(\set{x_1, x_2} \subset \set{a,b}\text{.}\) Hence \(f(x_1) = f(x_2)\text{.}\) For any \(x \in [a,b]\text{,}\) we have that \(f(x)\) is between \(f(x_1)\) and \(f(x_2)\text{.}\) Since \(f(x_1) = f(x_2)\text{,}\) it follows that \(f\) is constant on \([a,b]\text{.}\) It then follows from Example 7.3 that \(f'(x)= 0\) at every \(x \in (a,b)\text{,}\) e.g. for \(x := \frac{a+b}{2}\text{.}\)

    - 8.6. The mean value theorem.

    State and prove the mean value theorem.
    Solution.
    Theorem 8.7. Mean value theorem.
    Let \(f : [a,b] \to \R\) be a continuous function which is differentiable on \((a,b)\text{,}\) with \(a \lt b\text{.}\) Then there exists \(x_0 \in (a,b)\) such that
    \begin{equation} f(b) - f(a) = f'(x_0)(b-a).\tag{8.1} \end{equation}
    Proof.
    We deduce this result from Rolle’s theorem (Theorem 8.5). The idea of the proof is to try and find a continuous function \(g:[a,b] \to \R\) which is differentiable on \((a,b)\) with \(g(a) = g(b)\) and such that the identity \(g'(x_0) = 0\) implies (8.1). To this end set
    \begin{equation*} g(x) :=f(x) - \frac{f(b)-f(a)}{b-a} (x-a), \qquad (x \in [a,b]). \end{equation*}
    This is continuous on \([a,b]\) since constant functions are continuous (Example 5.22), the identity function is continuous (Example 5.23) and so is the sum/product of continuous functions (Proposition 6.2). The function \(g\) is differentiable on \((a,b)\) since constant functions are differentiable (Example 7.3), the identity function is differentiable (Example 7.4) and so is the sum/product of differentiable functions (Proposition 6.2). We have \(g(a) = f(a)\) and \(g(b) = f(a)\text{.}\) We may therefore apply Rolle’s theorem to deduce the existence of \(x_0 \in (a,b)\) such that \(g'(x_0) = 0\text{.}\) Employing the sum/product rules for differentiation, we have that
    \begin{equation*} g'(x_0) = f'(x_0) -\frac{f(b)-f(a)}{b-a}. \end{equation*}
    The result follows on re-arranging the identity
    \begin{equation*} 0 = f'(x_0) -\frac{f(b)-f(a)}{b-a}. \end{equation*}

    - 8.8. Monotonicity and the sign of the derivative.

    Prove how a function with non-negative derivative is increasing.
    Solution.
    Corollary 8.9.
    Let \(f : [a,b] \to \R\) be a continuous function which is differentiable on \((a,b)\text{,}\) with \(a \lt b\text{.}\)
    1. If \((\forall x \in (a,b))[f'(x) \geq 0]\) then \(f\) is increasing.
    2. If \((\forall x \in (a,b))[f'(x) \gt 0]\) then \(f\) is strictly increasing.
    3. If \((\forall x \in (a,b))[f'(x) \leq 0]\) then \(f\) is decreasing.
    4. If \((\forall x \in (a,b))[f'(x) \lt 0]\) then \(f\) is strictly decreasing.
    5. If \((\forall x \in (a,b))[f'(x) = 0]\) then \(f\) is constant.
    Proof.
    Let us first suppose that \((\forall x \in (a,b))[f'(x) \geq 0]\text{.}\) Let \(x_1, x_2 \in [a,b]\) with \(x_1 \lt x_2\text{.}\) Our task is to prove that \(f(x_1) \leq f(x_2)\text{.}\) Applying the mean value theorem to the function \(f: [x_1, x_2] \to \R\text{,}\) there exists \(x_0 \in (x_1, x_2)\) such that
    \begin{equation*} f(x_2)-f(x_1) = f'(x_0)(x_2-x_1) \geq 0, \end{equation*}
    as required. Notice that if \((\forall x \in (a,b))[f'(x) \gt 0]\) then \(f(x_2) -f(x_1) = f'(x_0)(x_2-x_1) \gt 0\text{,}\) which implies that the function is strictly increasing. The conclusions involving decreasing functions follow on applying the increasing results to \(-f\text{.}\) If \((\forall x \in (a,b))[f'(x) = 0]\) then \(f\) is both increasing and decreasing, hence for all \(x \in [a,b]\) we have \(f(x) = f(b)\text{.}\) Thus in the latter case \(f\) is constant.

    Subsection 8.2 Self-test questions for lecture 16

    - 8.10. The exponential function.

    (a)
    Prove that the exponential function is infinitely differentiable and calculate its derivative.
    Solution.
    Proposition 8.11.
    The exponential function \(\exp : \R \to \R\text{,}\) given by
    \begin{equation*} \exp(x) := \sum_{n=0}^\infty \frac{x^n}{n!} \end{equation*}
    is infinitely differentiable with first derivative
    \begin{equation} \exp'(x) = \exp(x).\tag{8.2} \end{equation}
    Proof.
    The infinite radius of convergence of the power series is proved in Proposition 5.7. Infinite differentiability and the formula (8.2) then both follow from Theorem 7.18.
    (b)
    State and prove a formula for the reciprocal of the exponential function.
    Solution.
    Proposition 8.12.
    For all \(x \in \R\) we have the formula
    \begin{equation*} \exp(-x) = \recip{\exp(x)}. \end{equation*}
    Proof.
    The addition law \(\exp(x+y) = \exp(x)\exp(y)\) is established in Proposition 5.10. From the addition law, it follows that
    \begin{equation*} \exp(x)\exp(-x) = \exp(0) = 1. \end{equation*}
    Thus \(\exp(-x) = 1/\exp(x)\text{.}\)
    (c)
    Justify why the exponential function is positive everywhere.
    Solution.
    By the inequality rule for sequences (Proposition 2.5), when \(x \geq 0\) we have
    \begin{equation*} \exp(x) = \lim_{N\to \infty} \brac{1 + x + \trecip{2} x^2 + \dots + \trecip{N!} x^N} \geq 1. \end{equation*}
    Hence \(\exp(x) \geq 1\) when \(x \geq 0\text{.}\) If \(x \geq 0\) then \(\exp(-x) = \recip{\exp(x)} \gt0\text{.}\) Thus for all \(x \in \R\) we have \(\exp(x) \gt 0\text{.}\)
    (d)
    State and prove the limit of the exponential function as its argument tends to infinity.
    Solution.
    \(\exp(x) \to \infty\) as \(x\to \infty\text{.}\)
    Proof.
    By the definition of function limits (Definition 5.12), we are required to show that if \(a_n\to \infty\) then \(\exp(a_n)\to \infty\text{.}\) We note that by definition of tending to infinity (Definition 3.2) there exists \(N\in \N\) such that for all \(n\geq N\) we have \(a_n\geq 0\text{.}\) For \(n\geq N\) we therefore have the inequality
    \begin{equation*} \exp(a_n) = 1+a_n+\trecip{2} a_n^2 + \dots \geq a_n . \end{equation*}
    Since \(a_n\to \infty\) the one-sided sandwich rule for sequence limits (Proposition 3.6) gives that \(\exp(a_n)\to \infty\text{.}\)
    (e)
    Justify why the exponential function is strictly increasing.
    Solution.
    Since \(\exp'(x) = \exp(x) \gt0\) and functions with positive derivative are strictly increasing (Corollary 8.9), we deduce that \(\exp\) is strictly increasing on \(\R\text{.}\)
    (f)
    State and prove how the exponential function is a bijection onto its image.
    Solution.
    Proposition 8.13.
    \(\exp\) is a bijection from \(\R\) to \((0, \infty)\text{.}\)
    Proof.
    Strictly increasing functions are injective, hence to show that \(\exp\) is bijective \(\R \to (0, \infty)\text{,}\) we must establish that every \(y\gt 0\) can be represented as \(y = \exp(x)\) for some \(x \in \R\text{.}\) Since \(\exp(x) \to \infty\) as \(x \to \infty\text{,}\) we have \(\exp(-x) = 1/\exp(x) \to 0\) as \(x \to \infty\text{.}\) Thus, for any \(y \in (0, \infty)\text{,}\) there exists \(x_1 \lt x_2\) such that
    \begin{equation*} \exp(x_1) \leq y\leq \exp(x_2). \end{equation*}
    Since differentiable functions are continuous, we may apply the intermediate value theorem to deduce the existence of \(x \in [x_1, x_2]\) such that \(\exp(x) = y\text{.}\)

    - 8.14. The logarithm function.

    (a)
    Define the logarithm function and justify why this definition is valid.
    Solution.
    Definition 8.15. Logarithm.
    Since the exponential function \(\exp : \R \to (0, \infty)\) is bijective (Proposition 8.13), it has a well-defined inverse \((0, \infty) \to \R\) which we denote by \(\log x\) for \(x \in (0, \infty)\text{.}\)
    (b)
    State and prove how the logarithm is a strictly increasing function, which is bijective onto its image.
    Solution.
    Proposition 8.16.
    \(\log\) is a strictly increasing bijection from \((0,\infty)\) to \(\R\text{.}\)
    Proof.
    The fact that \(\log\) is a bijection from \((0,\infty)\) to \(\R\) follows from its definition as the inverse of the bijection \(\exp : \R \to (0,\infty)\text{.}\)
    To prove that it is strictly increasing, we argue by contradiction. Let \(x_1, x_2 \in (0, \infty)\) with \(x_1 \lt x_2\) and suppose that \(\log(x_1) \geq \log(x_2)\text{.}\) The increasing nature of the exponential function (Proposition 8.11) gives that
    \begin{equation*} x_1 = \exp\brac{\log x_1} \geq \exp\brac{\log x_2} = x_2. \end{equation*}
    This is a contradiction, hence the logarithm is strictly increasing.
    (c)
    State and prove how the logarithm function is continuous.
    Solution.
    Proposition 8.17.
    \(\log\) is continuous on \((0,\infty)\text{.}\)
    Proof.
    Let \(x_0\in (0, \infty)\text{.}\) Using the epsilon-delta characterisation of continuity (Proposition 5.17), we are required to prove that for each \(\eps \gt 0\) there exists \(\delta = \delta(x_0,\eps) \gt 0\) such that
    \begin{equation*} |x-x_0| \leq \delta \quad \implies \quad |\log(x) - \log(x_0)| \leq \eps. \end{equation*}
    Writing \(y:=\log(x)\) and \(y_0:=\log(x_0)\) this is equivalent to showing that
    \begin{equation} |\exp(y)-\exp(y_0)| \leq \delta \quad \implies \quad |y - y_0| \leq \eps.\tag{8.3} \end{equation}
    By the mean value theorem (Theorem 8.7), there exists \(y_1\) between \(y\) and \(y_0\) such that
    \begin{equation} |\exp(y)-\exp(y_0)| = \exp(y_1)|y-y_0|. \tag{8.4} \end{equation}
    Since \(\exp\) is increasing, we have
    \begin{equation*} \exp(y_1) \geq \exp\brac{\min\set{y_0,y}} = \min\set{x_0,x}\text{.} \end{equation*}
    Hence we derive (8.3) on taking
    \begin{equation*} \delta(x_0,\eps) :=\min\set{\frac{x_0}{2}, \frac{\eps x_0}{2}}, \end{equation*}
    since with this choice we have
    \begin{equation*} \exp(y_1) \geq \min\set{x_0,x_0-\tfrac{x_0}{2}}=\tfrac{x_0}{2}\text{,} \end{equation*}
    which yields the required implication on substitution into (8.4).
    (d)
    State and prove how the logarithm function is differentiable.
    Solution.
    Proposition 8.18.
    \(\log\) is differentiable on \((0,\infty)\) with derivative
    \begin{equation*} \log'(x) = \recip{x}. \end{equation*}
    Proof.
    This follows from the inverse function theorem (Theorem 7.16) with \(f := \exp\text{,}\) since the hypotheses of that theorem are met, namely:
    • \(\exp : \R \to (0,\infty)\) is bijective (this follows from Proposition 8.13).
    • \(\exp^{-1}= \log\) is continuous at every \(x_0 \in (0,\infty)\) (this follows from Proposition 8.17).
    • \(\exp\) is differentiable at every \(x_0\in \R\) with non-zero derivative (this follows from Proposition 8.11).
    (e)
    State and prove the limit of the logarithm function as its argument tends to infinity.
    Solution.
    Proposition 8.19.
    \(\log x \to \infty\) as \(x\to\infty\text{.}\)
    Proof.
    By the definition of function limits (Definition 5.12), we are required to show that if \(a_n\to \infty\) then \(\log(a_n)\to \infty\text{.}\) Let \(M \gt 0\text{.}\) We note that by definition of tending to infinity (Definition 3.2) there exists \(N\in \N\) such that for all \(n\geq N\) we have \(a_n\geq \exp M\text{.}\) Since \(\log\) is increasing this gives \(\log a_n \geq M\) for \(n\geq N\text{.}\) It follows that \(\log(a_n)\to \infty\) by the sequential definition of tending to infinity (Definition 3.2).

    - 8.20. Exponentiation to an arbitrary real.

    (a)
    Define exponentiation to an arbitrary real and the logarithm to an arbitrary base.
    Solution.
    Definition 8.21. Exponentiation to an arbitrary real.
    Given \(a \in (0, \infty)\) and \(x \in \R\) we define
    \begin{equation*} a^x := \exp(x\log a). \end{equation*}
    We have previously defined \(a^n\) when \(n \in \Z\text{,}\) and [cross-reference to target(s) "prop-log-properties-4" missing or not unique] shows that our two definitions coincide when \(x = n \in \Z\text{.}\) [cross-reference to target(s) "prop-log-properties-5" missing or not unique] also shows that \(a^{1/n} = \sqrt[n]{a} \text{,}\) as we would expect.
    Definition 8.22. Logarithm to an arbitrary base.
    Given \(a \in (0, \infty)\) with \(a \neq 1\) and \(x \in \R\) we define
    \begin{equation*} \log_a(x) := \frac{\log x}{\log a}. \end{equation*}
    (b)
    Justify why \(a^x = b\) if and only if \(\log_a b = x\text{.}\)
    Solution.
    Since \(\log\) is injective on \((0,\infty)\text{,}\) we have
    \begin{equation*} a^x = b \iff \log(a^x) = \log(b). \end{equation*}
    By definition we have \(a^x := \exp(x\log a)\text{.}\) Since \(\log = \exp^{-1}\) we deduce that
    \begin{equation*} a^x = b \iff x\log a = \log b. \end{equation*}
    Since \(a \neq 1\text{,}\) we have \(\log a\neq0\text{,}\) thus we may divide by \(\log a\) to conclude that
    \begin{equation*} a^x = b \iff x = \frac{\log b}{\log a}. \end{equation*}
    By definition \(\log_a b = \log b / \log a\text{,}\) establishing the claim.
    (c)
    For \(a >0\) and \(x,y \in \R\text{,}\) prove the identities
    \begin{gather*} \log(a^x) = x\log a,\\ \brac{a^x}^y = a^{x y}. \end{gather*}
    Solution.
    By definition of exponentiation and the fact that \(\log = \exp^{-1}\text{,}\) we have
    \begin{equation*} \log(a^x) = \log\brac{\exp(x\log a)} = x \log a. \end{equation*}
    This establishes the first identity.
    Taking logarithms, the identity \(\brac{a^x}^y = a^{x y}\) is equivalent to
    \begin{equation*} \log\brac{\brac{a^x}^y} = \log\brac{a^{x y}}. \end{equation*}
    This holds on applying our first identity to both sides (twice on the left-hand side).
