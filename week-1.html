    Subsection 1.1 Self-test questions

    - 1.1. Upper and lower bounds.

    (a)
    Define what it means to be an upper/lower bound for a set/function.
    Solution.
    Definition 1.2. Upper/lower bound.
    We say that \(M \in \R\) is an upper bound for the set \(S\subset \R\) if
    \begin{equation} (\forall x \in S)[x \leq M].\tag{1.1} \end{equation}
    \(M\) is a lower bound for \(S\) if
    \begin{equation*} (\forall x \in S)[x \geq M]. \end{equation*}
    \(M\) is an upper/lower bound for the function \(f:D\to \R\) if it is an upper/lower bound for the image set
    \begin{equation*} f(D) := \set{f(x) : x \in D}. \end{equation*}
    (The notation "\(:=\)" should be read as "defined to equal".)
    Example 1.3.
    • If \(S =\set{1,2}\) then 3 is an upper bound for \(S\text{.}\)
      Proof.
      We must check that for every \(x\in S\) we have \(x \leq 3\text{.}\) The only elements of \(S\) are 1 and 2, and both of the inequalities \(1\leq 3\) and \(2\leq 3\) hold. We have therefore verified that the defining condition (1.1) of being an upper bound holds.
    • If \(S= \set{0,1}\) then 1 is an upper bound for \(S\text{.}\)
      Proof.
      We have \(0\leq1\) and \(1\leq 1\text{.}\) So for all \(x \in S\) we have \(x \leq 1\text{.}\)
    • If \(S= (0, \infty)\) then 0 is a lower bound for \(S\text{.}\)
      Proof.
      If \(x \in S\) then by definition \(x \geq 0\text{.}\)
    • If \(S= (-\infty,0)\) then \(-1\) is not an upper bound for \(S\text{.}\)
      Proof.
      Negating the definition (1.1) of being an upper bound, we must show that for \(M = -1\) we have
      \begin{equation} (\exists x \in S)[x \gt M].\tag{1.2} \end{equation}
      This holds since, for instance, \(x:= -1/2 \in S \) satisfies \(x > -1\text{.}\)
    (b)
    Define what it means for a set/function to be bounded above, bounded below and bounded.
    Solution.
    Definition 1.4. Bounded above/below.
    We say that a set \(S\subset\R\) is bounded above if it has an upper bound, that is
    \begin{equation} (\exists M \in \R)(\forall x \in S)[x \leq M].\tag{1.3} \end{equation}
    Similarly, a function \(f : D \to \R\) is bounded above if it has an upper bound, that is
    \begin{equation*} (\exists M \in \R)(\forall x \in D)[f(x) \leq M]. \end{equation*}
    A set/function is bounded below if it has a lower bound.
    A set/function is bounded if it is both bounded above and bounded below. For sets, this is equivalent to:
    \begin{equation} (\exists M \in \R)(\forall x \in S)[|x| \leq M].\tag{1.4} \end{equation}
    For functions, this is equivalent to:
    \begin{equation*} (\exists M \in \R)(\forall x \in D)[|f(x)| \leq M]. \end{equation*}

    - 1.5. Supremum and infimum.

    (a)
    Define what it means to be a supremum or infimum for a set of real numbers.
    Solution.
    Definition 1.6. Supremum.
    \(M \in \R\) is a supremum for \(S \subset \R\) if \(M\) is an upper bound for \(S\) and any any other upper bound \(\tilde M\) for \(S\) is larger, so that \(\tilde M \geq M\text{.}\)
    In other words, \(M\) is the least upper bound for \(S\text{,}\) and we denote this quantity \(\sup S\) (if it exists).
    Definition 1.7. Infimum.
    \(M \in \R\) is an infimum for \(S \subset \R\) if \(M\) is a lower bound for \(S\) and any any other lower bound \(\tilde M\) for \(S\) is smaller, so that \(\tilde M \leq M\text{.}\)
    In other words, \(M\) is the greatest lower bound for \(S\text{,}\) and we denote this quantity \(\inf S\) (if it exists).
    Example 1.8.
    • \(\sup\set{1,2,3} = 3.\)
      Proof.
      3 is an upper bound for \(\set{1,2,3}\text{.}\) We must show it is the least upper bound. Suppose that \(M\) is another upper bound for \(\set{1,2,3}\text{.}\) Then by definition of being an upper bound and the fact that 3 lies in our set, we have that \(3 \leq M\text{.}\) Thus 3 is smaller than any other upper bound, thereby satisfies the definition (Definition 1.6) of being a supremum.
    • \(\sup (-\infty, 1) = 1.\)
      Proof.
      By definition \((-\infty, 1) = \set{x\in \R : x \lt 1}\text{,}\) so 1 is certainly an upper bound for the set. It remains to show that 1 is the smallest upper bound. To this end, suppose that \(M\) is another upper bound for \((-\infty, 1)\text{.}\) Notice that if \(M \lt 1\text{,}\) then any element strictly between \(M\) and 1 (for instance the midpoint \((M+1)/2\) will be an element of \((-\infty, 1)\) which exceeds the upper bound \(M\) (a contradiction). Therefore we must have \(M\geq 1\text{,}\) so that 1 is the least upper bound (as claimed).
      Notice that this example shows that the supremum does not always lie in the set. When the supremum lies in the set we call it the maximum (see Definition 1.11).
    • \(\sup \emptyset\) does not exist.
      Proof.
      Every \(M\in \R\) is an upper bound for \(S = \emptyset\text{,}\) since the negation (1.2) does not hold (we say that a statement of the form \((\forall x \in \emptyset)[\dots]\) is true "vacuously", since the negation cannot hold due to there being no elements in the empty set). No real \(M\) can be the least upper bound for the empty set, since \(M-1\) is a smaller upper bound.
      This means that we can only take the supremum of a non-empty set.
    • If \(S\subset \R\) is not bounded above (for instance \(S=\R\)) then \(S\) has no upper bound, in particular no least upper bound. Thus \(\sup S\) does not exist.
      This means that we can only take the supremum of a set which is bounded above.
      The completeness principle (Theorem 1.15) says that if \(S\) is non-empty and bounded above, then \(\sup S\) exists.
    (b)
    Given a function \(f:D\to\R\text{,}\) define the notation \(\sup_S f\) and \(\inf_S f\) (when these quantities exist).
    Solution.
    Given a function \(f:D\to \R\) and a set \(S\subset D\) we write
    \begin{equation*} \sup_S f = \sup_{x\in S} f(x) := \sup\set{f(x) : x \in S}. \end{equation*}
    This supremum exists if and only if \(f\) is bounded above on \(S\) and \(S\) is non-empty.
    We write
    \begin{equation*} \inf_S f = \inf_{x\in S} f(x) := \inf\set{f(x) : x \in S}. \end{equation*}
    This infimum exists if and only if \(f\) is bounded below on \(S\) and \(S\) is non-empty.
    (c)
    State and prove how decreasing a supremum gives a non-upper bound.
    Solution.
    Lemma 1.9. Decreasing a sup gives a non-upper bound.
    Let \(S \subset \R\) be non-empty and bounded above. Then
    \begin{equation*} (\forall \eps > 0)(\exists x \in S)[x> \sup S-\eps]. \end{equation*}
    Proof.
    Let \(\eps\gt0\) be arbitrary; we are required to find \(x\in S\) such that \(x\gt \sup S-\eps\text{.}\) Aiming for a contradiction, let us suppose that this does not hold, so that for all \(x\in S\) we have \(x\leq \sup S-\eps\text{.}\) But then \(\sup S-\eps\) is an upper bound for \(S\) which is strictly smaller than \(\sup S\text{.}\) Since \(\sup S\) is the smallest upper bound for \(S\text{,}\) we have obtained our desired contradiction.

    - 1.10. Maximum and minimum.

    (a)
    Define the maximum and minimum for a set of real numbers (when these quantities exist).
    Solution.
    Definition 1.11. Maximum.
    We say that a set of real numbers \(S \subset \R\) has a maximum if \(\sup S\) exists and \(\sup S \in S\text{.}\) In this situation, we write \(\max S\) for \(\sup S\) and call it the maximum.
    Definition 1.12. Minimum.
    We say that a set of real numbers \(S \subset \R\) has a minimum if \(\inf S\) exists and \(\inf S \in S\text{.}\) In this situation, we write \(\min S\) for \(\inf S\) and call it the minimum.
    Example 1.13.
    • \(\max \set{1,2,3} = 3\text{.}\)
    • \(\max (-\infty, 1)\) does not exist.
    • For any finite non-empty set, both \(\max S\) and \(\min S\) exist.
    (b)
    Define the maximum and minimum for a real-valued function \(f : D \to \R\) (when these quantities exist).
    Solution.
    Given a set \(S \subset D\text{,}\) we write
    \begin{equation*} \max_S f = \max_{x\in S} f(x) := \max\set{f(x) : x \in S}, \end{equation*}
    provided that this quantity exists (i.e. provided that the set \(\set{f(x) : x \in S}\) has a supremum which belongs to the set).
    Similarly, we write
    \begin{equation*} \min_S f = \min_{x\in S} f(x) := \min\set{f(x) : x \in S}, \end{equation*}
    provided that this quantity exists.

    - 1.14. Completeness principle.

    State (but don’t prove) the "completeness principle" for the real numbers.
    Solution.
    Theorem 1.15. The completeness principle.
    Suppose that a set of real numbers \(S\subset \R\) is
    • non-empty
    • bounded above,
    then \(S\) has a supremum, \(\sup S\in \R\text{.}\)
    In other words, any set with an upper bound has a least upper bound (provided that the set is non-empty).
    Proof.
    NON-EXAMINABLE. The proof depends on how one constructs the real numbers, of which there are many alternative constructions (e.g. Dedekind cuts of rationals, decimal expansions, equivalence classes of Cauchy sequences, etc). See en.wikipedia.org/wiki/Completeness_of_the_real_numbers 1 .
    Remark 1.16.
    In this course, the completeness principle is the main result we use (repeatedly, sometimes in a hidden manner) in order to show that certain limits exist. For instance all of the following involve a limit in their construction, and the existence of this limit is guaranteed (under certain conditions) by the completeness principle:
    \begin{equation*} \lim_{n\to\infty} a_n, \qquad \sum_{n=1}^\infty a_n, \qquad \lim_{x\to x_0} f(x),\qquad f'(x_0), \qquad \int_a^b f. \end{equation*}
    Remark 1.17.
    The completeness principle is equivalent to the following statement involving the infimum:
    Corollary 1.18.
    Suppose that a set of real numbers \(S\subset \R\) is
    • non-empty
    • bounded below,
    then \(S\) has an infimum, \(\inf S\in \R\text{.}\)
    In other words, any set with a lower bound has a greatest lower bound (provided that the set is non-empty).
    The second statement can be derived from the first by reflecting the set through zero to obtain \(-S:=\set{-x: x\in S}\text{,}\) then observing that \(\sup(-S) = -\inf S\text{.}\)
    Example 1.19.
    As an example of how the completeness principle can be used, we outline how to construct a square-root of two from the rational numbers.
    First observe that the set
    \begin{equation*} \set{a/b \in \Q : (a/b)^2 \leq 2} \end{equation*}
    is bounded above (e.g. by 100) and non-empty (e.g. it contains 1). Hence by the completeness principle, it has a supremum
    \begin{equation*} x :=\sup\set{a/b \in \Q : (a/b)^2 \leq 2}. \end{equation*}
    We leave it as an exercise for the enthusiastic reader to demonstrate that \(x^2 = 2\text{,}\) so that \(x\) is a square-root of two.

    - 1.20. Sequences.

    (a)
    Define what it means to be a sequence of real numbers.
    Solution.
    A sequence of real numbers is a function \(f : \N \to \R\text{.}\)
    Remark 1.21.
    Rather than using function notation \(f:\N\to \R\text{,}\) we usually write a sequence as
    \begin{equation*} (a_n)_{n=1}^\infty = (a_1, a_2, a_3, \dots) \end{equation*}
    where \(a_n := f(n)\text{.}\)
    We may also write \((a_n)_{n\in \N}\) or \((a_n)_n\) or simply \((a_n)\text{.}\) We may also change the starting point of the sequence, for instance: \((a_n)_{n=0}^\infty\) or \((a_n)_{n=100}^\infty\text{,}\) etc.
    (b)
    State what it means for a sequence to be in a set of real numbers.
    Solution.
    Given \(S \subset \R\text{,}\) we say that a sequence \((a_n)_{n=1}^\infty\) is in \(S\) if for all \(n \in \N\) we have \(a_n \in S\text{.}\)
    (c)
    Define the "\(n\)th term" of a sequence.
    Solution.
    The "\(n\)th term" of the sequence \((a_n)_n\) is the number \(a_n\text{.}\)
    (d)
    Give some examples and non-examples of sequences. In particular, give an example of a function and a set which are not sequences (i.e. demonstrate your knowledge of the difference between a set, a function and a sequence).
    Solution.
    Some examples of sequences include:
    • The enumeration of the even numbers in increasing order
      \begin{equation*} (2n)_{n=1}^\infty = (2, 4, 6, 8, \dots)\text{.} \end{equation*}
    • An enumeration of the even numbers in a different order
      \begin{equation*} (4,2,8,6,12,10, \dots)\text{.} \end{equation*}
      Notice that this is a different sequence to our first example, since they have different 1st term. The order of a sequence matters.
    • A constant sequence, consisting of the same real number repeated at every term, e.g.
      \begin{equation*} (\pi, \pi, \pi, \dots)\text{,} \end{equation*}
      so that \(a_n = \pi\) for all \(n \in \N\text{.}\)
    • The sequence of values of the \(\sin\) function evaluated at positive integers
      \begin{equation*} (\sin(n))_{n=1}^\infty = (\sin(1), \sin(2), \sin(3), \dots)\text{.} \end{equation*}
    Some non-examples of sequences include:
    • The sine function \(\sin:\R\to\R\text{.}\) The problem is that the domain of this function is not a countable set.
    • The set of even numbers
      \begin{equation*} \set{2, 4, 6, \dots} = \set{4,2,8,6,12,10,\dots}\text{.} \end{equation*}
      The problem here is that there is no ordering associated with a set. It is not clear what the \(n\)th term of a set is.
    • The set
      \begin{equation*} \set{\pi, \pi, \pi, \dots} = \set{\pi}\text{.} \end{equation*}
      For a sequence, different terms may take the same value. It makes sense to say that the value \(\pi\) occurs infinitely often in a sequence. However, there are no repetitions of elements in a set: an element either belongs once or not at all.

    - 1.22. Convergent sequences.

    (a)
    Define what it means for a sequence of real numbers to converge to a real limit.
    Solution.
    Definition 1.23. Limit of convergent sequence.
    We say that a sequence of real numbers \((a_n)_{n=1}^\infty\) converges to \(a \in \R\text{,}\) and write \(a_n \to a\) as \(n\to \infty\) or \(\lim_{n\to\infty} a_n = a\text{,}\) if
    \begin{equation} (\forall \eps \gt 0)( \exists N = N(\eps))( \forall n \geq N)[ |a_n-a| \leq \eps].\tag{1.5} \end{equation}
    Remark 1.24.
    • Typically, one takes \(N = N(\eps)\) to be a positive integer, so \(N \in \N\) denotes the \(N\)th term of the sequence, after which all remaining terms of the sequence lie in the "\(\eps\)-neighbourhood of \(a\)":
      \begin{equation} [a-\eps, a+\eps] = \set{x \in \R : |x-a| \leq \eps}.\tag{1.6} \end{equation}
      However, it sometimes convenient not to have to worry about whether \(N\) is an integer or positive, so we allow for \(N \in \R\) in our definition (the two definitions are equivalent).
    • One can paraphrase "convergence to \(a\)" as "every \(\eps\)-neighbourhood of \(a\) contains all terms of the sequence from some point onwards".
    • You may have seen strict inequality "\([ |a_n-a| \lt \eps]\)" in the definition of convergence. Again, the two definitions are equivalent.
    (b)
    Give examples of sequences which converge and which don’t converge.
    Solution.
    Example 1.25. Constant sequences are convergent.
    Fix a real number \(a \in \R\text{,}\) say \(a = 100\text{.}\) For each \(n \in \N\) set \(a_n := a\text{.}\) Then we claim that \(a_n \to a\text{.}\) To prove this, we must verify that (1.5) holds. To this end, fix \(\eps \gt 0\text{.}\) We must find \(N = N(\eps)\) such that
    \begin{equation} n \geq N(\eps)\implies |a_n-a| \leq \eps.\tag{1.7} \end{equation}
    For this particular sequence, we have \(|a_n-a| = |a-a| = 0\) for all \(n\text{.}\) Hence (1.7) holds on taking \(N(\eps) :=1\text{.}\)
    Example 1.26. The reciprocals of the naturals converge to 0.
    \(1/n \to 0\) as \(n\to \infty\)
    Proof.
    We must verify (1.5). Fixing \(\eps \gt 0\text{,}\) this constitutes showing that, from some point onwards, all terms lie in the \(\eps\)-neighbourhood of zero \([-\eps,\eps]\text{.}\) In other words, we want to find \(N = N(\eps)\) such that
    \begin{equation*} n \geq N(\eps)\implies 1/n \in [-\eps, \eps]. \end{equation*}
    Setting \(N(\eps):= 1/\eps\) we see that if \(n\geq N(\eps) = 1/\eps\) then re-arranging the inequality gives \(1/n \leq \eps\text{,}\) and so \(|\recip{n}-0|\leq \eps\) (as required).
    Example 1.27. A non-convergent sequence.
    We claim that the sequence \((a_n)_n\) defined by \(a_n:=n\) for all \(n\) does not converge to any real number \(a \in \R\text{.}\) If \(a\in \R\) were the limit of this sequence, then taking \(\eps:=1/3\) in (1.5) gives the existence of \(N\) such that for all \(n\geq N\) we have \(|a-n| \leq 1/3\text{.}\) In particular, by the triangle inequality we have
    \begin{equation*} 1 = |(N+1)-N| \leq |(N+1)-a|+|a-N| \leq 2/3. \end{equation*}
    This is a contradiction.
    Example 1.28. Another non-convergent sequence.
    We claim that the sequence
    \begin{equation*} ((-1)^n)_{n=1}^\infty = (-1,1,-1,1,\dots) \end{equation*}
    does not converge to any real number \(a \in \R\text{.}\) If \(a\in \R\) were the limit of this sequence, then taking \(\eps:=1/2\) in (1.5) gives the existence of \(N\) such that for all \(n\geq N\) we have \(|a-(-1)^n| \leq 1/2\text{.}\) In particular, by the triangle inequality we have
    \begin{equation*} 2 = |(-1)^{N+1}-(-1)^N| \leq |(-1)^{N+1}-a|+|a-(-1)^N| \leq 1/2 + 1/2 = 1. \end{equation*}
    This is a contradiction.

    - 1.29. Convergent sequences are bounded.

    (a)
    State and prove how convergent sequences are bounded.
    Solution.
    Proposition 1.30. Convergent sequences are bounded.
    Suppose that \((a_n)\) is a convergent sequence of reals. Then \((a_n)\) is a bounded sequence.
    Proof.
    By definition of a bounded function (Definition 1.2) we must show that
    \begin{equation*} (\exists M \in \R)(\forall n \in \N)[|a_n| \leq M]. \end{equation*}
    Since the sequence is convergent, there exists \(a \in \R\) such that \(a_n \to a\text{.}\) Taking \(\eps :=1\) in the definition of the limit (Definition 1.23), there exists \(N = N(1)\) such that for all \(n \geq N\) we have \(|a_n - a| \leq 1\text{.}\) Therefore, when \(n \geq N\text{,}\) the triangle inequality gives
    \begin{equation*} |a_n| = |a_n - a + a| \leq |a_n-a| + |a| \leq 1 + |a|. \end{equation*}
    Rounding up if necessary, we may assume that \(N \in \N\text{,}\) so we can set
    \begin{equation*} M:= \max\set{|a_1|, \dots, |a_{N}|, 1+|a|}. \end{equation*}
    Then for all \(n \in \N\) we have \(|a_n| \leq M\text{.}\)
    (b)
    Give an example of a sequence which is bounded but not convergent.
    Solution.
    The alternating sequence \((-1,1,-1,1,-1, \dots)\) is bounded but not convergent (see Example 1.28).

    - 1.33. Exam-style self-test question for workshop.

    (a)
    Define what it means for the sequence \((a_n)_{n=1}^\infty\) to converge to 1.
    Solution.
    This is Task 1.22.a with \(a = 1\text{.}\)
    (b)
    Setting
    \begin{equation} a_n:= \frac{n^2}{n^2+1},\tag{1.8} \end{equation}
