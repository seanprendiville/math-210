Appendix E Exam information sheet

The following key information will be included in your summer examination paper.

  • Notation

    • "\(P \iff Q\)": "the proposition \(P\) holds if and only if the proposition \(Q\) holds".

    • "\(\neg P\)": "the proposition \(P\) does not hold".

    • "\(P \implies Q\)": "it is not the case that the proposition \(P\) holds and the proposition \(Q\) does not hold", equivalently \(\neg(P\& \neg Q)\text{.}\)

    • "\(x \in A\)": "\(x\) is an element of the set \(A\)".

    • "\(\forall x \in A\)": "for all \(x \in A\)".

    • "\(\exists x \in A\)": "there exists \(x \in A\)".

    • \(\N := \set{1,2, 3, \dots}\text{.}\)

    • \(\Z := \set{\dots, -3,-2,-1,0,1,2, 3, \dots}\text{.}\)

    • Non-negative integers \(:= \set{0,1,2,3, \dots}\text{.}\)

    • \(\Q := \set{\frac{a}{b} : a \in \Z,\ q\in \N}\text{.}\)

    • \(\R := \text{the set of real numbers}\text{.}\)

    • Non-negative reals \(:= \set{x \in \R : x \geq 0}\text{.}\)

    • Positive reals \(:= \set{x \in \R : x \gt 0}\text{.}\)

    • Negative reals \(:= \set{x \in \R : x \lt 0}\text{.}\)

    • \(A\cup B := \set{x : x \in A \text{ or } x\in B}\text{.}\)

    • \(A\cap B := \set{x : x \in A \text{ and } x\in B}\text{.}\)

    • \(A\setminus B := \set{x : x \in A \text{ and } x\notin B}\text{.}\)

    • \(\displaystyle\bigcup_{i=1}^n A_i := \set{x : x \in A_i \text{ for some } i =1, \dots, n}\text{.}\)

    • \(\displaystyle\bigcap_{i=1}^n A_i := \set{x : x \in A_i \text{ for all } i =1, \dots, n}\text{.}\)

    • \(\displaystyle(a,b) := \set{x \in \R : a \lt x \lt b}\text{.}\)

    • \(\displaystyle(a,b] := \set{x \in \R : a \lt x \leq b}\text{.}\)

    • \(\displaystyle[a,b) := \set{x \in \R : a \leq x \lt b}\text{.}\)

    • \(\displaystyle[a,b] := \set{x \in \R : a \leq x \leq b}\text{.}\)

  • Key properties of \(\R\)

    • Additive axioms:

      • Additive closure: \((\forall x,y \in \R)[x+y \in \R]\text{.}\)

      • Additive associativity: \((\forall x,y,z \in \R)[(x+y)+z = x+(y+z) \in \R]\text{.}\)

      • Additive identity: \((\exists 0 \in \R)(\forall x \in \R)[0+x = x]\text{.}\)

      • Additive inverses: \((\forall x \in \R)(\exists -x \in \R)[x+(-x) = 0]\text{.}\)

      • Addition is commutative: \((\forall x,y \in \R)[x+y = y+x]\text{.}\)

    • Multiplicative axioms:

      • Multiplicative closure: \((\forall x, y \in \R)[x\cdot y \in \R]\text{.}\)

      • Multiplicative associativity: \((\forall x,y,z \in \R)[(x\cdot y)\cdot z = x\cdot (y\cdot z) \in \R]\text{.}\)

      • Multiplicative identity: \((\exists 1 \in \R\setminus\set{0})(\forall x \in \R)[1\cdot x = x]\text{.}\)

      • Multiplicative inverses: \((\forall x \in \R\setminus\set{0})(\exists x^{-1} \in \R\setminus\set{0})[x\cdot x^{-1} = 1]\text{.}\)

      • Multiplication is commutative: \((\forall x,y \in \R)[x\cdot y = y\cdot x]\text{.}\)

    • Interaction of addition and multiplication:

      • Distributivity: \((\forall x,y,z \in \R)[x\cdot(y+z) = xy + xz]\text{.}\)

    • Order axioms:

      • Reflexivity: \((\forall x \in \R)[x \leq x]\text{.}\)

      • Anti-symmetry: \((\forall x, y \in \R)\sqbrac{(x \leq y\ \&\ y \leq x)\implies x = y}\text{.}\)

      • Transitivity: \((\forall x, y, z\in \R)\sqbrac{(x \leq y\ \&\ y \leq z)\implies x \leq z}\text{.}\)

      • Totality: \((\forall x, y \in \R)[x\leq y \text{ or } y \leq x]\text{.}\)

    • Interaction of order and addition/multiplication:

      • Additive invariance: \((\forall x, y, z \in \R)[ x \leq y \implies x+z \leq y + z]\text{.}\)

      • Non-negative multiplicative invariance:\\ \((\forall x, y, z \in \R)[(x \leq y \ \& \ 0 \leq z) \implies xz \leq yz]\text{.}\)

  • Triangle inequalities

    • \((\forall x, y \in \R)[\ |x\pm y| \leq |x| + |y|\ ]\text{.}\)

    • \((\forall x, y \in \R)[\ |x\pm y| \geq |x| - |y|\ ]\text{.}\)
